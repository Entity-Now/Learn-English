要求：
1. 输入：仅以我提供的词根列表为依据，**严格逐一处理，不添加额外词根或额外解释**。
2. 输出格式：采用 Markdown 格式，每个词根为一个独立单元，各单元之间用分隔符（***）隔开。
3. 每个词根单元的内容结构固定为以下 **4 个部分**（不能增删改）：
    - **1. 核心提炼（记）**：  
      - 词根含义（极简）  
      - 词根记忆关键词（1 句）

      - 生成 2 个高度形象化、生活化、易记的场景联想，帮助把词根与真实语义建立连接（要求自然可信，不得牵强）

    - 测验题需简明、可立即执行
