### 英语词根背诵清单

- **act**
  - 词根：act
  - 核心含义：做
  - 古人发明此词根的意思：源自拉丁语 *agere* 的词干形式，意为“做、驱动、执行”
  - 记忆技巧：联想中文“行动”，行动就是“做”事
  - 高频词（3个）：
    - **action** = act(做)+ion(名词后缀) → 行动，行为
    - **active** = act(做)+ive(形容词后缀) → 积极的，活跃的
    - **react** = re(反)+act(做) → 反应，作出回应

***
- **ag**
  - 词根：ag
  - 核心含义：做
  - 古人发明此词根的意思：源自拉丁语 *agere*，原始含义为“驱动、做、行动”
  - 记忆技巧：谐音“爱做”，直接关联词根含义“做”
  - 高频词（3个）：
    - **agent** = ag(做)+ent(表人/物的后缀) → 代理人，代理商
    - **agenda** = ag(做)+enda(名词后缀) → 议程，日程表
    - **agitate** = ag(做)+itate(动词后缀) → 鼓动，搅动

***
- **oper**
  - 词根：oper
  - 核心含义：工作
  - 古人发明此词根的意思：源自拉丁语 *opus* 或 *operis*，意为“工作、劳动、成果”
  - 记忆技巧：联想单词“operate”，操作就是一种“工作”行为
  - 高频词（3个）：
    - **operate** = oper(工作)+ate(动词后缀) → 操作，运转
    - **operation** = oper(工作)+ation(名词后缀) → 操作，手术
    - **cooperate** = co(共同)+oper(工作)+ate(动词后缀) → 合作，协作

***
- **fac/fact**
  - 词根：fac/fact
  - 核心含义：做
  - 古人发明此词根的意思：源自拉丁语 *facere*，意为“做、制作、完成”，*fact* 为其过去分词词干
  - 记忆技巧：谐音“法克”，联想“用方法做事情”
  - 高频词（3个）：
    - **factory** = fact(做)+ory(地点后缀) → 工厂
    - **manufacture** = manu(手)+fact(做)+ure(名词后缀) → 制造，生产
    - **facilitate** = fac(做)+ilitate(动词后缀) → 促进，使便利

***
- **fic**
  - 词根：fic
  - 核心含义：做
  - 古人发明此词根的意思：源自拉丁语 *facere* 的变体词干，意为“做、造成、完成”
  - 记忆技巧：谐音“费克”，联想“费力做事情”
  - 高频词（3个）：
    - **fiction** = fic(做)+tion(名词后缀) → 小说，虚构
    - **artificial** = arti(技巧)+fic(做)+ial(形容词后缀) → 人工的，人造的
    - **sacrifice** = sacri(神圣)+fic(做)+e → 牺牲，献祭

***
- **pos**
  - 词根：pos
  - 核心含义：放
  - 古人发明此词根的意思：源自拉丁语 *ponere* 的过去分词词干 *positus*，意为“放置、安放、设定”
  - 记忆技巧：谐音“破死”，联想“把东西放好，防止搞破坏”
  - 高频词（3个）：
    - **position** = pos(放)+ition(名词后缀) → 位置，职位
    - **deposit** = de(向下)+pos(放)+it → 存放，存款
    - **expose** = ex(向外)+pos(放)+e → 暴露，揭露

***
- **pon**
  - 词根：pon
  - 核心含义：放
  - 古人发明此词根的意思：源自拉丁语 *ponere*，原始含义为“放置、安放、摆置”
  - 记忆技巧：谐音“碰”，联想“放置东西时小心碰撞”
  - 高频词（3个）：
    - **component** = com(共同)+pon(放)+ent(名词后缀) → 组成部分，成分
    - **opponent** = op(相反)+pon(放)+ent(表人后缀) → 对手，反对者
    - **postpone** = post(后)+pon(放)+e → 推迟，延期

***
- **cap**
  - 词根：cap
  - 核心含义：拿
  - 古人发明此词根的意思：源自拉丁语 *capere*，意为“拿、抓住、掌握”
  - 记忆技巧：谐音“开普”，联想“打开盖子拿东西”
  - 高频词（3个）：
    - **capture** = cap(拿)+ture(名词/动词后缀) → 捕获，俘虏
    - **capable** = cap(拿)+able(能…的) → 有能力的，能胜任的
    - **capacity** = cap(拿)+acity(名词后缀) → 容量，能力

***
- **cept**
  - 词根：cept
  - 核心含义：拿
  - 古人发明此词根的意思：源自拉丁语 *capere* 的过去分词词干 *ceptus*，意为“拿、接受、捕获”
  - 记忆技巧：谐音“赛普特”，联想“比赛中拿取奖品”
  - 高频词（3个）：
    - **accept** = ac(加强)+cept(拿) → 接受，承认
    - **concept** = con(共同)+cept(拿) → 概念，观念
    - **except** = ex(除外)+cept(拿) → 除…之外，除外

***
- **cip**
  - 词根：cip
  - 核心含义：拿
  - 古人发明此词根的意思：源自拉丁语 *capere* 的变体词干，意为“拿、取、掌握”
  - 记忆技巧：谐音“西普”，联想“伸手拿取西边的物品”
  - 高频词（3个）：
    - **participate** = parti(部分)+cip(拿)+ate(动词后缀) → 参与，参加
    - **anticipate** = anti(预先)+cip(拿)+ate(动词后缀) → 预期，预料
    - **recipient** = re(再)+cip(拿)+ient(表人后缀) → 接收者，收件人
