*   #### **psych**

    *   **核心含义**：心灵、心理
    *   **词源背景**：源自希腊语 *psyche*，意为“灵魂、气息、心智”；古希腊戏剧与医术中常把人的“气息/灵魂状态”与行为、健康联系起来。
    *   **记忆技巧**：联想“心理（psych）活动像气息起伏”，情绪波动看得见。
    *   **高频词（3个）**：
        *   **psychology** = psych(心理)+logy(学科) → 心理学 /saɪˈkɒlədʒi/
        *   **psychological** = psych(心理)+log(学术)+ical(形容) → 心理的 /ˌsaɪkəˈlɒdʒɪkəl/
        *   **psychiatrist** = psych(心灵)+iatri(治疗)+st(人) → 精神科医生 /saɪˈkaɪətrɪst/
    *   **组合逻辑解释**：
        *   psychology：心理+学科 → 研究心智运作的学问 → 心理学
        *   psychological：心理+学术+形容词 → 与心理学相关的事物 → 心理的
        *   psychiatrist：心灵+治疗+人 → 以治疗心灵问题为职业的人 → 精神科医生
    *   **词缀总结**：-logy(学科) 表“……学”；-logical/-ical(形容) “……的”；-iatry/-iatrist 表“治疗/医师”。

*   #### **sens**

    *   **核心含义**：感觉、感知
    *   **词源背景**：源自拉丁语 *sentire*，意为“感觉、感受、思考”；罗马法与修辞中常以“感觉—判断”描述民众意见。
    *   **记忆技巧**：联想 “sensor 传感器”——用来“感受”。
    *   **高频词（3个）**：
        *   **sense** = sens(感觉)+e(名词) → 感觉；道理 /sens/
        *   **sensible** = sens(感觉)+ible(能……的) → 明智的 /ˈsensəbl/
        *   **sensitive** = sens(感觉)+itive(形容) → 敏感的 /ˈsensətɪv/
    *   **组合逻辑解释**：
        *   sense：感觉 → 能分辨冷热、对错 → 感觉/道理
        *   sensible：感觉+能够 → 能正确“感觉到”利弊 → 明智的
        *   sensitive：感觉+形容 → 感觉神经“放大” → 敏感的
    *   **词缀总结**：-ible(能够)；-ive(具有……性质的)；前缀 re-/pre-/de- 常与 sens 结合构成“再、预先、降低”的感知含义（如 *resent*, *pre-sense* 非常见但构词逻辑一致）。

*   #### **path**

    *   **核心含义**：感情；痛苦
    *   **词源背景**：源自希腊语 *pathos*，“情感、遭遇、痛感”；古希腊修辞三要素之一 *pathos* 指“打动情感”。
    *   **记忆技巧**：联想 “同感同病相怜（sym-pathy）”。
    *   **高频词（3个）**：
        *   **sympathy** = sym(共同)+path(感情) → 同情 /ˈsɪmpəθi/
        *   **empathy** = em(进入)+path(感情) → 共情 /ˈempəθi/
        *   **pathology** = path(病痛)+logy(学科) → 病理学 /pəˈθɒlədʒi/
    *   **组合逻辑解释**：
        *   sympathy：共同+感情 → 与他人“同频感受” → 同情
        *   empathy：进入+感情 → 设身处地“进入他人情感” → 共情
        *   pathology：痛苦/疾病+学科 → 研究病因与病变 → 病理学
    *   **词缀总结**：syn-/sym-(共同)；em-(进入)；-logy(学科)。与 *-path* 相关的 *-pathy* 多指“某种情感/病况”。

*   #### **mem**

    *   **核心含义**：记忆、记住
    *   **词源背景**：源自拉丁语 *memor*，“记得的、念及的”；罗马祭祀与家族谱系重视“记忆传统”。
    *   **记忆技巧**：联想 “memo 备忘录”——帮你记住要点。
    *   **高频词（3个）**：
        *   **memory** = mem(记忆)+ory(名词) → 记忆 /ˈmeməri/
        *   **remember** = re(再次)+mem(记住)+ber(动词词尾) → 记起 /rɪˈmembə/
        *   **memo** = mem(记)+o(名词) → 备忘录 /ˈmeməʊ/
    *   **组合逻辑解释**：
        *   memory：记忆+名词 → 大脑的“存档” → 记忆
        *   remember：再次+记住 → 让存档重新“弹出” → 记起
        *   memo：记+名词 → 把要点写下便于“记” → 备忘录
    *   **词缀总结**：re-(再次)；-ory/-er/-o 常名词化；与 mem 结合常见 re-, com-(*commemorate*) 等表达“再次/共同记”。

*   #### **gnos**

    *   **核心含义**：知道、认知
    *   **词源背景**：源自希腊语 *gnōsis*，“知识、领悟”；古希腊医学与神秘学中指对病情或真理的“辨识”。
    *   **记忆技巧**：联想 “diagnose 通过线索（dia-穿透）来知道”。
    *   **高频词（3个）**：
        *   **diagnose** = dia(穿透)+gnos(知道)+e → 诊断 /ˌdaɪəɡˈnəʊz/
        *   **prognosis** = pro(向前)+gnos(知道)+is → 预后 /prɒɡˈnəʊsɪs/
        *   **agnostic** = a(不)+gnos(知道)+tic(形容) → 不可知论的 /æɡˈnɒstɪk/
    *   **组合逻辑解释**：
        *   diagnose：穿透+知道 → 看透症状得出“知道” → 诊断
        *   prognosis：向前+知道 → 预先得知病情走向 → 预后
        *   agnostic：不+知道 → 认为“无法知道”终极真相 → 不可知论的
    *   **词缀总结**：dia-(穿透)、pro-(向前)、a-(否定)；-sis/-tic 常名词/形容词化。与拉丁同源 *cogn-*（如 *recognize*）意义相近。

*   #### **cred**

    *   **核心含义**：相信、信任
    *   **词源背景**：源自拉丁语 *credere*，“托付、相信”；在罗马社会，“信用”与“债务”相连。
    *   **记忆技巧**：联想 “credit 卡=信用卡”，别人愿意先信你。
    *   **高频词（3个）**：
        *   **credit** = cred(相信)+it(名词) → 信用；学分 /ˈkredɪt/
        *   **credible** = cred(相信)+ible(能……的) → 可信的 /ˈkredɪbəl/
        *   **incredible** = in(不)+cred(相信)+ible → 难以置信的 /ɪnˈkredɪbəl/
    *   **组合逻辑解释**：
        *   credit：相信+名词 → 别人愿意“先信你” → 信用/学分
        *   credible：相信+能够 → 有让人能信的条件 → 可信的
        *   incredible：不+相信+能够 → 让人“不能信” → 难以置信的
    *   **词缀总结**：in-(否定)；-ible(能够……的)。同源派生如 *credence, credential* 皆与信任、凭据相关。

*   #### **fid**

    *   **核心含义**：信任、忠诚
    *   **词源背景**：源自拉丁语 *fides*，“信、誓约”；古罗马以手握为誓（*fiducia*）象征承诺。
    *   **记忆技巧**：联想 “con-fide 倾诉秘密需要信任”。
    *   **高频词（3个）**：
        *   **confident** = con(共同)+fid(信任)+ent(形容) → 自信的 /ˈkɒnfɪdənt/
        *   **confide** = con(向…)+fid(信任)+e → 吐露 /kənˈfaɪd/
        *   **fidelity** = fid(信任)+elity(名词) → 忠诚 /fɪˈdeləti/
    *   **组合逻辑解释**：
        *   confident：共同+信任 → 对自己“信任度高” → 自信的
        *   confide：向……+信任 → 因信任而“说心里话” → 吐露
        *   fidelity：信任+名词 → 对承诺保持信 → 忠诚
    *   **词缀总结**：con-(共同/向)；-ent(形容)；-ity(名词性)。与 *fiduciary, diffident* 同源表达“信/不信”。

*   #### **mor**

    *   **核心含义**：风俗、道德
    *   **词源背景**：源自拉丁语 *mos, mores*，“习俗、礼法”；罗马社会通过“共同风俗”维系秩序。
    *   **记忆技巧**：联想 “moral 道德像社会的默认设置（default）”。
    *   **高频词（3个）**：
        *   **moral** = mor(风俗)+al(形容) → 道德的 /ˈmɒrəl/
        *   **morale** = mor(风俗/群体氛围)+ale(名词) → 士气 /məˈrɑːl/
        *   **morality** = mor(风俗)+ality(名词) → 道德规范 /məˈrælɪti/
    *   **组合逻辑解释**：
        *   moral：风俗+形容 → 符合社会习俗的行为标准 → 道德的
        *   morale：风俗/群体+名词 → 团队共同风气的“精神面貌” → 士气
        *   morality：风俗+名词 → 由风俗沉淀成规则 → 道德规范
    *   **词缀总结**：-al(形容)；-ity/-ity 族名词化；与 *immoral, moralize* 等派生表达否定/动词化。

***

*   #### **phil**

    *   **核心含义**：爱，喜好
    *   **词源背景**：源自希腊语 *philein*，“爱”；古希腊学术中“爱智慧（philo+sophia）”被视为求知的动力。
    *   **记忆技巧**：联想 “philosophy=爱智慧”。
    *   **高频词（3个）**：
        *   **philosophy** = phil(爱)+sophy(智慧) → 哲学 /fɪˈlɒsəfi/
        *   **philosopher** = phil(爱)+soph(智慧)+er(人) → 哲学家 /fəˈlɒsəfə/
        *   **bibliophile** = biblio(书)+phil(爱) → 爱书者 /ˈbɪbliəʊfaɪl/
    *   **组合逻辑解释**：
        *   philosophy：爱+智慧 → 因热爱而追问真理 → 哲学
        *   philosopher：爱+智慧+人 → 以爱智为业的人 → 哲学家
        *   bibliophile：书+爱 → 对书充满喜爱的人 → 爱书者
    *   **词缀总结**：-er(人)；复合前缀如 biblio-(书)、anglo-(英格兰) 与 phil 组合表“爱好某物/族群”。

***

*   #### **soph**

    *   **核心含义**：智慧；技巧
    *   **词源背景**：源自希腊语 *sophos*，“聪慧、善辩”；古希腊“智者（Sophists）”以辩术闻名。
    *   **记忆技巧**：联想 “sophisticated 设备=技术复杂、见多识广”。
    *   **高频词（3个）**：
        *   **sophisticated** = soph(智慧)+ist(人/术)+ic(形容)+ated(过去分词形容) → 老练的；复杂精密的 /səˈfɪstɪkeɪtɪd/
        *   **sophistry** = soph(辩术)+istry(行为/学问) → 诡辩 /ˈsɒfɪstri/
        *   **philosophy** = philo(爱)+sophy(智慧) → 哲学 /fɪˈlɒsəfi/
    *   **组合逻辑解释**：
        *   sophisticated：智慧/技巧+形容 → 经历与技术加持而“见多识广/复杂精密” → 老练的；精密的
        *   sophistry：智慧/辩术+行为 → 玩弄技巧的辩论 → 诡辩
        *   philosophy：爱+智慧 → 以热爱追求智慧 → 哲学
    *   **词缀总结**：-ist/-istry(从业/术)；-ic/-ated(形容/形动)；与 philo-/sophy 组合表达“爱智慧”的学问。
