## 第三天背词根

- **uni**
  - 词根：uni
  - 核心含义：一
  - 古人发明此词根的意思：源自拉丁语 *unus*，意为“单一的、一个”
  - 记忆技巧：谐音“有你”，想象“有你一个就足够”
  - 高频词（3个）：
    - **uniform** = uni(一)+form(形式) → 统一的；制服
    - **unique** = uni(一)+que(形容词后缀) → 独特的，唯一的
    - **unite** = uni(一)+te(动词后缀) → 联合，团结

***
- **bi**
  - 词根：bi
  - 核心含义：二
  - 古人发明此词根的意思：源自拉丁语 *bis* 或希腊语 *bios*，意为“两次、两个”
  - 记忆技巧：谐音“必”，想象“两件事必同时完成”
  - 高频词（3个）：
    - **bicycle** = bi(二)+cycle(轮子) → 自行车
    - **bilingual** = bi(二)+lingu(语言)+al(形容词后缀) → 双语的
    - **biannual** = bi(二)+ann(年)+ual(形容词后缀) → 一年两次的

***
- **tri**
  - 词根：tri
  - 核心含义：三
  - 古人发明此词根的意思：源自希腊语 *treis* 或拉丁语 *tres*，意为“三个”
  - 记忆技巧：联想“三角形（triangle）有三个边”
  - 高频词（3个）：
    - **triangle** = tri(三)+angle(角) → 三角形
    - **trinity** = tri(三)+nity(名词后缀) → 三位一体
    - **triple** = tri(三)+ple(折叠) → 三倍的；使增至三倍

***
- **quad**
  - 词根：quad
  - 核心含义：四
  - 古人发明此词根的意思：源自拉丁语 *quattuor*，意为“四个”
  - 记忆技巧：谐音“阔的”，想象“四边形（quadrangle）面积阔大”
  - 高频词（3个）：
    - **quadrangle** = quad(四)+angle(角) → 四边形；四合院
    - **quadruple** = quad(四)+ple(折叠) → 四倍的；使增至四倍
    - **quadrant** = quad(四)+rant(名词后缀) → 象限；四分之一圆

***
- **cent**
  - 词根：cent
  - 核心含义：百
  - 古人发明此词根的意思：源自拉丁语 *centum*，意为“一百”
  - 记忆技巧：联想“人民币分（cent），一元等于一百 cent”
  - 高频词（3个）：
    - **century** = cent(百)+ury(名词后缀) → 世纪，百年
    - **percent** = per(每)+cent(百) → 百分比，百分率
    - **centimeter** = cent(百)+meter(米) → 厘米

***
- **dec**
  - 词根：dec
  - 核心含义：十
  - 古人发明此词根的意思：源自拉丁语 *decem*，意为“十”
  - 记忆技巧：谐音“带可”，想象“十年（decade）带可以完成很多事”
  - 高频词（3个）：
    - **decade** = dec(十)+ade(名词后缀) → 十年
    - **decimal** = dec(十)+imal(形容词后缀) → 十进制的；小数
    - **decagon** = dec(十)+agon(角) → 十边形

***
- **milli**
  - 词根：milli
  - 核心含义：千
  - 古人发明此词根的意思：源自拉丁语 *mille*，意为“一千”
  - 记忆技巧：联想“毫米（millimeter），一米等于一千毫米”
  - 高频词（3个）：
    - **millimeter** = milli(千)+meter(米) → 毫米
    - **millennium** = milli(千)+ennium(年) → 千年，千禧年
    - **milligram** = milli(千)+gram(克) → 毫克

***
- **multi**
  - 词根：multi
  - 核心含义：多
  - 古人发明此词根的意思：源自拉丁语 *multus*，意为“许多的、大量的”
  - 记忆技巧：谐音“猫踢”，想象“猫踢翻多个杯子”
  - 高频词（3个）：
    - **multiply** = multi(多)+ply(折叠) → 乘；繁殖
    - **multimedia** = multi(多)+media(媒体) → 多媒体
    - **multiple** = multi(多)+ple(折叠) → 多重的；倍数

***
- **poly**
  - 词根：poly
  - 核心含义：多
  - 古人发明此词根的意思：源自希腊语 *polys*，意为“许多的、众多的”
  - 记忆技巧：谐音“跑累”，想象“跑过很多路会累”
  - 高频词（3个）：
    - **polygon** = poly(多)+gon(角) → 多边形
    - **polyglot** = poly(多)+glot(语言) → 通晓多种语言的人；多语言的
    - **polymer** = poly(多)+mer(部分) → 聚合物

***
- **semi**
  - 词根：semi
  - 核心含义：半
  - 古人发明此词根的意思：源自拉丁语 *semis*，意为“一半”
  - 记忆技巧：谐音“赛米”，想象“比赛只进行了一半（semi-）”
  - 高频词（3个）：
    - **semicircle** = semi(半)+circle(圆) → 半圆
    - **semifinal** = semi(半)+final(决赛) → 半决赛
    - **semiconductor** = semi(半)+conductor(导体) → 半导体
