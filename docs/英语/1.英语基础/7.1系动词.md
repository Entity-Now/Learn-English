# 📘 一、什么是系动词（Linking Verb）

**系动词 = 不表示动作，而是连接“主语”和“表语”的动词。**  
它的功能是：“给主语贴上标签”。

结构：

    主语 + 系动词 + 表语

例：

*   He **is** smart.（他很聪明）
*   The milk **became** sour.（牛奶变酸了）
*   She **feels** tired.（她感觉累）

👉 **核心：系动词不表示动作，只表达状态、性质、感受、变化。**

***

# 📘 二、系动词的完整分类（英语语法中最权威的分类）

系动词共分 **7 大类**：

***

# ① **be 系动词（最核心）**

**be：am / is / are / was / were / be / been / being**

功能：

*   描述 **身份**（He is a teacher.）
*   描述 **性质**（She is kind.）
*   描述 **状态**（The door is open.）
*   描述 **位置**（The bag is on the table.）

**be 是最常见、最典型、最纯粹的系动词。**

***

# ② **表示“感官” 的系动词（5 个）**

这些动词既能做动作动词，也能当系动词（重点！）

| 系动词   | 当“感官”时含义 | 示例                     |
| ----- | -------- | ---------------------- |
| look  | 看起来      | You look tired.        |
| sound | 听起来      | That sounds great.     |
| smell | 闻起来      | The soup smells good.  |
| taste | 尝起来      | The cake tastes sweet. |
| feel  | 感觉       | I feel cold.           |

当它们 + 表语 → 是系动词  
当它们 + 宾语 → 是动作动词

例：

*   The soup **smells good**.（系）
*   She **smelled the flowers**.（动作）

***

# ③ **表示“变化、变成”的系动词（8 个）**

| 系动词    | 含义            |
| ------ | ------------- |
| become | 成为            |
| get    | 变得            |
| turn   | 变成            |
| grow   | 变得（逐渐）        |
| go     | 变得（不好方向）      |
| come   | 变成（常用于“出乎意料”） |
| fall   | 变得（通常突然）      |
| run    | 变得（少见）        |

例：

*   She **became** angry.
*   The sky **turned** red.
*   He **grew** old.
*   The milk **went** bad.

⚠ go + 表语 常表示“变坏”

*   go bad
*   go crazy
*   go wrong
*   go missing

***

# ④ **表示“持续、保持”的系动词（5 个）**

| 系动词       | 含义           |
| --------- | ------------ |
| keep      | 保持           |
| stay      | 保持           |
| remain    | 仍然           |
| continue  | 继续保持         |
| lie/stand | 保持在…状态（比喻意义） |

例：

*   He **remained** silent.
*   She **stayed** calm.
*   The door **continued** open.

***

# ⑤ **表示“状态存在”的系动词（4 个）**

| 系动词      | 含义    |
| -------- | ----- |
| seem     | 似乎    |
| appear   | 似乎/显得 |
| prove    | 结果是   |
| turn out | 结果是   |

例：

*   He **seems** happy.
*   It **proved** impossible.
*   The plan **turned out** great.

***

# ⑥ **表示“状况呈现”的系动词（2 个）**

| 系动词    | 含义  |
| ------ | --- |
| look   | 显得  |
| appear | 显现为 |

（与感官系动词重叠，但用法不同）

例：

*   She **appears** nervous.
*   He **looks** confident.

***

# ⑦ **半系动词（半联系动词）**

这些词不属于典型系动词，但常用于“带补语的结构”：

*   make
*   leave
*   find
*   consider
*   elect

例：

*   They **made** him angry.
*   We **found** the story interesting.

它们属于 **V + O + OC** 结构，不是真系动词，但和系动词接近。

***

# 📘 三、系动词 + 表语（必须掌握的黄金规则）

表语的形式有 7 种：

| 表语形式    | 示例                       |
| ------- | ------------------------ |
| 名词      | He is a doctor.          |
| 形容词     | She is smart.            |
| 数词      | The price is five.       |
| 代词      | It is mine.              |
| 介词短语    | The book is on the desk. |
| 不定式（少见） | His dream is to travel.  |
| 分词      | The door is closed.      |

最常见的是：  
✔ 形容词  
✔ 名词  
✔ 介词短语

***

# 📘 四、如何判断一个动词是不是系动词？（超关键！）

✔ 看后面是否接 “表语（形容词/名词）”。  
✔ 是否能替换成 be（部分系动词可以“试着替换”）。

判断技巧：

    看起来 / 听起来 / 闻起来 / 尝起来 / 感觉起来
    = 系动词

例：

*   He looks **happy**.（系）
*   She looks **at him**.（动作）

问：为什么不看“happy”？  
因为 happy 是形容词 → 系动词

***

# 📘 五、系动词 vs. 动作动词（最重要的对比）

| 动词    | 系动词用法                      | 动作动词用法                     |
| ----- | -------------------------- | -------------------------- |
| smell | The soup smells good.（闻起来） | I smelled the flower.（我闻了） |
| taste | The food tastes bad.（尝起来）  | She tasted the cake.（她尝了）  |
| feel  | I feel cold.（感到）           | I felt the cat.（摸到）        |
| look  | You look tired.（看起来）       | He looked at me.（看）        |

口诀：

    接形容词 → 系动词
    接名词做宾语 → 动作动词

***

# 📘 六、系动词 13 大易混误区

1.  **系动词后不能接副词（只能接形容词）**
    *   ❌ He is happily.
    *   ✔ He is happy.

2.  smell/taste/look 误用

3.  动词 seem 永远是系动词

4.  become 是动作词还是状态？

5.  go bad 等固定表达

6.  appear vs. seem

7.  keep 与 keep doing 区别

8.  stay（保持）与 remain（仍然）

9.  prove/turn out 用法

10. 系动词不能表示动作

11. 表语不能用动词原形

12. 表语与主语不一致（名词单复数问题）

13. 表语中的形容词必须符合逻辑（不能主观乱用）
