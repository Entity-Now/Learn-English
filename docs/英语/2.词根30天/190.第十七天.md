### 今日词根学习（第五天：10个随机常用词根）

#### 1. 词根: nephr + 同义/同源词根（nephro） + /ˈnɛfr/
2. 核心含义：肾  
3. 词源背景：来自希腊语“nephros”（肾），现代医学中用于肾脏相关专业词汇  
4. 记忆技巧：想象“nephr”像“nephron”（肾单位），肾脏里无数小“工厂”在过滤血液  
5. 3个高频词：
   - nephrology /nɪˈfrɑːlədʒi/：nephr（肾） + ology（研究），指“肾脏病学”，专门研究“肾”
   - nephron /ˈnɛfrɑːn/：nephr（肾） + on（单位），意思是“肾单位”，肾脏的基本功能单位
   - nephritis /nɪˈfraɪtɪs/：nephr（肾） + itis（炎症），指“肾炎”，肾脏发炎
6. 同源词根的常见音变规律：nephr → nephro（前缀学术形式，常加o）  
7. 高频词中出现的词缀总结：ology /ˈɑːlədʒi/（研究）；on /ɑːn/（小单位）；itis /ˈaɪtɪs/（炎症）

#### 2. 词根: system + 同义/同源词根（syst/syste） + /ˈsɪstəm/
2. 核心含义：组合、系统、整体  
3. 词源背景：来自希腊语“systema”（由多部分组成的整体），现代用于各种系统概念  
4. 记忆技巧：把“system”想象成一台电脑的“系统”，所有部件“组合”在一起工作  
5. 3个高频词：
   - system /ˈsɪstəm/：直接使用词根，意思是“系统”，由多个部分“组合”而成
   - systematic /ˌsɪstəˈmætɪk/：system（系统） + atic（形容词），指“有系统的”，按系统方式进行
   - ecosystem /ˈiːkoʊsɪstəm/：eco（生态） + system（系统），指“生态系统”，生物与环境的“整体组合”
6. 同源词根的常见音变规律：system基本不变，常直接作为词根或前缀使用  
7. 高频词中出现的词缀总结：atic /ˈætɪk/（形容词）；eco /ˈiːkoʊ/（生态）

#### 3. 词根: popul + 同义/同源词根（popul/pop） + /ˈpɑːpjəl/
2. 核心含义：人民、民众  
3. 词源背景：来自拉丁语“populus”（人民、民众），用于人口、流行等词汇  
4. 记忆技巧：想到“popular”=受“民众”喜欢的东西，“population”就是全部“人民”数量  
5. 3个高频词：
   - population /ˌpɑːpjəˈleɪʃn/：popul（人民） + ation（结果），指“人口”，全体“人民”的总数
   - popular /ˈpɑːpjələr/：popul（人民） + ar（形容词），意思是“流行的/受欢迎的”，受“人民”喜爱的
   - populist /ˈpɑːpjəlɪst/：popul（人民） + ist（人/主义），指“民粹主义者”，主张一切以“人民”为先
6. 同源词根的常见音变规律：popul → pop（简化形式，如popular中的pop）  
7. 高频词中出现的词缀总结：ation /eɪʃn/（结果名词）；ar /ər/（形容词）；ist /ɪst/（人/主义）

#### 4. 词根: crim + 同义/同源词根（crime/crimin） + /krɪm/
2. 核心含义：罪、犯罪  
3. 词源背景：来自拉丁语“crimen”（指控、罪行），用于法律与犯罪相关词汇  
4. 记忆技巧：把“crime”想象成“criminal”戴着手铐的样子，代表“罪”  
5. 3个高频词：
   - crime /kraɪm/：直接使用词根，意思是“犯罪/罪行”，就是“罪”本身
   - criminal /ˈkrɪmɪnl/：crim（罪） + inal（形容词/人），指“犯罪的/罪犯”，与“罪”相关的人或性质
   - discriminate /dɪˈskrɪmɪneɪt/：dis（分开） + crimin（分辨/罪，原义“区分”），指“歧视”，把人“分开对待”
6. 同源词根的常见音变规律：crim → crimin（形容词/加强形式，加in）  
7. 高频词中出现的词缀总结：inal /ɪnl/（形容词/人）；dis /dɪs/（分开）；ate /eɪt/（动词）

#### 5. 词根: sacchar + 同义/同源词根（saccharo） + /ˈsækər/
2. 核心含义：糖  
3. 词源背景：来自希腊语“sakcharon”（糖），用于甜味、糖类相关词汇  
4. 记忆技巧：想到“saccharine”就是“人工糖精”，味道超级“甜”像糖  
5. 3个高频词：
   - saccharine /ˈsækəriːn/：sacchar（糖） + ine（形容词），指“糖精/甜得发腻的”，极甜的
   - polysaccharide /ˌpɑːlɪˈsækəraɪd/：poly（多） + sacchar（糖） + ide（化合物），指“多糖”，由多个“糖”单位组成
   - saccharose /ˈsækəroʊs/：sacchar（糖） + ose（糖类后缀），指“蔗糖”，普通食糖的学名
6. 同源词根的常见音变规律：sacchar → saccharo（前缀形式，加o）  
7. 高频词中出现的词缀总结：ine /iːn/（形容词）；poly /ˈpɑːli/（多）；ide /aɪd/（化合物）；ose /oʊs/（糖类）

#### 6. 词根: possess + 同义/同源词根（possess/posses） + /pəˈzɛs/
2. 核心含义：拥有、占有  
3. 词源背景：来自拉丁语“possidere”（占有、拥有），由potis（能）+sedere（坐）演变，意为“坐在上面控制”  
4. 记忆技巧：想象一个人“坐在”财产上说“这是我的”，就是“拥有”  
5. 3个高频词：
   - possess /pəˈzɛs/：直接使用词根，意思是“拥有/占有”，把东西控制在自己手里
   - possession /pəˈzɛʃn/：possess（拥有） + ion（结果），指“所有物/占有”，拥有的东西
   - possessive /pəˈzɛsɪv/：possess（拥有） + ive（形容词），指“占有欲强的/所有格的”，想“占有”的
6. 同源词根的常见音变规律：possess基本稳定，ss双写是拉丁过去分词特征  
7. 高频词中出现的词缀总结：ion /ən/（结果名词）；ive /ɪv/（形容词）

#### 7. 词根: fort + 同义/同源词根（forti/force） + /fɔːrt/
2. 核心含义：强、力量  
3. 词源背景：来自拉丁语“fortis”（强壮的），用于力量、堡垒等词汇  
4. 记忆技巧：想到“fort”像“fortress”（堡垒），坚固=“强”  
5. 3个高频词：
   - fortune /ˈfɔːrtʃən/：fort（强/运气，原义好运带来的力量） + une，意思是“命运/财富”，强的运气
   - fortify /ˈfɔːrtɪfaɪ/：fort（强） + ify（使…），指“加强/加固”，使某物变得“更强”
   - forte /fɔːrt/：直接使用词根，意思是“强项/擅长”，个人的“强”处
6. 同源词根的常见音变规律：fort → forti（形容词/加强形式，加i）  
7. 高频词中出现的词缀总结：une /ən/（名词）；ify /ɪfaɪ/（使…动词）

#### 8. 词根: fac/fact/fic + 同义/同源词根（fac/fact/fic） + /fæk/
2. 核心含义：做、制造  
3. 词源背景：来自拉丁语“facere”（做、制造），fic是派生形式，广泛用于动作与结果  
4. 记忆技巧：把“factory”想象成“制造”东西的地方，“fact”是已经“做”出来的事实  
5. 3个高频词：
   - factory /ˈfæktəri/：fact（做） + ory（地方），指“工厂”，做东西的地方
   - fact /fækt/：直接使用词根，意思是“事实”，已经“做成/完成”的事情
   - efficient /ɪˈfɪʃnt/：e（出） + fic（做） + ient（形容词），指“高效的”，做事“效果好”
6. 同源词根的常见音变规律：fac（动词）；fact（过去/名词）；fic（形容词/复合）  
7. 高频词中出现的词缀总结：ory /əri/（地方）；ient /iənt/（形容词）；e /ɪ/（出/加强）

#### 9. 词根: fid + 同义/同源词根（fide/fid） + /fɪd/
2. 核心含义：信任、忠实  
3. 词源背景：来自拉丁语“fides”（信任、忠诚），用于信心、信仰等  
4. 记忆技巧：想到“confidence”=“有信心”（con + fid），就是充分“信任”自己  
5. 3个高频词：
   - confidence /ˈkɑːnfɪdəns/：con（加强） + fid（信任） + ence，意思是“信心/信任”，强烈的“信任”
   - fidelity /fɪˈdɛləti/：fid（信任） + elity（名词），指“忠实/保真度”，保持“忠诚”
   - diffident /ˈdɪfɪdənt/：dif（不） + fid（信任） + ent，意思是“缺乏自信的”，不“信任”自己
6. 同源词根的常见音变规律：fid → fide（古典形式，如bona fide）  
7. 高频词中出现的词缀总结：con /kən/（加强）；ence /əns/（名词）；elity /ələti/（名词）；dif /dɪf/（否定）；ent /ənt/（形容词）

#### 10. 词根: pos/posit + 同义/同源词根（pose/posit） + /pɑːz/
2. 核心含义：放置、姿势  
3. 词源背景：来自拉丁语“ponere”（放置），posit是过去分词形式  
4. 记忆技巧：把“position”想象成你“放置”身体的位置，“propose”是“向前放置”一个想法  
5. 3个高频词：
   - position /pəˈzɪʃn/：posit（放置） + ion，意思是“位置/职位”，被“放置”的地方或角色
   - compose /kəmˈpoʊz/：com（一起） + pose（放置），指“组成/作曲”，把东西“一起放置”
   - deposit /dɪˈpɑːzɪt/：de（下） + posit（放置），意思是“存款/沉淀”，把钱“放下去”
6. 同源词根的常见音变规律：pos（动词形式）；posit（名词/过去形式，添加it）  
7. 高频词中出现的词缀总结：ion /ən/（结果）；com /kəm/（一起）；de /dɪ/（下）
