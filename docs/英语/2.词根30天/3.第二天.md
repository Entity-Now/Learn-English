### 英语词根背诵清单

***

*   #### **spect**

    *   **词根**：spect
    *   **核心含义**：看、观察
    *   **词源背景**：来自拉丁语 *specere*（看、审视），古罗马人用它指军队侦查、元老院巡视等正式“观察”行为
    *   **记忆技巧**：联想“inspect 检查要仔细看”
    *   **高频词（3个）**：
        *   **inspect** = in(进入)+spect(看) → 检查 /ɪnˈspekt/
        *   **respect** = re(再次)+spect(看) → 尊重 /rɪˈspekt/
        *   **prospect** = pro(向前)+spect(看) → 前景 /ˈprɒspekt/
    *   **组合逻辑解释**：
        *   inspect：进入+看 → 看到内部 → 检查
        *   respect：再次+看 → 再三看重 → 尊重
        *   prospect：向前+看 → 看到未来 → 前景
    *   **同源词根及音变规律**：
        *   同源：spect ↔ spec（看）
        *   音变：ct ↔ c（词尾弱化）
    *   **词缀总结**：
        *   in-(进入)
        *   re-(再次)
        *   pro-(向前)

***

*   #### **vid / vis**

    *   **词根**：vid / vis
    *   **核心含义**：看、视觉
    *   **词源背景**：来自拉丁语 *videre*（看），古罗马用于“正式目击、证实”场景
    *   **记忆技巧**：联想 video 是“能看到的东西”
    *   **高频词（3个）**：
        *   **video** = vid(看)+eo → 视频 /ˈvɪdiəʊ/
        *   **vision** = vis(看)+ion → 视觉 /ˈvɪʒn/
        *   **visible** = vis(看)+ible(能…) → 可见的 /ˈvɪzəbl/
    *   **组合逻辑解释**：
        *   video：看 → 能看的内容 → 视频
        *   vision：看+名词 → 视觉能力 → 视力
        *   visible：看+能 → 能被看到 → 可见
    *   **同源词根及音变规律**：
        *   同源：vid ↔ vis
        *   音变：d ↔ s（辅音交替）
    *   **词缀总结**：
        *   -ion(名词)
        *   -ible(能…)

***

*   #### **aud**

    *   **词根**：aud
    *   **核心含义**：听
    *   **词源背景**：来自拉丁语 *audire*（听见），古代用于法庭审讯、公开演说等“认真倾听”的场景
    *   **记忆技巧**：联想 audio 音频要“听”
    *   **高频词（3个）**：
        *   **audio** = aud(听)+io → 音频 /ˈɔːdiəʊ/
        *   **audience** = aud(听)+ence(人群) → 听众 /ˈɔːdiəns/
        *   **audit** = aud(听)+it → 审计 /ˈɔːdɪt/
    *   **组合逻辑解释**：
        *   audio：听 → 与声音相关 → 音频
        *   audience：听+人 → 听讲的人群 → 听众
        *   audit：听 → 最初通过“听账”检查 → 审计
    *   **同源词根及音变规律**：
        *   同源：aud ↔ aur（耳）
        *   音变：d ↔ r（元音环境变化）
    *   **词缀总结**：
        *   -ence(人群)
        *   -it(行为)

***

*   #### **phon**

    *   **词根**：phon
    *   **核心含义**：声音
    *   **词源背景**：源自希腊语 *phōnē*（声音），古代用于描述人声、乐声等自然发声
    *   **记忆技巧**：联想 phone 电话传递声音
    *   **高频词（3个）**：
        *   **telephone** = tele(远)+phone(声音) → 电话 /ˈtelɪfəʊn/
        *   **symphony** = sym(一起)+phony(声音) → 交响乐 /ˈsɪmfəni/
        *   **microphone** = micro(小)+phone(声音) → 麦克风 /ˈmaɪkrəfəʊn/
    *   **组合逻辑解释**：
        *   telephone：远+声音 → 听到远方声音 → 电话
        *   symphony：一起+声音 → 众声合一 → 交响乐
        *   microphone：小+声音 → 捕捉微小声音 → 麦克风
    *   **同源词根及音变规律**：
        *   同源：phon ↔ phono
        *   音变：ph ↔ f（发音相同）
    *   **词缀总结**：
        *   tele-(远)
        *   sym-(一起)
        *   micro-(小)

*   #### **dict**

    *   **词根**：dict
    *   **核心含义**：说
    *   **词源背景**：来自拉丁语 *dicere*（说、宣告），用于法官宣判、神谕宣布等正式语言场景
    *   **记忆技巧**：联想 dictionary 字典记录“说出来的话”
    *   **高频词（3个）**：
        *   **predict** = pre(前)+dict(说) → 预言 /prɪˈdɪkt/
        *   **dictionary** = dict(说)+ionary(集合) → 字典 /ˈdɪkʃənri/
        *   **indicate** = in(向内)+dic(说)+ate → 表明 /ˈɪndɪkeɪt/
    *   **组合逻辑解释**：
        *   predict：前+说 → 提前说 → 预言
        *   dictionary：说+集合 → 收录所有词语 → 字典
        *   indicate：向内+说 → 指出、告诉 → 表明
    *   **同源词根及音变规律**：
        *   同源：dict ↔ dic
        *   音变：t ↔ c（词尾变化）
    *   **词缀总结**：
        *   pre-(前)
        *   -ionary(集合)
        *   -ate(动词)

***

*   #### **voc / vok**

    *   **词根**：voc / vok
    *   **核心含义**：叫、喊
    *   **词源背景**：来自拉丁语 *vocare*（呼喊），古罗马法庭常靠“叫声”召集群众
    *   **记忆技巧**：联想 vocal 声音的
    *   **高频词（3个）**：
        *   **vocal** = voc(叫)+al(形容词) → 声音的 /ˈvəʊkl/
        *   **invoke** = in(向内)+vok(叫) → 祈求；援引 /ɪnˈvəʊk/
        *   **advocate** = ad(向)+voc(叫)+ate → 支持 /ˈædvəkeɪt/
    *   **组合逻辑解释**：
        *   vocal：叫+形容词 → 与发声相关 → 声音的
        *   invoke：向内+叫 → 向神灵呼求 → 祈求
        *   advocate：向+叫 → 为某事“发声” → 支持
    *   **同源词根及音变规律**：
        *   同源：voc ↔ vok
        *   音变：c ↔ k（发音一致）
    *   **词缀总结**：
        *   in-(向内)
        *   ad-(向)
        *   -al、-ate

*   #### **log**

    *   **词根**：log
    *   **核心含义**：说、言论
    *   **词源背景**：来自希腊语 *logos*（言辞、理性），在古希腊哲学中指“通过语言表达的思想”
    *   **记忆技巧**：联想 logic 逻辑=用语言表达的推理
    *   **高频词（3个）**：
        *   **logic** = log(说)+ic(学科) → 逻辑 /ˈlɒdʒɪk/
        *   **dialogue** = dia(对话)+logue(说) → 对话 /ˈdaɪəlɒɡ/
        *   **apology** = apo(离)+logy(说) → 道歉 /əˈpɒlədʒi/
    *   **组合逻辑解释**：
        *   logic：说+学科 → 用语言推理 → 逻辑
        *   dialogue：对着+说 → 两人互说 → 对话
        *   apology：离+说 → 离题解释 → 后引申为道歉
    *   **同源词根及音变规律**：
        *   同源：log ↔ logue
        *   音变：os ↔ ue（希腊→英语）
    *   **词缀总结**：
        *   dia-(穿过/相对)
        *   apo-(离开)
        *   -ic、-logy

*   #### **nounce**

    *   **词根**：nounce
    *   **核心含义**：宣布、说出
    *   **词源背景**：来自拉丁语 *nuntiare*（宣布、传达），用于传信官公开发布消息
    *   **记忆技巧**：联想 announce 公布
    *   **高频词（3个）**：
        *   **announce** = an(向)+nounce(宣布) → 宣布 /əˈnaʊns/
        *   **pronounce** = pro(向前)+nounce(说) → 发音／宣布 /prəˈnaʊns/
        *   **denounce** = de(向下)+nounce(说) → 谴责 /dɪˈnaʊns/
    *   **组合逻辑解释**：
        *   announce：向+宣布 → 对外宣布
        *   pronounce：向前+说 → 清楚地说出来 → 发音
        *   denounce：向下+说 → 指责 → 谴责
    *   **同源词根及音变规律**：
        *   同源：nounce ↔ nunci（使者）
        *   音变：ou ↔ un（元音变化）
    *   **词缀总结**：
        *   an-(向)
        *   pro-(向前)
        *   de-(向下)

*   #### **claim**

    *   **词根**：claim
    *   **核心含义**：呼喊、宣称
    *   **词源背景**：来自拉丁语 *clamare*（呼喊），古代公共场所靠大声呼喊引起注意
    *   **记忆技巧**：联想 exclaim 大喊
    *   **高频词（3个）**：
        *   **proclaim** = pro(向前)+claim(喊) → 宣告 /prəˈkleɪm/
        *   **exclaim** = ex(向外)+claim(喊) → 呼喊 /ɪksˈkleɪm/
        *   **acclaim** = ac(向)+claim(喊) → 称赞 /əˈkleɪm/
    *   **组合逻辑解释**：
        *   proclaim：向前+喊 → 对外宣布
        *   exclaim：向外+喊 → 激动地喊出
        *   acclaim：向+喊 → 大声叫好 → 赞扬
    *   **同源词根及音变规律**：
        *   同源：claim ↔ clam
        *   音变：i ↔ a（元音）
    *   **词缀总结**：
        *   pro-(向前)
        *   ex-(向外)
        *   ac-(向)

*   #### **graph**

    *   **词根**：graph
    *   **核心含义**：写、记录
    *   **词源背景**：来自希腊语 *graphein*（写、刻），古代用于石碑刻字、地图绘制
    *   **记忆技巧**：联想 graph 图表“记录信息”
    *   **高频词（3个）**：
        *   **autograph** = auto(自己)+graph(写) → 亲笔签名 /ˈɔːtəɡrɑːf/
        *   **telegraph** = tele(远)+graph(写) → 电报 /ˈtelɪɡrɑːf/
        *   **paragraph** = para(旁)+graph(写) → 段落 /ˈpærəɡrɑːf/
    *   **组合逻辑解释**：
        *   autograph：自己+写 → 自己写的名字 → 签名
        *   telegraph：远+写 → 向远方发送文字 → 电报
        *   paragraph：旁+写 → 写在旁边的分段 → 段落
    *   **同源词根及音变规律**：
        *   同源：graph ↔ gram（写）
        *   音变：ph ↔ m（词尾变化）
    *   **词缀总结**：
        *   auto-(自己)
        *   tele-(远)
        *   para-(旁)
