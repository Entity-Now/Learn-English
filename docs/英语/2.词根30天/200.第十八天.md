### 今日词根学习（第六天：10个随机常用词根）

#### 1. 词根: vor + 同义/同源词根（vora/vorac） + /vɔːr/
2. 核心含义：吃、吞食  
3. 词源背景：来自拉丁语“vorare”（吞噬、贪吃），常用于动物食性与贪婪相关词汇  
4. 记忆技巧：想象一个“voracious”食客像鲨鱼一样“vor”地大口吞食一切  
5. 3个高频词：
   - devour /dɪˈvaʊər/：de（彻底） + vor（吃），指“狼吞虎咽/吞没”，彻底地“吃”掉
   - carnivore /ˈkɑːrnɪvɔːr/：carni（肉） + vor（吃） + e，指“食肉动物”，专门“吃”肉的生物
   - herbivore /ˈhɜːrbɪvɔːr/：herbi（草） + vor（吃） + e，指“食草动物”，以“草”为食
6. 同源词根的常见音变规律：vor → vora（形容词形式，如voracious）；vorac（加强形式）  
7. 高频词中出现的词缀总结：de /dɪ/（彻底）；carni /ˈkɑːrni/（肉）；herbi /ˈhɜːrbi/（草）；e（连接）

#### 2. 词根: path + 同义/同源词根（pathy/pathos） + /pæθ/
2. 核心含义：感觉、感情、病  
3. 词源背景：来自希腊语“pathos”（痛苦、感情），既表示情感也表示疾病  
4. 记忆技巧：想到“sympathy”=“共同感觉”别人的痛苦，“pathology”则是“研究病”  
5. 3个高频词：
   - sympathy /ˈsɪmpəθi/：sym（共同） + path（感觉） + y，指“同情”，与别人“共同感受”
   - pathology /pəˈθɑːlədʒi/：path（病） + ology（研究），指“病理学”，研究“疾病”
   - apathetic /ˌæpəˈθetɪk/：a（无） + path（感觉） + etic，指“冷漠的/无动于衷的”，没有“感觉”
6. 同源词根的常见音变规律：path → pathy（名词形式，加y）；pathos（感情原形）  
7. 高频词中出现的词缀总结：sym /sɪm/（共同）；y /i/（名词）；ology /ˈɑːlədʒi/（研究）；a /ə/（无）；etic /ˈetɪk/（形容词）

#### 3. 词根: sta/stat + 同义/同源词根（sta/stat/stas） + /steɪ/
2. 核心含义：站立、状态、稳定  
3. 词源背景：来自拉丁语“stare”（站立），用于状态、位置、稳定相关词汇  
4. 记忆技巧：把“state”想象成“站立”的国家形态，“stable”就是“站得稳”  
5. 3个高频词：
   - state /steɪt/：直接使用词根，意思是“状态/国家”，某种“站立”的形式
   - stable /ˈsteɪbl/：sta（站立） + ble（能…的），指“稳定的”，能够“站得住”的
   - status /ˈsteɪtəs/：stat（站立） + us，指“地位/状态”，所“站”的位置
6. 同源词根的常见音变规律：sta（动词/基础）；stat（名词常见）；stas（少见，如static）  
7. 高频词中出现的词缀总结：ble /bl/（能…的）；us /əs/（名词）

#### 4. 词根: nov + 同义/同源词根（nova/novi） + /nɑːv/
2. 核心含义：新  
3. 词源背景：来自拉丁语“novus”（新的），用于创新、更新等词汇  
4. 记忆技巧：想到“novel”是一本“新”书，“innovation”就是“带来新的东西”  
5. 3个高频词：
   - novel /ˈnɑːvl/：nov（新） + el，指“新颖的/小说”，新奇的故事
   - innovation /ˌɪnəˈveɪʃn/：in（进入） + nov（新） + ation，指“创新”，使新的东西“进入”世界
   - renovate /ˈrenəveɪt/：re（重新） + nov（新） + ate，指“翻新”，重新使之“变新”
6. 同源词根的常见音变规律：nov → nova（形容词/名词，如nova星）；novi（少见）  
7. 高频词中出现的词缀总结：el /əl/（形容词/名词）；in /ɪn/（进入）；ation /eɪʃn/（结果）；re /ri/（重新）；ate /eɪt/（动词）

#### 5. 词根: mob/mot/mov + 同义/同源词根（mobi/motu/mov） + /moʊb/ /moʊt/ /muːv/
2. 核心含义：动、移动  
3. 词源背景：来自拉丁语“movere”（移动），用于运动、汽车等词汇  
4. 记忆技巧：把“move”直接想到“移动”，“mobile”就是“可移动的”  
5. 3个高频词：
   - mobile /ˈmoʊbl/：mob（动） + ile，指“移动的/手机”，能“动”的东西
   - motion /ˈmoʊʃn/：mot（动） + ion，指“运动/动作”，移动的过程
   - automobile /ˈɔːtəmoʊbiːl/：auto（自己） + mobile（动），指“汽车”，自己“动”的交通工具
6. 同源词根的常见音变规律：mob（形容词）；mot（名词）；mov（动词）  
7. 高频词中出现的词缀总结：ile /aɪl/（能…的）；ion /ən/（过程）；auto /ˈɔːtoʊ/（自己）

#### 6. 词根: struct + 同义/同源词根（struct/structur） + /strʌkt/
2. 核心含义：建造、结构  
3. 词源背景：来自拉丁语“struere”（堆积、建造），用于建筑、结构等  
4. 记忆技巧：想到“structure”就是“结构”，“construct”是“一起建造”  
5. 3个高频词：
   - structure /ˈstrʌktʃər/：struct（建造） + ure，指“结构/构造”，被“建造”出来的样子
   - construct /kənˈstrʌkt/：con（一起） + struct（建造），指“建造/构造”，一起“建造”
   - instruct /ɪnˈstrʌkt/：in（进入） + struct（建造），指“指导/教导”，把知识“建”进别人脑中
6. 同源词根的常见音变规律：struct基本稳定，常直接使用  
7. 高频词中出现的词缀总结：ure /ər/（结果）；con /kən/（一起）；in /ɪn/（进入）

#### 7. 词根: form + 同义/同源词根（forma/form） + /fɔːrm/
2. 核心含义：形状、形式、形成  
3. 词源背景：来自拉丁语“forma”（形状、外形），用于形态、格式等  
4. 记忆技巧：把“form”想象成“表格”（form），或“变形”（transform）改变形状  
5. 3个高频词：
   - form /fɔːrm/：直接使用词根，意思是“形式/形状/表格”
   - transform /trænsˈfɔːrm/：trans（穿过/改变） + form（形状），指“转变/变形”，改变“形状”
   - information /ˌɪnfərˈmeɪʃn/：in（进入） + form（形状） + ation，指“信息”，把知识“形成”进脑中
6. 同源词根的常见音变规律：form稳定，常直接作为词根  
7. 高频词中出现的词缀总结：trans /træns/（改变）；in /ɪn/（进入）；ation /eɪʃn/（结果）

#### 8. 词根: numer + 同义/同源词根（numero/numer） + /ˈnuːmər/
2. 核心含义：数字、数目  
3. 词源背景：来自拉丁语“numerus”（数目），用于数字相关词汇  
4. 记忆技巧：想到“number”就是“数字”，“numerous”是“数目多的”  
5. 3个高频词：
   - number /ˈnʌmbər/：直接使用词根，意思是“数字/数目”
   - numerous /ˈnuːmərəs/：numer（数字） + ous，指“众多的”，数目很多的
   - enumerate /ɪˈnuːməreɪt/：e（出） + numer（数字） + ate，指“列举/计数”，把数字“列出来”
6. 同源词根的常见音变规律：numer → numero（古典形式，少见）  
7. 高频词中出现的词缀总结：ous /əs/（形容词）；e /ɪ/（出）；ate /eɪt/（动词）

#### 9. 词根: leg + 同义/同源词根（legis/lect） + /lɛɡ/
2. 核心含义：法律、读、选择（多义词根）  
3. 词源背景：来自拉丁语“lex/legis”（法律）和“legere”（读、选择），有两个主要含义分支  
4. 记忆技巧：想到“legal”是“合法的”，“lecture”是“读/讲课”  
5. 3个高频词：
   - legal /ˈliːɡl/：leg（法律） + al，指“合法的/法律的”，符合“法律”的
   - lecture /ˈlɛktʃər/：lect（读） + ure，指“讲座/讲课”，老师“读”讲义给学生听
   - select /sɪˈlɛkt/：se（分开） + lect（选择），指“选择/挑选”，把好的“选”出来
6. 同源词根的常见音变规律：leg（法律）；lect（读/选择，g→ct音变）  
7. 高频词中出现的词缀总结：al /l/（形容词）；ure /ər/（结果）；se /sɪ/（分开）

#### 10. 词根: phon + 同义/同源词根（phone/phon） + /foʊn/
2. 核心含义：声音  
3. 词源背景：来自希腊语“phone”（声音），用于声音、电话等  
4. 记忆技巧：把“phone”直接想到“电话”，就是传“声音”的工具  
5. 3个高频词：
   - telephone /ˈtelɪfoʊn/：tele（远） + phon（声音），指“电话”，远距离传“声音”
   - symphony /ˈsɪmfəni/：sym（共同） + phon（声音） + y，指“交响乐”，多种乐器“共同发声”
   - phonetics /fəˈnɛtɪks/：phon（声音） + etics（学科），指“语音学”，研究“声音”的科学
6. 同源词根的常见音变规律：phon → phone（常见后缀形式）  
7. 高频词中出现的词缀总结：tele /ˈtɛli/（远）；sym /sɪm/（共同）；y /i/（名词）；etics /ˈetɪks/（学科）

### 小作文：A Day Full of Movement and New Beginnings

Today started with a stable routine, but I wanted something novel. I devoured a delicious breakfast (I'm definitely a carnivore at heart), then listened to a symphony on my telephone while getting ready. Feeling apathetic about my old habits, I decided to renovate my study space—rearranging the structure of my desk and adding new forms of organization. Later, I took my mobile phone and went out to select some fresh ingredients for dinner. The numerous choices at the market were exciting! In the evening, I attended an online lecture about phonetics, which helped me understand how sounds form words. By the end of the day, I felt more motivated and ready for new challenges.

（注：routine /ruːˈtiːn/（常规）；rearranging /ˌriːəˈreɪndʒɪŋ/（重新整理）；ingredients /ɪnˈɡriːdiənts/（原料）；motivated /ˈmoʊtɪveɪtɪd/（有动力的））

**中文总结**：这篇文章描述了作者一天的生活：原本稳定的日常让他感到厌倦，于是决定改变——大口吃早餐、听交响乐、翻新学习空间、挑选食材、参加语音学讲座，最终感到更有动力，迎接新的开始。
