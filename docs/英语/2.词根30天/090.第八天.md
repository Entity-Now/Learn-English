*   #### **form**

    *   **词根**：form
    *   **核心含义**：形状、形态
    *   **词源背景**：来自拉丁语 *forma*，指“外形、模样”，古罗马人用其描述器物的外观，也用于指社会制度的“形式”
    *   **记忆技巧**：联想“form 表格需要固定格式和形状”
    *   **高频词（3个）**：
        *   **uniform** = uni(一)+form(形状) → 制服；统一的 /ˈjuːnɪfɔːm/
        *   **transform** = trans(改变)+form(形状) → 转变 /trænsˈfɔːm/
        *   **inform** = in(进入)+form(形状) → 通知；告知 /ɪnˈfɔːm/
    *   **组合逻辑解释**：
        *   uniform：一个+形状 → 全员穿“一个样” → 制服
        *   transform：改变+形状 → 从一种形态变到另一种 → 转变
        *   inform：将形状“输入”你脑中 → 告诉你信息 → 通知
    *   **同源词根及音变规律**：
        *   同源：form ↔ figure（fig-），含“形状”
        *   音变：f ↔ ph（如 *photo*）
    *   **词缀总结**：
        *   uni-(一)
        *   trans-(穿过/改变)
        *   in-(进入)

*   #### **morph**

    *   **词根**：morph
    *   **核心含义**：形态、形状变化
    *   **词源背景**：源自希腊语 *morphē*，意为“外形、样貌”，古希腊雕塑中常用此概念描述人物形象
    *   **记忆技巧**：联想“变形金刚变形 morph”
    *   **高频词（3个）**：
        *   **amorphous** = a(无)+morph(形态)+ous(形容词) → 无定形的 /əˈmɔːfəs/
        *   **metamorphosis** = meta(变化)+morph(形态)+osis(过程) → 变形 /ˌmetəˈmɔːfəsɪs/
        *   **morphology** = morph(形态)+logy(学科) → 形态学 /mɔːˈfɒlədʒi/
    *   **组合逻辑解释**：
        *   amorphous：无+形态 → 没有固定形状 → 无定形
        *   metamorphosis：变化+形态 → 形态彻底改变 → 变态、变形
        *   morphology：形态+学科 → 研究外部结构 → 形态学
    *   **同源词根及音变规律**：
        *   同源：morph ↔ form（形状）
        *   音变：m ↔ f（部分同义替换，不严格音变）
    *   **词缀总结**：
        *   a-(无)
        *   meta-(改变)
        *   -logy(学科)

*   #### **struct**

    *   **词根**：struct
    *   **核心含义**：建造、构建
    *   **词源背景**：源自拉丁语 *struere*，原意“堆砌、搭建”，用于描述建筑工人堆置木石的动作
    *   **记忆技巧**：联想“structure 结构就是搭建出来的”
    *   **高频词（3个）**：
        *   **structure** = struct(建造)+ure(名词) → 结构 /ˈstrʌktʃə/
        *   **construct** = con(一起)+struct(建造) → 建造 /kənˈstrʌkt/
        *   **instruct** = in(进入)+struct(建造) → 指导 /ɪnˈstrʌkt/
    *   **组合逻辑解释**：
        *   structure：建造+名词 → 建造出的形态 → 结构
        *   construct：一起+建造 → 合力造 → 建造
        *   instruct：向内+建造 → 往你脑中“搭知识” → 指导
    *   **同源词根及音变规律**：
        *   同源：struct ↔ stroy（destroy 中）
        *   音变：stru- ↔ stro-
    *   **词缀总结**：
        *   con-(一起)
        *   in-(进入)
        *   -ure(名词)

*   #### **arch**

    *   **词根**：arch
    *   **核心含义**：统治；主要
    *   **词源背景**：源自希腊语 *arkhos*（统治者）和 *arkhein*（统治、开始），古希腊政治体系中指最高权力者
    *   **记忆技巧**：联想“monarch 国王是最高统治者”
    *   **高频词（3个）**：
        *   **monarch** = mono(单一)+arch(统治) → 君主 /ˈmɒnɑːk/
        *   **architect** = arch(主要)+tect(建造) → 建筑师 /ˈɑːkɪtekt/
        *   **archenemy** = arch(主要)+enemy(敌人) → 头号敌人 /ˌɑːtʃˈenəmi/
    *   **组合逻辑解释**：
        *   monarch：单一+统治 → 一人统治 → 君主
        *   architect：主要+建造 → 掌控建筑整体的人 → 建筑师
        *   archenemy：最主要+敌人 → 头号敌人
    *   **同源词根及音变规律**：
        *   同源：arch ↔ prim(第一)
        *   音变：ar- ↔ pri-（意近不一定音近）
    *   **词缀总结**：
        *   mono-(单一)
        *   -tect(建造)
        *   -enemy(敌人)

*   #### **centr**

    *   **词根**：centr
    *   **核心含义**：中心
    *   **词源背景**：来自希腊语 *kentron*（尖点、中心），古代几何中指圆规的尖端点，即“中心”
    *   **记忆技巧**：联想“central 在中心位置”
    *   **高频词（3个）**：
        *   **central** = centr(中心)+al(形容词) → 中心的 /ˈsentrəl/
        *   **concentrate** = con(一起)+centr(中心)+ate(动词) → 集中 /ˈkɒnsntreɪt/
        *   **eccentric** = ec(离开)+centr(中心)+ic(形容词) → 古怪的 /ɪkˈsentrɪk/
    *   **组合逻辑解释**：
        *   central：中心+形容词 → 位于中心 → 中心的
        *   concentrate：一起+中心 → 全部集中到一点 → 专注/集中
        *   eccentric：离开+中心 → 不在常规中心轨道 → 古怪
    *   **同源词根及音变规律**：
        *   同源：centr ↔ center
        *   音变：e ↔ a（元音替换）
    *   **词缀总结**：
        *   con-(一起)
        *   ec-(离开)
        *   -al、-ate、-ic

*   #### **cycl**

    *   **词根**：cycl
    *   **核心含义**：圆、循环
    *   **词源背景**：源自希腊语 *kyklos*，意为“圆环”，古代用于描述车轮和周期性事件
    *   **记忆技巧**：联想 bicycle 的两个“圆轮”
    *   **高频词（3个）**：
        *   **cycle** = cycl(圆)+e → 循环 /ˈsaɪkl/
        *   **bicycle** = bi(二)+cycle(圆) → 自行车 /ˈbaɪsɪkl/
        *   **recycle** = re(再)+cycle(循环) → 回收 /ˌriːˈsaɪkl/
    *   **组合逻辑解释**：
        *   cycle：圆 → 反复转圈 → 循环
        *   bicycle：二+圆 → 两个轮子 → 自行车
        *   recycle：再+循环 → 再次利用 → 回收
    *   **同源词根及音变规律**：
        *   同源：cycl ↔ circle
        *   音变：k → c（发音一致）
    *   **词缀总结**：
        *   bi-(二)
        *   re-(再)

*   #### **sphere**

    *   **词根**：sphere
    *   **核心含义**：球、球状空间
    *   **词源背景**：来自希腊语 *sphaira*（球），古希腊天文学用其指“天球”
    *   **记忆技巧**：联想“sphere 像球一样圆”
    *   **高频词（3个）**：
        *   **atmosphere** = atmo(空气)+sphere(球) → 大气层 /ˈætməsfɪə/
        *   **hemisphere** = hemi(半)+sphere(球) → 半球 /ˈhemɪsfɪə/
        *   **biosphere** = bio(生命)+sphere(球) → 生物圈 /ˈbaɪəʊsfɪə/
    *   **组合逻辑解释**：
        *   atmosphere：空气+球 → 包裹地球的一层空气 → 大气层
        *   hemisphere：半+球 → 半个地球 → 半球
        *   biosphere：生命+球 → 地球生命活动的范围 → 生物圈
    *   **同源词根及音变规律**：
        *   同源：sphere ↔ spher-
        *   音变：ai ↔ e（希腊语→英语）
    *   **词缀总结**：
        *   atmo-(空气)
        *   hemi-(半)
        *   bio-(生命)

*   #### **rect**

    *   **词根**：rect
    *   **核心含义**：直、正
    *   **词源背景**：来自拉丁语 *regere*（使笔直、指导），原用于形容王者“端正地统治”
    *   **记忆技巧**：联想“correct 改正”
    *   **高频词（3个）**：
        *   **rectangle** = rect(直)+angle(角) → 长方形 /ˈrektæŋɡl/
        *   **correct** = cor(加强)+rect(正) → 改正 /kəˈrekt/
        *   **direct** = di(向)+rect(直) → 指导；直接的 /daɪˈrekt/
    *   **组合逻辑解释**：
        *   rectangle：直+角 → 四个直角 → 长方形
        *   correct：加强+正 → 使变正 → 改正
        *   direct：向+直 → 指向直路 → 指导
    *   **同源词根及音变规律**：
        *   同源：rect ↔ reg（统治）
        *   音变：c ↔ g
    *   **词缀总结**：
        *   cor-(加强)
        *   di-(向)
        *   -angle(角)

*   #### **equ**

    *   **词根**：equ
    *   **核心含义**：相等
    *   **词源背景**：源自拉丁语 *aequus*，表示“平、均等”，古罗马法律中常用此词表达公平
    *   **记忆技巧**：联想 equal = 相等
    *   **高频词（3个）**：
        *   **equal** = equ(相等)+al(形容词) → 相等的 /ˈiːkwəl/
        *   **equation** = equ(相等)+ation(名词) → 方程式 /ɪˈkweɪʃn/
        *   **equity** = equ(平等)+ity(名词) → 公平 /ˈekwəti/
    *   **组合逻辑解释**：
        *   equal：相等+形容词 → 相同的 → 相等
        *   equation：相等+名词 → 等式 → 方程式
        *   equity：平等+名词 → 公正 → 公平
    *   **同源词根及音变规律**：
        *   同源：equ ↔ eve-（even 平坦）
        *   音变：qu ↔ v
    *   **词缀总结**：
        *   -al(形容词)
        *   -ation(名词)
        *   -ity(名词)

*   #### **orth**

    *   **词根**：orth
    *   **核心含义**：正、直
    *   **词源背景**：来自希腊语 *orthos*，意为“笔直、正确”，常用于医学描述骨骼“端正”
    *   **记忆技巧**：联想 orthodontist（牙齿矫正医生）
    *   **高频词（3个）**：
        *   **orthodox** = ortho(正)+dox(观点) → 正统的 /ˈɔːθədɒks/
        *   **orthodontist** = ortho(正)+dont(牙)+ist(人) → 正畸医生 /ˌɔːθəˈdɒntɪst/
        *   **orthopaedic** = ortho(正)+paed(儿童/教育)+ic → 矫形的 /ˌɔːθəˈpiːdɪk/
    *   **组合逻辑解释**：
        *   orthodox：正+观点 → 正确的传统思想 → 正统
        *   orthodontist：正+牙 → 让牙齿变端正的人 → 正畸医生
        *   orthopaedic：正+儿童 → 最初指矫正儿童骨骼 → 矫形
    *   **同源词根及音变规律**：
        *   同源：orth ↔ right
        *   音变：o ↔ r（概念转化）
    *   **词缀总结**：
        *   -dox(观点)
        *   -dont(牙)
        *   -ic(形容词)
