*   **词根**：**terr**
    *   **核心含义**：土地，地面
    *   **词源背景**：来自拉丁语 *terra*，古罗马人用它指“脚下的坚实土地”，与农耕、领土概念紧密相关
    *   **记忆技巧**：联想“territory（领土）一定是占据土地的”
    *   **高频词（3个）**：
        *   **territory** = terr(土地)+itory(地方) → 领土 /ˈterətəri/
        *   **terrain** = terr(土地)+ain(名词后缀) → 地形 /təˈreɪn/
        *   **terrestrial** = terr(土地)+estrial(……的) → 陆地的 /təˈrestriəl/
    *   **组合逻辑解释**：
        *   territory：土地+地方 → 属于某群体的土地 → 领土
        *   terrain：土地+名物 → 脚下的土地状态 → 地形
        *   terrestrial：土地+……的 → 生活在土地上的 → 陆地的
    *   **同源词根及音变规律**：
        *   同源：geo(地)，hum(地面)
        *   音变：t ↔ d（清浊转换）
    *   **词缀总结**：
        *   -ory（地方）
        *   -ain（名词后缀）
        *   -estrial（……的）

*   **词根**：**aer**
    *   **核心含义**：空气，气流
    *   **词源背景**：源自希腊语 *aēr*，古希腊人以此指“可感却不可见的气体”，常与呼吸、天空联系
    *   **记忆技巧**：联想 airplane（飞机）在空气中飞
    *   **高频词（3个）**：
        *   **aeroplane** = aero(空气)+plane(平面) → 飞机 /ˈeərəpleɪn/
        *   **aerobic** = aero(空气)+bic(与……有关) → 有氧的 /eəˈrəʊbɪk/
        *   **aerosol** = aero(空气)+sol(溶液) → 气溶胶 /ˈeərəʊsɒl/
    *   **组合逻辑解释**：
        *   aeroplane：空气+平面 → 在空气中“平稳滑行”的器具 → 飞机
        *   aerobic：空气+相关 → 要空气参与的活动 → 有氧
        *   aerosol：空气+溶液 → 悬浮在空气中的微滴 → 气溶胶
    *   **同源词根及音变规律**：
        *   同源：air（空气）
        *   音变：ae → ai（元音简化）
    *   **词缀总结**：
        *   -bic（与……相关）
        *   -plane（平面→飞行器）
        *   -sol（溶液）

*   **词根**：**flor**
    *   **核心含义**：花
    *   **词源背景**：来自拉丁语 *flos/floris*，古罗马人以花象征繁荣、美与祭祀
    *   **记忆技巧**：联想 florist（花店店主）
    *   **高频词（3个）**：
        *   **floral** = flor(花)+al(形容词) → 花的 /ˈflɔːrəl/
        *   **florist** = flor(花)+ist(从事者) → 花商 /ˈflɒrɪst/
        *   **flourish** = flor(花)+ish(动词化) → 兴旺 /ˈflʌrɪʃ/
    *   **组合逻辑解释**：
        *   floral：花+形容词 → 像花一样 → 花的
        *   florist：花+从事者 → 卖花的人
        *   flourish：花+动词 → 像花一样盛开 → 兴旺
    *   **同源词根及音变规律**：
        *   同源：blossom（意象相同）
        *   音变：flor ↔ flour（元音弱化）
    *   **词缀总结**：
        *   -al（……的）
        *   -ist（从事者）
        *   -ish（使……）

*   **词根**：**foli**
    *   **核心含义**：叶子
    *   **词源背景**：来自拉丁语 *folium*，古人常以叶记录、包裹、标记
    *   **记忆技巧**：联想 foliage（树叶）
    *   **高频词（3个）**：
        *   **foliage** = foli(叶)+age(集合) → 叶子总称 /ˈfəʊliɪdʒ/
        *   **portfolio** = port(携带)+foli(叶)+o → 文稿夹 /pɔːtˈfəʊliəʊ/
        *   **foliate** = foli(叶)+ate(使……) → 加叶状装饰 /ˈfəʊlieɪt/
    *   **组合逻辑解释**：
        *   foliage：叶+集合 → 所有叶子 → 树叶
        *   portfolio：携带+叶 → 带着“纸页”的夹子 → 文件夹
        *   foliate：叶+使 → 让物件呈叶子形状
    *   **同源词根及音变规律**：
        *   同源：foil（薄片→来自叶）
        *   音变：foli → foil（元音交换）
    *   **词缀总结**：
        *   -age（集合）
        *   -o（词尾组合）
        *   -ate（使……）

        *   **arborist** = arbor(树)+ist(从事者) → 树木专家 /ˈɑːbərɪst/
        *   **arboretum** = arbor(树)+etum(场所) → 树木园 /ˌɑːbəˈriːtəm/
    *   **组合逻辑解释**：
        *   arboreal：树+……的 → 与树有关 → 树栖的
        *   arborist：树+从事者 → 研究管理树的人
        *   arboretum：树+场所 → 种树的地方 → 树木园
    *   **同源词根及音变规律**：
        *   同源：arbus（灌木）
        *   音变：or ↔ us（词尾变化）
    *   **词缀总结**：
        *   -eal（……的）
        *   -ist（从事者）
        *   -etum（场所）

    *   **组合逻辑解释**：
        *   ignite：火+使 → 让产生火 → 点燃
        *   ignition：火+动作 → 启动火的过程 → 点火
        *   igneous：火+……的 → 由火形成 → 火成
    *   **同源词根及音变规律**：
        *   同源：fire（意象相同）
        *   音变：ign → in（辅音弱化）
    *   **词缀总结**：
        *   -ite（使……）
        *   -ition（动作）
        *   -eous（……的）

*   **词根**：**luc/lum**
    *   **核心含义**：光
    *   **词源背景**：来自拉丁语 *lux/lucis*，古罗马文化中光象征启示与知识
    *   **记忆技巧**：联想 illuminate（照亮）
    *   **高频词（3个）**：
        *   **lucid** = luc(光)+id(……的) → 清晰的 /ˈluːsɪd/
        *   **illuminate** = il(向内)+lumin(光)+ate(使……) → 照亮 /ɪˈluːmɪneɪt/
        *   **luminous** = lumin(光)+ous(……的) → 发光的 /ˈluːmɪnəs/
    *   **组合逻辑解释**：
        *   lucid：光+……的 → 像光一样清楚 → 清晰
        *   illuminate：内+光+使 → 让光进入 → 照亮
        *   luminous：光+……的 → 含光 → 发光
    *   **同源词根及音变规律**：
        *   同源：lux（原形）
        *   音变：c ↔ m（词形延展）
    *   **词缀总结**：
        *   -id（……的）
        *   -ate（使……）
        *   -ous（……的）

*   **词根**：**noct**
    *   **核心含义**：夜晚
    *   **词源背景**：来自拉丁语 *nox/noctis*，古罗马时间划分中“夜”常与守卫、星象相关
    *   **记忆技巧**：联想 nocturnal（夜行性的动物）
    *   **高频词（3个）**：
        *   **nocturnal** = noct(夜)+urnal(……的) → 夜间的 /nɒkˈtɜːnl/
        *   **equinox** = equi(相等)+nox(夜) → 昼夜平分点 /ˈiːkwɪnɒks/
        *   **nocturne** = noct(夜)+urne(曲) → 夜曲 /ˈnɒktɜːn/
    *   **组合逻辑解释**：
        *   nocturnal：夜+……的 → 在夜间活动的
        *   equinox：相等+夜 → 昼夜时间相等
        *   nocturne：夜+曲 → 描绘夜晚的音乐
    *   **同源词根及音变规律**：
        *   同源：night（意象相同）
        *   音变：ct ↔ gh（辅音弱化）
    *   **词缀总结**：
        *   -urnal（……的）
        *   -urne（曲）
        *   equi-（相等）

*   **词根**：**sol**
    *   **核心含义**：太阳
    *   **词源背景**：来自拉丁语 *sol*，古罗马太阳神象征力量与时间节律
    *   **记忆技巧**：联想 solar（太阳能）
    *   **高频词（3个）**：
        *   **solar** = sol(太阳)+ar(……的) → 太阳的 /ˈsəʊlə/
        *   **solstice** = sol(太阳)+stice(站立) → 至日 /ˈsɒlstɪs/
        *   **parasol** = para(遮挡)+sol(太阳) → 阳伞 /ˈpærəsɒl/
    *   **组合逻辑解释**：
        *   solar：太阳+……的 → 与太阳相关
        *   solstice：太阳+站立 → 太阳“停在最高/最低点” → 至日
        *   parasol：遮挡+太阳 → 用来挡太阳的物品 → 阳伞
    *   **同源词根及音变规律**：
        *   同源：heli（希腊语“日”）
        *   音变：s ↔ h（跨语音变）
    *   **词缀总结**：
        *   -ar（……的）
        *   -stice（站立）
        *   para-（遮挡）

*   **词根**：**lun**
    *   **核心含义**：月亮
    *   **词源背景**：来自拉丁语 *luna*，古人以月亮记录潮汐与月相，并牵引情绪象征
    *   **记忆技巧**：联想 lunar（月球）
    *   **高频词（3个）**：
        *   **lunar** = lun(月)+ar(……的) → 月亮的 /ˈluːnə/
        *   **lunatic** = lun(月)+atic(……的) → 疯癫的 /ˈluːnətɪk/
        *   **lunate** = lun(月)+ate(形状) → 弯月形的 /ˈluːneɪt/
    *   **组合逻辑解释**：
        *   lunar：月+……的 → 与月亮相关
        *   lunatic：月+……的 → 古人认为情绪受月亮影响 → 疯癫的
        *   lunate：月+形状 → 像月亮 → 弯月形
    *   **同源词根及音变规律**：
        *   同源：moon（意象相同）
        *   音变：l ↔ m（辅音转换）
    *   **词缀总结**：
        *   -ar（……的）
        *   -atic（……的）
        *   -ate（形状）
