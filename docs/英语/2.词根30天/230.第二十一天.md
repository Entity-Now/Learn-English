### 今日词根学习（第九天：10个随机常用词根）

#### 1. 词根: merc + 同义/同源词根（merch/merc） + /mɜːrk/
2. 核心含义：贸易、商业  
3. 词源背景：来自拉丁语“merces”（报酬、商品），源于罗马商业神Mercury，常用于买卖相关词汇  
4. 记忆技巧：想象“merchant”像一个商人用“mercury”般快速交易货物  
5. 3个高频词：
   - merchant /ˈmɜːrtʃənt/：merc（贸易） + ant（人后缀），指“商人”，因为商人从事“贸易”活动的人
   - commerce /ˈkɑːmɜːrs/：com（共同） + merc（贸易） + e（连接），意思是“商业”，因为商业涉及多人“共同贸易”
   - mercy /ˈmɜːrsi/：merc（报酬，原义仁慈作为报酬） + y（抽象），指“仁慈”，像给予“贸易”外的宽容
6. 同源词根的常见音变规律：merc → merch（名词形式，加h，如merch-ant）  
7. 高频词中出现的词缀总结：ant /ənt/（人）；com /kəm/（共同）；e /i/（连接）；y /i/（抽象名词）

#### 2. 词根: em/em/empt + 同义/同源词根（empt/emp） + /ɛm/ /ɛmpt/
2. 核心含义：买、拿  
3. 词源背景：来自拉丁语“emere”（买、拿），empt为其过去分词，用于豁免、赎回等词汇  
4. 记忆技巧：想到“empty”像“拿”空了东西，“redeem”是“重新买”回  
5. 3个高频词：
   - empty /ˈɛmpti/：em（拿） + pty（形容词），指“空的”，因为东西被“拿”光了
   - exempt /ɪɡˈzɛmpt/：ex（出） + empt（拿），意思是“豁免的”，从责任中“拿”出去
   - redeem /rɪˈdiːm/：re（回） + em（买），指“赎回/挽回”，重新“买”回东西或声誉
6. 同源词根的常见音变规律：em → empt（过去形式，加pt）；emp（简化）  
7. 高频词中出现的词缀总结：pty /pti/（形容词）；ex /ɪɡ/（出）；re /rɪ/（回）

#### 3. 词根: sum/sumpt + 同义/同源词根（sum/sumpt） + /sʌm/
2. 核心含义：花、消费  
3. 词源背景：来自拉丁语“sumere”（拿、消耗），sumpt为其过去分词，用于消费、假设等词汇  
4. 记忆技巧：想象“consume”像“彻底花”掉东西，“assume”是“拿”来假设  
5. 3个高频词：
   - consume /kənˈsuːm/：con（彻底） + sum（花），指“消耗/消费”，彻底“花”掉资源
   - assume /əˈsuːm/：ad（向） + sum（拿），意思是“假设”，向自己“拿”来一个想法
   - sumptuous /ˈsʌmptʃuəs/：sumpt（消费） + uous（倾向的），指“奢侈的”，倾向于高“消费”的
6. 同源词根的常见音变规律：sum（动词）；sumpt（过去/形容词，加pt）  
7. 高频词中出现的词缀总结：con /kən/（彻底）；ad /ə/（向）；uous /uəs/（倾向的形容词）

#### 4. 词根: val/vail + 同义/同源词根（val/vail） + /væl/
2. 核心含义：价值、强  
3. 词源背景：来自拉丁语“valere”（强壮、有价值），用于价值、有效等词汇  
4. 记忆技巧：想到“value”就是“价值”，“prevail”是“强”于他人  
5. 3个高频词：
   - value /ˈvæljuː/：val（价值） + ue（抽象），指“价值”，直接表示事物的“价值”
   - valid /ˈvælɪd/：val（强） + id（形容词），意思是“有效的/有力的”，有“强”效力的
   - prevail /prɪˈveɪl/：pre（前） + vail（强），指“盛行/获胜”，“强”于前方或他人
6. 同源词根的常见音变规律：val → vail（动词变体，加i）  
7. 高频词中出现的词缀总结：ue /juː/（抽象名词）；id /ɪd/（形容词）；pre /prɪ/（前）

#### 5. 词根: ven/vent + 同义/同源词根（ven/vent） + /vɛn/
2. 核心含义：来、到达  
3. 词源背景：来自拉丁语“venire”（来），用于发明、冒险等词汇  
4. 记忆技巧：想象“adventure”像“来”到冒险，“prevent”是“前来到”阻止  
5. 3个高频词：
   - venture /ˈvɛntʃər/：vent（来） + ure（动作），指“冒险/企业”，敢于“来”到未知领域
   - prevent /prɪˈvɛnt/：pre（前） + vent（来），意思是“防止”，提前“来”阻止
   - convene /kənˈviːn/：con（一起） + ven（来），指“召集”，使人们“一起来”
6. 同源词根的常见音变规律：ven（动词）；vent（过去形式，加t）  
7. 高频词中出现的词缀总结：ure /ər/（动作名词）；pre /prɪ/（前）；con /kən/（一起）

#### 6. 词根: pend/pens + 同义/同源词根（pend/pens） + /pɛnd/
2. 核心含义：挂、支付、悬挂  
3. 词源背景：来自拉丁语“pendere”（挂、称重），pens为其过去分词，用于依赖、花费等词汇  
4. 记忆技巧：想到“pending”像“挂”着的任务，“expense”是“支付”出去的  
5. 3个高频词：
   - depend /dɪˈpɛnd/：de（下） + pend（挂），指“依靠”，像“挂”在下面依赖他人
   - expense /ɪkˈspɛns/：ex（出） + pens（支付），意思是“花费”，钱“支付”出去
   - pending /ˈpɛndɪŋ/：pend（挂） + ing（进行中），指“待定的”，“挂”着的状态
6. 同源词根的常见音变规律：pend（动词）；pens（过去形式，加s）  
7. 高频词中出现的词缀总结：de /dɪ/（下）；ex /ɪk/（出）；ing /ɪŋ/（进行中状态）

#### 7. 词根: numer + 同义/同源词根（numero/numer） + /ˈnuːmər/
2. 核心含义：数字、数目  
3. 词源背景：来自拉丁语“numerus”（数目），用于数字、枚举等词汇  
4. 记忆技巧：想象“number”就是“数字”，“enumerate”是“数”出来列表  
5. 3个高频词：
   - numeral /ˈnuːmərəl/：numer（数字） + al（相关的），指“数字的/数词”，与“数字”相关的符号
   - numerous /ˈnuːmərəs/：numer（数字） + ous（多的），意思是“众多的”，数目“多”的
   - enumerate /ɪˈnuːməreɪt/：e（出） + numer（数字） + ate，指“列举”，把项目“数”出来
6. 同源词根的常见音变规律：numer → numero（古典形式，加o）  
7. 高频词中出现的词缀总结：al /əl/（相关的形容词）；ous /əs/（形容词）；e /ɪ/（出）；ate /eɪt/（动词）

#### 8. 词根: bon/bene + 同义/同源词根（bon/bene） + /bɑːn/ /ˈbɛni/
2. 核心含义：好、善  
3. 词源背景：来自拉丁语“bonus”（好）、“bene”（好地），用于好处、仁慈等词汇  
4. 记忆技巧：想到“bonus”就是“好”的奖励，“benefit”是“好”的益处  
5. 3个高频词：
   - bonus /ˈboʊnəs/：bon（好） + us（名词），指“奖金/额外的好处”，“好”的额外奖励
   - benefit /ˈbɛnɪfɪt/：bene（好） + fit（做，原义做好事），意思是“利益/好处”，“好”的结果
   - benevolent /bəˈnɛvələnt/：bene（好） + vol（意愿） + ent，指“仁慈的”，“好”意愿的
6. 同源词根的常见音变规律：bon（名词）；bene（副词/前缀，加e）  
7. 高频词中出现的词缀总结：us /əs/（名词）；fit /fɪt/（做）；vol /vəl/（意愿）；ent /ənt/（形容词）

#### 9. 词根: mal + 同义/同源词根（mali/mal） + /mæl/
2. 核心含义：坏、恶  
3. 词源背景：来自拉丁语“malus”（坏的），用于恶意、故障等词汇  
4. 记忆技巧：想象“malfunction”像“坏”的功能，“malicious”是“恶”意的  
5. 3个高频词：
   - malfunction /ˌmælˈfʌŋkʃn/：mal（坏） + function（功能），指“故障”，“坏”的功能运作
   - malicious /məˈlɪʃəs/：mal（恶） + icious（倾向的），意思是“恶意的”，“恶”意的倾向
   - malice /ˈmælɪs/：mal（坏） + ice（抽象），指“恶意”，“坏”的意图
6. 同源词根的常见音变规律：mal稳定；mali（加强形式，加i）  
7. 高频词中出现的词缀总结：function /ˈfʌŋkʃn/（功能）；icious /ɪʃəs/（倾向的形容词）；ice /ɪs/（抽象名词）

#### 10. 词根: cost + 同义/同源词根（cost） + /kɔːst/
2. 核心含义：花费  
3. 词源背景：来自拉丁语“constare”（花费、站立），用于成本、昂贵等词汇  
4. 记忆技巧：想到“cost”直接是“花费”，“costly”是“花费”大的  
5. 3个高频词：
   - cost /kɔːst/：直接使用词根，意思是“成本/花费”，直接表示“花费”的金额
   - costly /ˈkɔːstli/：cost（花费） + ly（方式），指“昂贵的”，“花费”高的
   - accost /əˈkɔːst/：ad（向） + cost（站，原义站到身边），指“搭讪”，“向”别人靠近说话
6. 同源词根的常见音变规律：cost稳定，无明显变体  
7. 高频词中出现的词缀总结：ly /li/（方式形容词）；ad /ə/（向）

### 小作文：Shopping with Care and Wisdom

When I went shopping, I looked for items of real value, not just costly ones. A wise merchant explained the benefits of each product, helping me avoid malicious sellers who might overcharge. I decided to redeem a coupon to reduce expenses, assuming it would prevent unnecessary spending. The store had numerous options, from benevolent brands that donate to charity to those with sumptuous designs. In the end, I ventured to buy only what I needed, hanging onto my budget like a pending decision. This way, I enjoyed the commerce without any malice toward my wallet.

（注：coupon /ˈkuːpɑːn/（优惠券）；overcharge /ˌoʊvərˈtʃɑːrdʒ/（多收费）；donate /ˈdoʊneɪt/（捐赠）；wallet /ˈwɑːlɪt/（钱包））

**中文总结**：这篇文章描述了作者小心而明智的购物经历：寻找真正有价值而非昂贵的物品、听取商人解释益处、避免恶意卖家、使用优惠券减少花费、假设能防止多余消费、面对众多选择、从慈善品牌到奢侈设计，最终只买必需品，保持预算，像待定决定般悬挂，避免对钱包的恶意，从而享受商业过程。
