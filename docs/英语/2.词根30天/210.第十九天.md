### 今日词根学习（第七天：10个随机常用词根）

#### 1. 词根: dent + 同义/同源词根（denti/dent） + /dɛnt/
2. 核心含义：牙齿  
3. 词源背景：来自拉丁语“dens/dentis”（牙齿），常用于牙科相关词汇  
4. 记忆技巧：想象“dentist”像在“dent”一个洞来修牙齿  
5. 3个高频词：
   - dental /ˈdɛntl/：dent（牙齿） + al（相关的），指“牙齿的/牙科的”，因为它描述与“牙齿”相关的健康或治疗
   - dentist /ˈdɛntɪst/：dent（牙齿） + ist（专家），意思是“牙医”，专门处理“牙齿”问题的人
   - denture /ˈdɛntʃər/：dent（牙齿） + ure（物品），指“假牙”，仿造的“牙齿”装置
6. 同源词根的常见音变规律：dent → denti（形容词/加强形式，如dental中的隐形i）  
7. 高频词中出现的词缀总结：al /l/（相关的形容词）；ist /ɪst/（专家）；ure /ər/（物品名词）

#### 2. 词根: corp + 同义/同源词根（corpo/corpus） + /kɔːrp/
2. 核心含义：身体  
3. 词源背景：来自拉丁语“corpus”（身体），用于身体、团体等词汇  
4. 记忆技巧：想到“corporation”像一群人组成一个“身体”般的公司  
5. 3个高频词：
   - corporation /ˌkɔːrpəˈreɪʃn/：corp（身体） + oration（集体），指“公司/法人”，像多人组成的“身体”实体
   - corpse /kɔːrps/：corp（身体） + se（状态），意思是“尸体”，无生命的“身体”
   - corporal /ˈkɔːrpərəl/：corp（身体） + al（相关的），指“身体的/下士”，与“身体”相关的或军衔
6. 同源词根的常见音变规律：corp → corpus（原拉丁名词形式）；corpo（少见）  
7. 高频词中出现的词缀总结：oration /əˈreɪʃn/（集体名词）；se /s/（状态）；al /əl/（相关的形容词）

#### 3. 词根: dict/dic + 同义/同源词根（dict/dic） + /dɪkt/
2. 核心含义：说、讲  
3. 词源背景：来自拉丁语“dicere”（说、宣称），用于言语、预测等词汇  
4. 记忆技巧：想象“dictionary”是一本书在“dictate”单词给你“说”  
5. 3个高频词：
   - dictate /ˈdɪkteɪt/：dic（说） + tate（使…），指“口授/命令”，像权威地“说”出内容
   - dictionary /ˈdɪkʃəneri/：dict（说） + ion + ary（物品），意思是“词典”，收集单词“说”法的书
   - predict /prɪˈdɪkt/：pre（前） + dict（说），指“预测”，提前“说”出未来
6. 同源词根的常见音变规律：dict（动词/名词）；dic（简化形式，如indicate）  
7. 高频词中出现的词缀总结：tate /teɪt/（使…动词）；ion /ən/（过程）；ary /eri/（物品名词）；pre /prɪ/（前）

#### 4. 词根: log/logo + 同义/同源词根（log/logo/logy） + /lɔːɡ/
2. 核心含义：说、研究、学  
3. 词源背景：来自希腊语“logos”（话语、逻辑），用于科学、对话等词汇  
4. 记忆技巧：把“logic”想象成用“log”来“说”理，“biology”是“研究生命”  
5. 3个高频词：
   - logic /ˈlɑːdʒɪk/：log（说/研究） + ic，指“逻辑”，有理的“说法”或思考
   - dialogue /ˈdaɪəlɔːɡ/：dia（通过） + log（说），意思是“对话”，两人“通过说”交流
   - biology /baɪˈɑːlədʒi/：bio（生命） + logy（研究），指“生物学”，“研究”生命的科学
6. 同源词根的常见音变规律：log → logo（符号形式）；logy（学科后缀，加y）  
7. 高频词中出现的词缀总结：ic /ɪk/（形容词）；dia /ˈdaɪə/（通过）；bio /baɪˈɑː/（生命）；y /i/（名词化）

#### 5. 词根: prim + 同义/同源词根（prime/primi） + /prɪm/
2. 核心含义：第一、最初  
3. 词源背景：来自拉丁语“primus”（第一），用于首要、原始等词汇  
4. 记忆技巧：想到“primary”就是“第一”学校，“prime”像“首要”时间  
5. 3个高频词：
   - primary /ˈpraɪmeri/：prim（第一） + ary，指“首要的/初级的”，“第一”级别的或基础的
   - prime /praɪm/：直接使用词根，意思是“首要的/质数”，最初或“第一”选择的
   - primitive /ˈprɪmətɪv/：prim（第一） + itive，指“原始的/基本的”，“第一”时代的或简单
6. 同源词根的常见音变规律：prim → prime（形容词形式）；primi（少见）  
7. 高频词中出现的词缀总结：ary /eri/（相关的形容词）；itive /ɪtɪv/（形容词）

#### 6. 词根: term + 同义/同源词根（termi/term） + /tɜːrm/
2. 核心含义：结束、界限  
3. 词源背景：来自拉丁语“terminus”（界限、终点），用于期限、终止等词汇  
4. 记忆技巧：想象“terminal”是一个“结束”站，“determine”是“定下界限”  
5. 3个高频词：
   - terminal /ˈtɜːrmɪnl/：term（结束） + inal，指“终端/终点的”，“结束”处的设备或地点
   - terminate /ˈtɜːrmɪneɪt/：term（结束） + inate，指“终止”，使某事“结束”
   - determine /dɪˈtɜːrmɪn/：de（完全） + term（界限） + ine，指“决定/确定”，完全“定下界限”
6. 同源词根的常见音变规律：term稳定；termi（名词/形容词形式，加i）  
7. 高频词中出现的词缀总结：inal /ɪnl/（地方/形容词）；inate /ɪneɪt/（动词）；de /dɪ/（完全）；ine /ɪn/（动词）

#### 7. 词根: mort + 同义/同源词根（morti/mort） + /mɔːrt/
2. 核心含义：死  
3. 词源背景：来自拉丁语“mors/mortis”（死亡），用于死亡相关词汇  
4. 记忆技巧：想到“mortal”就是“会死”的，“immortal”是不死的  
5. 3个高频词：
   - mortal /ˈmɔːrtl/：mort（死） + al，指“凡人的/致命的”，会“死”的或导致死的
   - immortal /ɪˈmɔːrtl/：im（不） + mort（死） + al，指“不朽的”，不会“死”的
   - mortgage /ˈmɔːrɡɪdʒ/：mort（死） + gage（抵押），指“抵押贷款”，原义“死”的抵押（还清才结束）
6. 同源词根的常见音变规律：mort → morti（形容词形式，加i）  
7. 高频词中出现的词缀总结：al /l/（形容词）；im /ɪm/（不）；gage /ɡɪdʒ/（抵押名词）

#### 8. 词根: nat + 同义/同源词根（nati/nat） + /næt/
2. 核心含义：出生、自然  
3. 词源背景：来自拉丁语“natus”（出生），用于出生、国家等词汇  
4. 记忆技巧：把“nature”想象成大自然的“出生”万物，“nation”是从同一“出生”的人群  
5. 3个高频词：
   - nature /ˈneɪtʃər/：nat（出生） + ure，指“自然/本性”，万物“出生”的状态
   - native /ˈneɪtɪv/：nat（出生） + ive，指“本土的/本地的”，从“出生”地来的
   - nation /ˈneɪʃn/：nat（出生） + ion，指“国家/民族”，同一“出生”背景的人群
6. 同源词根的常见音变规律：nat稳定；nati（名词形式，加i）  
7. 高频词中出现的词缀总结：ure /ər/（状态名词）；ive /ɪv/（形容词）；ion /ən/（集体名词）

#### 9. 词根: vit/viv + 同义/同源词根（vita/vivi） + /vɪt/ /vɪv/
2. 核心含义：生命、活  
3. 词源背景：来自拉丁语“vita”（生命），viv是其变体，用于活力、生存等词汇  
4. 记忆技巧：想到“vital”就是“生命的”，“survive”是从“活”下来  
5. 3个高频词：
   - vital /ˈvaɪtl/：vit（生命） + al，指“重要的/生命的”，关乎“生命”的
   - vitamin /ˈvaɪtəmɪn/：vita（生命） + min（胺类），指“维生素”，维持“生命”的必需物质
   - survive /sərˈvaɪv/：sur（上/超） + viv（活），指“幸存”，“活”下来超过困难
6. 同源词根的常见音变规律：vit → viv（动词/形容词变体）；vita（完整名词）  
7. 高频词中出现的词缀总结：al /l/（形容词）；min /mɪn/（物质）；sur /sər/（超）

#### 10. 词根: juven + 同义/同源词根（juveni/juve） + /ˈdʒuːvən/
2. 核心含义：年轻  
3. 词源背景：来自拉丁语“juvenis”（年轻的），用于青春、复兴等词汇  
4. 记忆技巧：想象“juvenile”像“juven”一个年轻人，“rejuvenate”是“重新年轻”  
5. 3个高频词：
   - juvenile /ˈdʒuːvənaɪl/：juven（年轻） + ile，指“青少年的/幼稚的”，属于“年轻”阶段的
   - rejuvenate /rɪˈdʒuːvəneɪt/：re（重新） + juven（年轻） + ate，指“使年轻/恢复活力”，重新“变年轻”
   - juvenescence /ˌdʒuːvəˈnɛsəns/：juven（年轻） + escence（过程），指“青春期”，“年轻”化的过程
6. 同源词根的常见音变规律：juven → juveni（形容词形式，加i）  
7. 高频词中出现的词缀总结：ile /aɪl/（形容词）；re /rɪ/（重新）；ate /eɪt/（使…动词）；escence /ˈɛsəns/（过程名词）

### 小作文：The Cycle of Life and Growth

From the moment of birth, nature guides us through vital stages. As a juvenile, I was full of energy, exploring the world with a primitive curiosity that felt immortal. My parents would dictate simple rules to keep me safe, like brushing my dental health every day to avoid any corporal issues. In school, the primary lessons came from a dictionary and biology books, teaching us about life from nativity to mortality. Now, as an adult, I aim to rejuvenate my spirit by setting clear terms for personal goals, perhaps joining a corporation where innovation is key. Though life has its mortal limits, staying vital helps us predict a brighter future.

（注：curiosity /ˌkjʊriˈɑːsəti/（好奇心）；nativity /nəˈtɪvəti/（出生）；mortality /mɔːrˈtæləti/（死亡率）；innovation /ˌɪnəˈveɪʃn/（创新））

**中文总结**：这篇文章描述了生命的循环与成长：从出生开始，自然引导我们经历重要阶段，年轻时充满活力和原始好奇，父母口授规则维护牙齿和身体健康，学校教导基础知识和生物学，现在作为成人，通过设定明确目标和加入公司来恢复青春，尽管生命有死亡界限，但保持活力能预测更光明未来。
