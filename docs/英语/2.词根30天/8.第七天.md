#### **mot**

*   **核心含义**：移动
*   **词源背景**：源自拉丁语 *movere*，意为“移动、推动”，古罗马人用它描述军队行进或物体被推动的场景
*   **记忆技巧**：联想“motor（发动机）让车动起来”
*   **高频词**：
    *   **motor** = mot(移动)+or(名词后缀) → 发动机 /ˈməʊtə/
    *   **motion** = mot(移动)+ion(动作) → 运动 /ˈməʊʃən/
    *   **promote** = pro(向前)+mot(移动)+e → 促进 /prəˈməʊt/
*   **组合逻辑解释**：
    *   motor：移动+名词 → 让物体移动的装置 → 发动机
    *   motion：移动+动作 → 移动的状态 → 运动
    *   promote：向前+移动 → 推动事情向前发展 → 促进
*   **同源词根及音变规律**：
    *   mob（移动） ↔ mot（移动）
    *   音变规律：b ↔ t（辅音替换）
*   **词缀总结**：
    *   pro-(向前)
    *   -or(名词后缀)
    *   -ion(动作、状态)

#### **mob**

*   **核心含义**：移动
*   **词源背景**：源自拉丁语 *mobilis*，意为“可动的”，古代用于描述灵活的物体或人群
*   **记忆技巧**：联想“mobile phone（手机）随时移动”
*   **高频词**：
    *   **mobile** = mob(移动)+ile(形容词后缀) → 可移动的 /ˈməʊbaɪl/
    *   **mobility** = mob(移动)+ility(能力) → 移动性 /məʊˈbɪləti/
    *   **immobile** = im(不)+mob(移动)+ile → 不动的 /ɪˈməʊbaɪl/
*   **组合逻辑解释**：
    *   mobile：移动+形容词 → 能移动的 → 可移动的
    *   mobility：移动+能力 → 移动的能力 → 移动性
    *   immobile：不+移动 → 无法移动 → 不动的
*   **同源词根及音变规律**：
    *   mot（移动） ↔ mob（移动）
    *   音变规律：t ↔ b（辅音替换）
*   **词缀总结**：
    *   im-(不)
    *   -ile(形容词后缀)
    *   -ity(性质、能力)

#### **migr**

*   **核心含义**：迁移
*   **词源背景**：源自拉丁语 *migrare*，意为“迁移、搬迁”，古罗马人用它描述人口或军队的迁徙
*   **记忆技巧**：联想“migrate 像鸟类迁徙”
*   **高频词**：
    *   **migrate** = migr(迁移)+ate(动词后缀) → 迁移 /maɪˈɡreɪt/
    *   **immigrant** = im(进入)+migr(迁移)+ant(人) → 移民 /ˈɪmɪɡrənt/
    *   **emigrate** = e(出去)+migr(迁移)+ate → 移居国外 /ˈemɪɡreɪt/
*   **组合逻辑解释**：
    *   migrate：迁移+动作 → 进行迁移 → 迁移
    *   immigrant：进入+迁移+人 → 迁移到某地的人 → 移民
    *   emigrate：出去+迁移 → 从本国迁移出去 → 移居国外
*   **同源词根及音变规律**：
    *   migr ↔ migrat（加后缀变化）
    *   音变规律：无明显辅音替换，主要是后缀变化
*   **词缀总结**：
    *   im-(进入)
    *   e-(出去)
    *   -ant(人)
    *   -ate(动词后缀)

#### **cede**

*   **核心含义**：走
*   **词源背景**：源自拉丁语 *cedere*，意为“走、退让”，古代用于描述军队撤退或让步
*   **记忆技巧**：联想“cede 像‘走的’”
*   **高频词**：
    *   **precede** = pre(在前)+cede(走) → 先于 /prɪˈsiːd/
    *   **concede** = con(一起)+cede(走) → 让步 /kənˈsiːd/
    *   **recede** = re(回)+cede(走) → 后退 /rɪˈsiːd/
*   **组合逻辑解释**：
    *   precede：在前+走 → 走在前面 → 先于
    *   concede：一起+走 → 退一步 → 让步
    *   recede：回+走 → 向后走 → 后退
*   **同源词根及音变规律**：
    *   ceed ↔ cede ↔ cess
    *   音变规律：d ↔ s（辅音替换）
*   **词缀总结**：
    *   pre-(在前)
    *   con-(一起)
    *   re-(回)

*   **组合逻辑解释**：
    *   access：去+走 → 走进去 → 进入
    *   process：向前+走 → 前进的过程 → 过程
    *   success：下+走 → 接续完成 → 成功
*   **同源词根及音变规律**：
    *   cede ↔ ceed ↔ cess
    *   音变规律：d ↔ s
*   **词缀总结**：
    *   ac-(去)

*   **高频词**：

    *   **gradual** = grad(步)+ual(形容词后缀) → 逐步的 /ˈɡrædʒuəl/
    *   **graduate** = grad(步)+uate(动词后缀) → 毕业 /ˈɡrædʒueɪt/
*   **组合逻辑解释**：
    *   grade：步+名词 → 等级像一步步 → 等级
    *   gradual：步+形容词 → 一步步的 → 逐步的
    *   graduate：步+动作 → 跨过学习的步伐 → 毕业
*   **同源词根及音变规律**：
    *   gress ↔ grad
    *   音变规律：d ↔ ss
*   **词缀总结**：
    *   -ual(形容词后缀)
    *   -uate(动词后缀)

#### **volv**

*   **核心含义**：转
*   **词源背景**：源自拉丁语 *volvere*，意为“滚动、旋转”，古代用于描述车轮或天体的转动
*   **记忆技巧**：联想“revolve 像‘地球自转’”
*   **高频词**：
    *   **revolve** = re(再)+volv(转)+e → 旋转 /rɪˈvɒlv/
    *   **evolve** = e(向外)+volv(转)+e → 演变 /ɪˈvɒlv/
    *   **involve** = in(进入)+volv(转)+e → 涉及 /ɪnˈvɒlv/
*   **组合逻辑解释**：
    *   revolve：再+转 → 不断转动 → 旋转
    *   evolve：向外+转 → 向外变化 → 演变
    *   involve：进入+转 → 卷入其中 → 涉及
*   **同源词根及音变规律**：
    *   volv ↔ volu（音变后缀）
    *   音变规律：v ↔ u（元音变化）
*   **词缀总结**：
    *   re-(再)
    *   e-(向外)
    *   in-(进入)

*   **高频词**：

    *   **rotor** = rot(转)+or(名词后缀) → 转子 /ˈrəʊtə/
*   **组合逻辑解释**：
    *   rotate：转+动作 → 转动 → 旋转
    *   rotation：转+动作 → 转动的过程 → 旋转
    *   rotor：转+名词 → 转动的部件 → 转子
*   **同源词根及音变规律**：
    *   rot ↔ rota ↔ rotat
    *   音变规律：a ↔ at（后缀变化）
*   **词缀总结**：

    *   -ation(动作)
    *   -or(名词后缀)
