*   #### **sub**

    *   **词根**：sub
    *   **核心含义**：下、下方
    *   **词源背景**：来自拉丁语 *sub*（在……之下），用于土地测量、建筑中指“下层结构”
    *   **记忆技巧**：联想 subway 地铁在“下方”
    *   **高频词（3个）**：
        *   **subway** = sub(下)+way(路) → 地铁 /ˈsʌbweɪ/
        *   **submarine** = sub(下)+marine(海) → 潜艇 /ˌsʌbməˈriːn/
        *   **subconscious** = sub(下)+conscious(意识) → 潜意识的 /ˌsʌbˈkɒnʃəs/
    *   **组合逻辑解释**：
        *   subway：下+路 → 地下的路 → 地铁
        *   submarine：下+海 → 在海下活动 → 潜艇
        *   subconscious：下+意识 → 意识下层 → 潜意识
    *   **同源词根及音变规律**：
        *   同源：sub ↔ hypo（下）
        *   音变：s ↔ h（跨语系）
    *   **词缀总结**：
        *   -way(道路)
        *   -marine(海)
        *   -conscious(意识)

*   #### **super**

    *   **词根**：super
    *   **核心含义**：上、超越
    *   **词源背景**：来自拉丁语 *super*（在上方），古罗马用来指“高等阶层/上级权力”
    *   **记忆技巧**：联想 superman 超人=超等级的人
    *   **高频词（3个）**：
        *   **superman** = super(超)+man(人) → 超人 /ˈsuːpəmæn/
        *   **supervise** = super(在上)+vise(看) → 监督 /ˈsuːpəvaɪz/
        *   **superficial** = super(表面)+ficial(相关) → 表面的 /ˌsuːpəˈfɪʃl/
    *   **组合逻辑解释**：
        *   superman：超+人 → 超能力的人 → 超人
        *   supervise：上+看 → 从上往下看 → 监督
        *   superficial：表面+相关 → 只在表面 → 肤浅
    *   **同源词根及音变规律**：
        *   同源：super ↔ sur（法语影响）
        *   音变：super → sur（弱化）
    *   **词缀总结**：
        *   -man(人)
        *   -vise(看)
        *   -ficial(相关)

*   #### **inter**

    *   **词根**：inter
    *   **核心含义**：在……之间
    *   **词源背景**：源自拉丁语 *inter*，古罗马道路系统常说“城与城之间”
    *   **记忆技巧**：联想 internet 连接“之间”
    *   **高频词（3个）**：
        *   **international** = inter(之间)+national(国家) → 国际的 /ˌɪntəˈnæʃnəl/
        *   **interact** = inter(之间)+act(做) → 互动 /ˌɪntərˈækt/
        *   **interview** = inter(之间)+view(看) → 面试 /ˈɪntəvjuː/
    *   **组合逻辑解释**：
        *   international：国家+之间 → 国际
        *   interact：之间+做 → 彼此作用 → 互动
        *   interview：之间+看 → 面对面看 → 面试
    *   **同源词根及音变规律**：
        *   同源：inter ↔ intra（内）
        *   音变：er ↔ ra（对立关系）
    *   **词缀总结**：
        *   -national(国家)
        *   -act(做)
        *   -view(看)

*   #### **trans**

    *   **词根**：trans
    *   **核心含义**：穿越、跨越
    *   **词源背景**：来自拉丁语 *trans*（穿过），古代用于描述跨河、跨地界的移动
    *   **记忆技巧**：联想 transport 运输要“跨过去”
    *   **高频词（3个）**：
        *   **transport** = trans(穿越)+port(港/运) → 运输 /trænsˈpɔːt/
        *   **translate** = trans(跨越)+late(带) → 翻译 /trænsˈleɪt/
        *   **transform** = trans(改变)+form(形状) → 转变 /trænsˈfɔːm/
    *   **组合逻辑解释**：
        *   transport：穿越+运 → 把东西运过去 → 运输
        *   translate：跨越+带 → 把意思带到另一语言 → 翻译
        *   transform：穿越+形状 → 形态改变 → 转变
    *   **同源词根及音变规律**：
        *   同源：trans ↔ tra（弱化）
        *   音变：ns 脱落为 n
    *   **词缀总结**：
        *   -port(运输)
        *   -late(带)
        *   -form(形状)

*   #### **peri**

    *   **词根**：peri
    *   **核心含义**：周围
    *   **词源背景**：来自希腊语 *peri*（环绕），常用于几何“周长”、军事“包围阵”
    *   **记忆技巧**：联想 perimeter 周长
    *   **高频词（3个）**：
        *   **perimeter** = peri(周围)+meter(测量) → 周长 /pəˈrɪmɪtə/
        *   **peripheral** = peri(周围)+pher(带来)+al → 边缘的 /pəˈrɪfərəl/
        *   **period** = peri(环)+od(路) → 时期 /ˈpɪəriəd/
    *   **组合逻辑解释**：
        *   perimeter：周围+测 → 测四周 → 周长
        *   peripheral：周围+带 → 带在外圈 → 边缘
        *   period：环+路 → 循环的路 → 时段
    *   **同源词根及音变规律**：
        *   同源：peri ↔ circum
        *   音变：p ↔ c（跨语系）
    *   **词缀总结**：
        *   -meter(测量)
        *   -al(形容词)
        *   -od(路/过程)

*   #### **circum**

    *   **词根**：circum
    *   **核心含义**：环绕
    *   **词源背景**：来自拉丁语 *circum*（环绕），用于军事包围、城市环道等场景
    *   **记忆技巧**：联想 circle 圆圈
    *   **高频词（3个）**：
        *   **circumference** = circum(环绕)+ference(携带) → 圆周 /səˈkʌmfərəns/
        *   **circumstance** = circum(周围)+stance(站) → 情况 /ˈsɜːkəmstæns/
        *   **circulate** = circul(环)+ate → 循环 /ˈsɜːkjəleɪt/
    *   **组合逻辑解释**：
        *   circumference：环绕+带 → 绕一圈 → 圆周
        *   circumstance：周围+站 → 所处的周遭 → 情况
        *   circulate：环+使 → 使成循环 → 流通
    *   **同源词根及音变规律**：
        *   同源：circum ↔ peri
        *   音变：c ↔ p（跨语源）
    *   **词缀总结**：
        *   -ference(带来)
        *   -stance(站)
        *   -ate(动词)

*   #### **ex**

    *   **词根**：ex
    *   **核心含义**：外、向外
    *   **词源背景**：来自拉丁语 *ex*（从……出来），常用于描述人或物“离开原位”
    *   **记忆技巧**：联想 exit 出口
    *   **高频词（3个）**：
        *   **exit** = ex(外)+it → 出口 /ˈeksɪt/
        *   **export** = ex(外)+port(运) → 出口（产品） /ˈekspɔːt/
        *   **expand** = ex(向外)+pand(伸) → 扩张 /ɪkˈspænd/
    *   **组合逻辑解释**：
        *   exit：外+走 → 出去 → 出口
        *   export：外+运 → 运出去 → 出口贸易
        *   expand：向外+伸 → 扩大
    *   **同源词根及音变规律**：
        *   同源：ex ↔ e-（弱化形式）
        *   音变：x ↔ s（发音简化）
    *   **词缀总结**：
        *   -port(运)
        *   -pand(伸)
        *   -it(动作)

*   #### **in / im**

    *   **词根**：in / im
    *   **核心含义**：内、进入
    *   **词源背景**：来自拉丁语 *in*（向内），辅音前变形为 im
    *   **记忆技巧**：联想 input → 输入
    *   **高频词（3个）**：
        *   **input** = in(内)+put(放) → 输入 /ˈɪnpʊt/
        *   **import** = im(向内)+port(运) → 进口 /ɪmˈpɔːt/
        *   **impact** = im(进入)+pact(压) → 影响 /ˈɪmpækt/
    *   **组合逻辑解释**：
        *   input：内+放 → 放进去 → 输入
        *   import：内+运 → 运进来 → 进口
        *   impact：进入+压 → 撞进去带来压力 → 影响
    *   **同源词根及音变规律**：
        *   同源：in ↔ en
        *   音变：i ↔ e（元音弱化）
    *   **词缀总结**：
        *   -put(放)
        *   -port(运)
        *   -pact(压)

*   #### **ad**

    *   **词根**：ad
    *   **核心含义**：向、朝
    *   **词源背景**：来自拉丁语 *ad*（朝向），在古罗马法律中表示“向某人提交”
    *   **记忆技巧**：联想 advance 向前
    *   **高频词（3个）**：
        *   **advance** = ad(向)+vance(走) → 前进 /ədˈvɑːns/
        *   **adapt** = ad(向)+apt(适合) → 适应 /əˈdæpt/
        *   **address** = ad(向)+dress(引导) → 地址；处理 /əˈdres/
    *   **组合逻辑解释**：
        *   advance：向+走 → 向前走 → 前进
        *   adapt：向+适合 → 朝适合方向调整 → 适应
        *   address：向+引导 → 指向某地/某问题 → 地址/处理
    *   **同源词根及音变规律**：
        *   同源：ad ↔ ac/af/ag（辅音同化）
        *   音变：d ↔ c/g/f
    *   **词缀总结**：
        *   -vance(走)
        *   -apt(适)
        *   -dress(引导)

*   #### **contra**

    *   **词根**：contra
    *   **核心含义**：反、对立
    *   **词源背景**：来自拉丁语 *contra*（反对），常出现于法律争端、辩论场合
    *   **记忆技巧**：联想 contrast 对比
    *   **高频词（3个）**：
        *   **contradict** = contra(反)+dict(说) → 反驳 /ˌkɒntrəˈdɪkt/
        *   **contrast** = contra(反)+st(站) → 对比 /ˈkɒntrɑːst/
        *   **controversy** = contro(反)+vers(转)+y → 争论 /ˈkɒntrəvɜːsi/
    *   **组合逻辑解释**：
        *   contradict：反+说 → 说反话 → 反驳
        *   contrast：反+站 → 站在反方向 → 对比
        *   controversy：反+转 → 意见互相转来转去 → 争论
    *   **同源词根及音变规律**：
        *   同源：contra ↔ counter
        *   音变：tra ↔ ter（弱化）
    *   **词缀总结**：
        *   -dict(说)
        *   -st(站)
        *   -vers(转)
