### 英语词根背诵清单

*   #### **act**

    *   **词根**：act
    *   **核心含义**：做、行动
    *   **词源背景**：来自拉丁语 *agere*（做、驱使），古罗马法律文本中常用于描述“采取行动”的行为
    *   **记忆技巧**：联想“action 电影动作不停”
    *   **高频词（3个）**：
        *   **action** = act(做)+ion(名词) → 行动；行为 /ˈækʃn/
        *   **active** = act(做)+ive(形容词) → 活跃的 /ˈæktɪv/
        *   **activate** = act(做)+ivate(使…) → 激活 /ˈæktɪveɪt/
    *   **组合逻辑解释**：
        *   action：做+名词 → 所做之事 → 行动
        *   active：做+形容词 → 不停在做 → 活跃
        *   activate：做+使… → 让它开始“动起来” → 激活
    *   **同源词根及音变规律**：
        *   同源：act ↔ ag（做）
        *   音变：ct ↔ g（辅音弱化）
    *   **词缀总结**：
        *   -ion(名词)
        *   -ive(形容词)
        *   -ate(使…)

***

*   #### **ag**

    *   **词根**：ag
    *   **核心含义**：做、驱动
    *   **词源背景**：同源于拉丁语 *agere*，古时用于表示牲畜驱赶、推动事物前进
    *   **记忆技巧**：联想“agent 特工一直在行动”
    *   **高频词（3个）**：
        *   **agent** = ag(做)+ent(人) → 代理人 /ˈeɪdʒənt/
        *   **agenda** = ag(做)+enda(要做的事) → 议程 /əˈdʒendə/
        *   **aggressive** = ag(做/驱动)+gressive(走) → 侵略性的 /əˈɡresɪv/
    *   **组合逻辑解释**：
        *   agent：做+人 → 代你办事的人 → 代理人
        *   agenda：做+要做 → 待办清单 → 议程
        *   aggressive：驱动+走 → 往前硬推 → 侵略性
    *   **同源词根及音变规律**：
        *   同源：ag ↔ act（做）
        *   音变：g ↔ ct（辅音变换）
    *   **词缀总结**：
        *   -ent(人)
        *   -enda(待做事项)
        *   -ive(形容词)

*   #### **oper**

    *   **词根**：oper
    *   **核心含义**：工作、运作
    *   **词源背景**：来自拉丁语 *opus*（工作、成果），古罗马人用其指工匠的劳动
    *   **记忆技巧**：联想“operate 操作机器工作”
    *   **高频词（3个）**：
        *   **operate** = oper(工作)+ate(动词) → 操作；运作 /ˈɒpəreɪt/
        *   **operation** = oper(工作)+ation(名词) → 操作；手术 /ˌɒpəˈreɪʃn/
        *   **cooperate** = co(一起)+operate(工作) → 合作 /kəʊˈɒpəreɪt/
    *   **组合逻辑解释**：
        *   operate：工作+动词 → 让东西开始工作 → 操作
        *   operation：工作+名词 → 工作过程 → 操作/手术
        *   cooperate：一起+工作 → 同心协力 → 合作
    *   **同源词根及音变规律**：
        *   同源：oper ↔ opus
        *   音变：u ↔ er（元音变化）
    *   **词缀总结**：
        *   -ate(动词)
        *   -ation(名词)
        *   co-(一起)

*   #### **fac / fact**

    *   **词根**：fac / fact
    *   **核心含义**：做、制造
    *   **词源背景**：来自拉丁语 *facere*（生产、创造），广泛用于描述手工劳动
    *   **记忆技巧**：联想 factory 工厂在“做东西”
    *   **高频词（3个）**：
        *   **factory** = fact(做)+ory(场所) → 工厂 /ˈfæktri/
        *   **manufacture** = manu(手)+fact(做)+ure → 制造 /ˌmænjuˈfæktʃə/
        *   **factor** = fac(做)+or(东西) → 因素 /ˈfæktə/
    *   **组合逻辑解释**：
        *   factory：做+场所 → 做东西的地方 → 工厂
        *   manufacture：手+做 → 手工制作（后泛指制造）
        *   factor：做+东西 → 影响“做成结果”的东西 → 因素
    *   **同源词根及音变规律**：
        *   同源：fac ↔ fic（做）
        *   音变：a ↔ i（元音分化）
    *   **词缀总结**：
        *   manu-(手)
        *   -ory(场所)
        *   -ure(名词)

*   #### **fic**

    *   **词根**：fic
    *   **核心含义**：做、形成
    *   **词源背景**：同源于拉丁语 *facere* 的另一音变形式，多用于抽象“创造、塑造”概念
    *   **记忆技巧**：联想 “artificial 人工做出来的”
    *   **高频词（3个）**：
        *   **fiction** = fic(做)+tion(名词) → 小说；虚构 /ˈfɪkʃn/
        *   **artificial** = arti(艺术)+fic(做)+ial(形容词) → 人工的 /ˌɑːtɪˈfɪʃl/
        *   **efficient** = ef(出)+fic(做)+ient(形容词) → 高效的 /ɪˈfɪʃnt/
    *   **组合逻辑解释**：
        *   fiction：做+名词 → 编造出来的 → 虚构
        *   artificial：艺术+做 → 做出的艺术品般 → 人工
        *   efficient：向外+做 → 做事顺畅 → 高效
    *   **同源词根及音变规律**：
        *   同源：fic ↔ fac
        *   音变：i ↔ a（元音变化）
    *   **词缀总结**：
        *   -tion(名词)
        *   -ial(形容词)
        *   ef-(向外)

*   #### **pos**

    *   **词根**：pos
    *   **核心含义**：放、置
    *   **词源背景**：来自拉丁语 *ponere*（放置），常用于军事布阵与土地标记
    *   **记忆技巧**：联想 position 位置就是“放在哪里”
    *   **高频词（3个）**：
        *   **position** = pos(放)+ition(名词) → 位置 /pəˈzɪʃn/
        *   **deposit** = de(向下)+posit(放) → 存放 /dɪˈpɒzɪt/
        *   **compose** = com(一起)+pose(放) → 组成 /kəmˈpəʊz/
    *   **组合逻辑解释**：
        *   position：放+名词 → 放置的位置 → 位置
        *   deposit：向下+放 → 放进“下面的地方” → 存款/沉积
        *   compose：一起+放 → 把部分放在一起 → 组成
    *   **同源词根及音变规律**：
        *   同源：pos ↔ posit
        *   音变：o ↔ o（拼写变化）
    *   **词缀总结**：
        *   de-(向下)
        *   com-(一起)
        *   -tion(名词)

*   #### **pon**

    *   **词根**：pon
    *   **核心含义**：放、置
    *   **词源背景**：与 pos 同源，来自 *ponere* 的另一拼写形态，多用于复合词中
    *   **记忆技巧**：联想“component 组件被放在一起”
    *   **高频词（3个）**：
        *   **component** = com(一起)+pon(放)+ent(名词) → 组件 /kəmˈpəʊnənt/
        *   **opponent** = op(反)+pon(放)+ent → 对手 /əˈpəʊnənt/
        *   **postpone** = post(后)+pone(放) → 推迟 /pəʊstˈpəʊn/
    *   **组合逻辑解释**：
        *   component：一起+放 → 放在一起构成整体 → 组件
        *   opponent：反+放 → 站在对立面 → 对手
        *   postpone：后+放 → 放到后面 → 推迟
    *   **同源词根及音变规律**：
        *   同源：pon ↔ pos ↔ posit
        *   音变：o ↔ o（位置变化）
    *   **词缀总结**：
        *   com-(一起)
        *   op-(反)
        *   post-(后)

*   #### **cap**

    *   **词根**：cap
    *   **核心含义**：拿、抓
    *   **词源背景**：来自拉丁语 *capere*（抓取、拿取），曾用于描述战场上“俘获”的动作
    *   **记忆技巧**：联想 capture 抓住
    *   **高频词（3个）**：
        *   **capture** = cap(抓)+ture(名词) → 捕获 /ˈkæptʃə/
        *   **capable** = cap(拿)+able(能…) → 有能力的 /ˈkeɪpəbl/
        *   **capacity** = cap(拿)+acity(能力) → 容量；能力 /kəˈpæsəti/
    *   **组合逻辑解释**：
        *   capture：抓+名词 → 抓住 → 捕获
        *   capable：拿+能 → 能“拿下任务” → 有能力
        *   capacity：拿+能力 → 能容纳多少东西 → 容量
    *   **同源词根及音变规律**：
        *   同源：cap ↔ cept ↔ cip（拿）
        *   音变：a ↔ e ↔ i（元音级变）
    *   **词缀总结**：
        *   -ture(名词)
        *   -able(能…)
        *   -acity(能力)

***

*   #### **cept**

    *   **词根**：cept
    *   **核心含义**：拿、接收
    *   **词源背景**：源自 *capere* 的过去分词形式 *captus*，演变为 cept，表达“接受、抓住”
    *   **记忆技巧**：联想 accept 接受
    *   **高频词（3个）**：
        *   **accept** = ac(去)+cept(拿) → 接受 /əkˈsept/
        *   **concept** = con(一起)+cept(拿) → 概念 /ˈkɒnsept/
        *   **reception** = re(回)+cept(拿)+ion → 接待 /rɪˈsepʃn/
    *   **组合逻辑解释**：
        *   accept：去+拿 → 主动拿过来 → 接受
        *   concept：一起+拿 → 大脑把信息“抓起来” → 概念
        *   reception：回+拿 → 接住返回的信息 → 接待
    *   **同源词根及音变规律**：
        *   同源：cept ↔ cap ↔ cip
        *   音变：e ↔ a ↔ i
    *   **词缀总结**：
        *   ac-(去)
        *   con-(一起)
        *   -ion(名词)

***

*   #### **cip**

    *   **词根**：cip
    *   **核心含义**：拿、抓
    *   **词源背景**：来自 *capere* 的另一变体，常见于动词构词中
    *   **记忆技巧**：联想 participate 参与=拿一份
    *   **高频词（3个）**：
        *   **participate** = part(部分)+cip(拿)+ate → 参与 /pɑːˈtɪsɪpeɪt/
        *   **anticipate** = anti(前)+cip(拿)+ate → 预期 /ænˈtɪsɪpeɪt/
        *   **principal** = prin(第一)+cip(拿)+al → 最重要的 /ˈprɪnsəpl/
    *   **组合逻辑解释**：
        *   participate：部分+拿 → 拿一部分责任 → 参与
        *   anticipate：前+拿 → 提前拿到信息 → 预期
        *   principal：第一+拿 → 拿第一位置的人/事 → 最重要的
    *   **同源词根及音变规律**：
        *   同源：cip ↔ cap ↔ cept
        *   音变：i ↔ a ↔ e
    *   **词缀总结**：
        *   -ate(动词)
        *   anti-(前)
        *   prin-(第一)
