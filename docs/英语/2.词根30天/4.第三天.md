## 第三天背词根

*   #### **uni**

    *   **词根**：uni
    *   **核心含义**：一、单一
    *   **词源背景**：来自拉丁语 *unus*（一），古罗马行政中用来强调“统一、单一权力”
    *   **记忆技巧**：联想 uniform → 一样的制服
    *   **高频词（3个）**：
        *   **uniform** = uni(一)+form(形状) → 制服 /ˈjuːnɪfɔːm/
        *   **unique** = uni(一)+que(形容词) → 独一无二 /juˈniːk/
        *   **unify** = uni(一)+fy(使…) → 统一 /ˈjuːnɪfaɪ/
    *   **组合逻辑解释**：
        *   uniform：一+形状 → 都一个样 → 制服
        *   unique：一+性质 → 独一份 → 独特
        *   unify：一+使 → 让多变成一 → 统一
    *   **同源词根及音变规律**：
        *   同源：uni ↔ mono
        *   音变：u ↔ o（元音替换）
    *   **词缀总结**：
        *   -form(形状)
        *   -fy(使…)
        *   -que(形容词结尾)

*   #### **bi**

    *   **词根**：bi
    *   **核心含义**：二
    *   **词源背景**：来自拉丁语 *bis*（两次），日常用于计数、度量
    *   **记忆技巧**：联想 bicycle 两个轮子
    *   **高频词（3个）**：
        *   **bicycle** = bi(二)+cycle(轮) → 自行车 /ˈbaɪsɪkl/
        *   **bilingual** = bi(二)+lingu(语言)+al → 双语的 /ˌbaɪˈlɪŋɡwəl/
        *   **binary** = bi(二)+nary(状态) → 二元的 /ˈbaɪnəri/
    *   **组合逻辑解释**：
        *   bicycle：二+轮 → 两个轮子 → 自行车
        *   bilingual：二+语言 → 会两种语言 → 双语
        *   binary：二+状态 → 两种值 → 二进制
    *   **同源词根及音变规律**：
        *   同源：bi ↔ di
        *   音变：b ↔ d（辅音替换）
    *   **词缀总结**：
        *   -cycle(轮)
        *   -lingu(语言)
        *   -al、-ary

    *   **高频词（3个）**：
        *   **triangle** = tri(三)+angle(角) → 三角形 /ˈtraɪæŋɡl/
        *   **trinity** = tri(三)+nity(组) → 三合一 /ˈtrɪnəti/
        *   **triple** = tri(三)+ple(倍数) → 三倍 /ˈtrɪpl/
    *   **组合逻辑解释**：
        *   triangle：三+角 → 三角形
        *   trinity：三+集合 → 三者合一
        *   triple：三+倍 → 三倍
    *   **同源词根及音变规律**：
        *   同源：tri ↔ tre
        *   音变：i ↔ e（元音变化）
    *   **词缀总结**：
        *   -angle(角)
        *   -ity(集合)
        *   -ple(倍数)

*   #### **quad**

    *   **词根**：quad
    *   **核心含义**：四
    *   **词源背景**：来自拉丁语 *quattuor*（四），常见于土地划分“方形地块”
    *   **记忆技巧**：联想“quadrangle 四边形”
    *   **高频词（3个）**：
        *   **quadrangle** = quad(四)+angle(角) → 四边形 /ˈkwɒdræŋɡl/
        *   **quadruple** = quad(四)+ruple(倍) → 四倍 /ˈkwɒdrʊpl/
        *   **quadrant** = quad(四)+rant(部分) → 象限 /ˈkwɒdrənt/
    *   **组合逻辑解释**：
        *   quadrangle：四+角 → 四边形
        *   quadruple：四+倍 → 四倍
        *   quadrant：四+部分 → 四等分之一 → 象限
    *   **同源词根及音变规律**：
        *   同源：quad ↔ quart（四分之一）
        *   音变：qu ↔ qua（拼写差异）
    *   **词缀总结**：
        *   -angle(角)
        *   -ruple(倍)
        *   -rant(部分)

*   #### **cent**

    *   **词根**：cent
    *   **核心含义**：百
    *   **词源背景**：来自拉丁语 *centum*（一百），古罗马军队“百人队”即由此得名
    *   **记忆技巧**：联想 century = 一世纪=100年
    *   **高频词（3个）**：
        *   **century** = cent(百)+ury(时期) → 世纪 /ˈsentʃəri/
        *   **percent** = per(每)+cent(百) → 百分比 /pəˈsent/
        *   **centigrade** = cent(百)+grade(度) → 摄氏度 /ˈsentɪɡreɪd/
    *   **组合逻辑解释**：
        *   century：百+时期 → 一百年
        *   percent：每+百 → 每一百当中 → 百分比
        *   centigrade：百+度 → 一百等分刻度 → 摄氏度
    *   **同源词根及音变规律**：
        *   同源：cent ↔ hecto（希腊百）
        *   音变：c ↔ h（语源不同产生差异）
    *   **词缀总结**：
        *   per-(每)
        *   -grade(度)
        *   -ury(时期)

*   #### **milli**

    *   **词根**：milli
    *   **核心含义**：千、千分之一
    *   **词源背景**：来自拉丁语 *mille*（一千），用于罗马计量体系
    *   **记忆技巧**：联想 millimeter 毫米
    *   **高频词（3个）**：
        *   **millimeter** = milli(千分之一)+meter(测量) → 毫米 /ˈmɪlɪˌmiːtə/
        *   **millennium** = milli(千)+ennium(年) → 千年 /mɪˈleniəm/
        *   **milligram** = milli(千分之一)+gram(克) → 毫克 /ˈmɪlɪɡræm/
    *   **组合逻辑解释**：
        *   millimeter：千分之一+米 → 毫米
        *   millennium：千+年 → 千年
        *   milligram：千分之一+克 → 毫克
    *   **同源词根及音变规律**：
        *   同源：milli ↔ kilo（千）
        *   音变：m ↔ k（语源不同）
    *   **词缀总结**：
        *   -meter(测量)
        *   -gram(克)
        *   -ennium(年)

*   #### **multi**

    *   **词根**：multi
    *   **核心含义**：多
    *   **词源背景**：来自拉丁语 *multus*（许多），用于形容数量丰富
    *   **记忆技巧**：联想 multimedia 多媒体
    *   **高频词（3个）**：
        *   **multiply** = multi(多)+ply(折/倍) → 乘；倍增 /ˈmʌltɪplaɪ/
        *   **multimedia** = multi(多)+media(媒介) → 多媒体 /ˌmʌltiˈmiːdiə/
        *   **multinational** = multi(多)+national(国家的) → 跨国的 /ˌmʌltiˈnæʃnəl/
    *   **组合逻辑解释**：
        *   multiply：多+倍 → 变多 → 乘法
        *   multimedia：多+媒体 → 多种媒介 → 多媒体
        *   multinational：多+国家 → 涉及多国 → 跨国
    *   **同源词根及音变规律**：
        *   同源：multi ↔ poly
        *   音变：m ↔ p（辅音替换）
    *   **词缀总结**：
        *   -ply(倍/折)
        *   -media(媒介)
        *   -national(国家的)

*   #### **poly**

    *   **词根**：poly
    *   **核心含义**：多
    *   **词源背景**：来自希腊语 *polys*（许多），用于哲学、政治中表示“多元”
    *   **记忆技巧**：联想 polygon 多边形
    *   **高频词（3个）**：
        *   **polygon** = poly(多)+gon(角) → 多边形 /ˈpɒlɪɡɒn/
        *   **polyglot** = poly(多)+glot(舌) → 通晓多语者 /ˈpɒlɪɡlɒt/
        *   **polynomial** = poly(多)+nomial(项) → 多项式 /ˌpɒlɪˈnəʊmiəl/
    *   **组合逻辑解释**：
        *   polygon：多+角 → 多边形
        *   polyglot：多+舌 → 多语言
        *   polynomial：多+项 → 多项式
    *   **同源词根及音变规律**：
        *   同源：poly ↔ multi
        *   音变：p ↔ m（含义一致）
    *   **词缀总结**：
        *   -gon(角)
        *   -glot(舌/语言)
        *   -nomial(项)

*   #### **semi**

    *   **词根**：semi
    *   **核心含义**：半
    *   **词源背景**：来自拉丁语 *semi*（一半），用于农业量度，表示“半份”
    *   **记忆技巧**：联想 semicircle 半圆
    *   **高频词（3个）**：
        *   **semicircle** = semi(半)+circle(圆) → 半圆 /ˈsemisɜːkl/
        *   **semifinal** = semi(半)+final(最后) → 半决赛 /ˌsemɪˈfaɪnl/
        *   **semiannual** = semi(半)+annual(年) → 半年一次 /ˌsemiˈænjuəl/
    *   **组合逻辑解释**：
        *   semicircle：半+圆 → 半圆形
        *   semifinal：半+最后 → 决赛前的“半步” → 半决赛
        *   semiannual：半+年 → 每半年一次
    *   **同源词根及音变规律**：
        *   同源：semi ↔ hemi（希腊语“半”）
        *   音变：s ↔ h（语源不同）
    *   **词缀总结**：
        *   -circle(圆)
        *   -final(最后)
        *   -annual(年)
