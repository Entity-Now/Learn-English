### **merc**

*   **词根**：merc
*   **核心含义**：商业、交易
*   **词源背景**：来自拉丁语 *merx*，意为“商品、货物”；古罗马市场中的商贩围绕 *merx* 进行交易，该词逐渐延展为“商业活动”。
*   **记忆技巧**：联想 market（市场）里都是 merchant 做 commerce → merc＝商
*   **高频词（3个）**：
    *   **merchant** = merc(商)+ant(人) → 商人 /ˈmɜːtʃənt/
    *   **commerce** = com(共同)+merc(交易) → 商业 /ˈkɒmɜːs/
    *   **mercantile** = merc(商)+antile(形容词) → 商业的 /ˈmɜːkəntaɪl/
*   **组合逻辑解释**：
    *   merchant：商+人 → 做买卖的人 → 商人
    *   commerce：共同+交易 → 人与人彼此买卖 → 商业往来
    *   mercantile：商+性质 → 和商业相关的
*   **同源词根及音变规律**：
    *   merch（商品）、merx（货物）同源
    *   音变：x → ch（古拉丁语向中古英语变化）
*   **词缀总结**：
    *   ant（人）
    *   com-（共同）
    *   -ile（形容词，具有…性质）

### **nom**

*   **词根**：nom
*   **核心含义**：规则、分配
*   **词源背景**：源自希腊语 *nomos*，意为“法律、习俗、分配的方式”；古希腊通过“分配土地”引申为“秩序规定”。
*   **记忆技巧**：联想 “norm＝规范” 与 nom 发音接近。
*   **高频词（3个）**：
    *   **economy** = eco(家)+nom(分配) → 经济 /iːˈkɒnəmi/
    *   **autonomy** = auto(自己)+nom(规则) → 自治 /ɔːˈtɒnəmi/
    *   **nominate** = nom(规定)+ate(动词) → 任命 /ˈnɒmɪneɪt/
*   **组合逻辑解释**：
    *   economy：家+分配 → 管理家庭资源 → 后指国家经济
    *   autonomy：自己+规则 → 自己制定规则 → 自治
    *   nominate：规则+做 → 按规则推举某人
*   **同源词根及音变规律**：
    *   nom ↔ norm（规范）
    *   音变：o → or（在拉丁语吸收时常出现）
*   **词缀总结**：
    *   auto-（自我）
    *   eco-（家庭→环境→经济）
    *   -ate（动词）

### **deb**

*   **词根**：deb
*   **核心含义**：欠、债
*   **词源背景**：来自拉丁语 *debere*，意为“欠下、必须给予”，常用于罗马税务、债务记录。
*   **记忆技巧**：联想 debt（债）中 b 不发音，但能提醒“欠”。
*   **高频词（3个）**：
    *   **debit** = deb(欠)+it(名词) → 借方 /ˈdɛbɪt/
    *   **debtor** = deb(欠)+or(人) → 债务人 /ˈdɛtər/
    *   **indebted** = in(进入)+deb(欠)+ed(状态) → 负债的 /ɪnˈdɛtɪd/
*   **组合逻辑解释**：
    *   debit：欠+记录 → 记录欠款的一栏
    *   debtor：欠+人 → 欠钱的人
    *   indebted：进入+欠 → 掉进欠账状态
*   **同源词根及音变规律**：
    *   debt、debit、due 同源
    *   音变：b 在部分派生词中弱化为不发音
*   **词缀总结**：
    *   -or（人）
    *   in-（进入某状态）
    *   -ed（形容词）

### **fin**

*   **词根**：fin
*   **核心含义**：结束、界限
*   **词源背景**：来自拉丁语 *finis*，意为“界线、终点”；古罗马用它限定土地边界。
*   **记忆技巧**：电影结尾 often shows “The End” → fin＝终点
*   **高频词（3个）**：
    *   **final** = fin(结束)+al(形容) → 最终的 /ˈfaɪnəl/
    *   **finance** = fin(定限)+ance(名词) → 财政 /ˈfaɪnæns/
    *   **define** = de(向下)+fin(界限) → 下界限→下定义 /dɪˈfaɪn/
*   **组合逻辑解释**：
    *   final：结束+性质 → 最终的
    *   finance：界限+事务 → 给钱做界定 → 财政管理
    *   define：向下+界限 → 明确划界 → 下定义
*   **同源词根及音变规律**：
    *   finish、finite 同源
    *   音变：finis → fin（结尾元音弱化）
*   **词缀总结**：
    *   -al（形容词）
    *   -ance（名词）
    *   de-（向下、加强）

### **valu**

*   **词根**：valu
*   **核心含义**：价值
*   **词源背景**：源自拉丁语 *valere*，“有力量、有价值”，侧重“力量带来价值”的概念。
*   **记忆技巧**：value＝价值 → valu ＝ 值得
*   **高频词（3个）**：
    *   **value** = valu(价值)+e → 价值 /ˈvæljuː/
    *   **evaluate** = e(出)+valu(价值)+ate(动词) → 评估 /ɪˈvæljueɪt/
    *   **valuable** = valu(价值)+able(能…的) → 有价值的 /ˈvæljuəbl/
*   **组合逻辑解释**：
    *   value：价值本体
    *   evaluate：取出+价值 → 把价值“评出来”
    *   valuable：价值+能 → 具有价值
*   **同源词根及音变规律**：
    *   val、vale 同源（如 valid）
    *   音变：val → valu（加元音缓冲）
*   **词缀总结**：
    *   e-（向外）
    *   -ate（动词）
    *   -able（形容词）

### **tax**

*   **词根**：tax
*   **核心含义**：安排、整理
*   **词源背景**：来自拉丁语 *taxare*，“评估、整理”，政府用它整理账目、评估税额，后变为“征税”。
*   **记忆技巧**：tax＝政府“安排”你交的钱
*   **高频词（3个）**：
    *   **tax** = tax(安排) → 税 /tæks/
    *   **taxation** = tax(税)+ation(行为) → 征税 /tækˈseɪʃən/
    *   **syntax** = syn(一起)+tax(安排) → 句法 /ˈsɪntæks/
*   **组合逻辑解释**：
    *   tax：安排你付钱
    *   taxation：税+行为 → 征税过程
    *   syntax：一起+安排 → 把词组合安排成句 → 句法
*   **同源词根及音变规律**：
    *   taxare → tax
    *   音变：taxare 的 -are 消失，保留词干 tax
*   **词缀总结**：
    *   syn-（一起）
    *   -ation（名词行为）

### **patr**

*   **词根**：patr
*   **核心含义**：父亲、家族
*   **词源背景**：源于拉丁语 *pater*，“父亲”；古罗马家族由家族父权者（patriarch）掌权。
*   **记忆技巧**：联想 paternal（父亲的）
*   **高频词（3个）**：
    *   **patron** = patr(父系)+on(人) → 资助人 /ˈpeɪtrən/
    *   **patriarch** = patr(父)+arch(统治) → 家长 /ˈpeɪtriɑːk/
    *   **patriot** = patr(祖国=父土)+iot(人) → 爱国者 /ˈpeɪtriət/
*   **组合逻辑解释**：
    *   patron：父+人 → 像“父亲”一样支持你的人 → 赞助人
    *   patriarch：父+统治 → 家中掌权者
    *   patriot：父土+人 → 热爱祖先土地的人 → 爱国者
*   **同源词根及音变规律**：
    *   pater（父）→ patr（词干）
    *   音变：e → r（拉丁语词干化）
*   **词缀总结**：
    *   -arch（统治）
    *   -on（人）
    *   -ot/ -iot（人）

### **matr**

*   **词根**：matr
*   **核心含义**：母亲
*   **词源背景**：源于拉丁语 *mater*，“母亲”；古罗马继承制度常以母亲的家系作为一类身份依据。
*   **记忆技巧**：联想 mother → matr 音似
*   **高频词（3个）**：
    *   **maternal** = matr(母)+nal(形容) → 母亲的 /məˈtɜːnəl/
    *   **matriarch** = matr(母)+arch(统治) → 女家长 /ˈmeɪtriɑːk/
    *   **matrix** = matr(母体)+ix(名词) → 母体 /ˈmeɪtrɪks/
*   **组合逻辑解释**：
    *   maternal：母+性质 → 母亲的
    *   matriarch：母+统治 → 女性家长
    *   matrix：母体 → 原本寓意“生成万物的母体”
*   **同源词根及音变规律**：
    *   mater → matr
    *   音变：e 脱落形成词干
*   **词缀总结**：
    *   -nal（形容词）
    *   -arch（统治）
    *   -ix（名词）

### **corpor**

*   **词根**：corpor
*   **核心含义**：身体、共同体
*   **词源背景**：源自拉丁语 *corpus*，“身体”，在罗马法中被延伸为“具有统一身体的组织”，即公司。
*   **记忆技巧**：corporation＝公司像一个“大身体”
*   **高频词（3个）**：
    *   **corporation** = corpor(团体)+ation(机构) → 公司 /ˌkɔːpəˈreɪʃən/
    *   **corporate** = corpor(团体)+ate(形容) → 公司的 /ˈkɔːpərət/
    *   **incorporate** = in(进入)+corpor(整体)+ate(动词) → 合并 /ɪnˈkɔːpəreɪt/
*   **组合逻辑解释**：
    *   corporation：身体+机构 → 像一个整体运作的组织
    *   corporate：身体+性质 → 具有整体性的 → 公司的
    *   incorporate：进入+整体 → 合并进一个组织
*   **同源词根及音变规律**：
    *   corpus → corpor → corp
    *   音变：u → o（弱化）
*   **词缀总结**：
    *   in-（进入）
    *   -ation（名词）
    *   -ate（动词/形容）

### **priv**

*   **词根**：priv
*   **核心含义**：私人、个人
*   **词源背景**：来自拉丁语 *privus*，“单独的、个人的”，古罗马政治中用于区分“公有（public）”与“私有（privus）”。
*   **记忆技巧**：private＝私人 → priv＝私
*   **高频词（3个）**：
    *   **private** = priv(个人)+ate(形容) → 私人的 /ˈpraɪvət/
    *   **privilege** = priv(特权来自“个人”)+lege(法律) → 特权 /ˈprɪvəlɪdʒ/
    *   **deprive** = de(离开)+priv(个人/拥有) → 剥夺 /dɪˈpraɪv/
*   **组合逻辑解释**：
    *   private：个人+性质 → 私人的
    *   privilege：个人+法律 → 给个人的特别待遇 → 特权
    *   deprive：离开+个人 → 把属于个人的拿走 → 剥夺
*   **同源词根及音变规律**：
    *   privus → priv
    *   音变：v 在派生词中保持稳定
*   **词缀总结**：
    *   de-（离开、否定）
    *   -ate（形容）
    *   -lege（法律相关）
