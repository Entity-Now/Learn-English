### 今日词根学习（第十天：10个随机常用词根）

#### 1. 词根: uni/mono + 同义/同源词根（uni/mono） + /ˈjuːni/ /ˈmɑːnoʊ/
2. 核心含义: 一、单一  
3. 词源背景: uni来自拉丁语“unus”（一），mono来自希腊语“monos”（单一），常用于统一、独一词汇  
4. 记忆技巧: 想象“universe”是一个“uni”大整体，“monologue”是一个人“mono”自言自语  
5. 3个高频词:
   - uniform /ˈjuːnɪfɔːrm/：uni（一） + form（形状），指“制服/统一的”，因为它使形状“成一”种样式
   - universe /ˈjuːnɪvɜːrs/：uni（一） + verse（转），意思是“宇宙”，所有东西“转成一”体
   - monologue /ˈmɑːnəlɔːɡ/：mono（单一） + log（说），表示“独白”，一个人“单一说”的话
6. 同源词根的常见音变规律: uni（拉丁常用）；mono（希腊形式，u→mo音变）  
7. 高频词中出现的词缀总结: form /fɔːrm/（形状）；verse /vɜːrs/（转）；log /lɔːɡ/（说）

#### 2. 词根: bi/di + 同义/同源词根（bi/di） + /baɪ/ /daɪ/
2. 核心含义: 二、双  
3. 词源背景: bi来自拉丁语“bis”（两次），di来自希腊语“dis”（二），常用于双重、两部分词汇  
4. 记忆技巧: 想到“bicycle”有“bi”两个轮子，“divide”像“di”分成两半  
5. 3个高频词:
   - bicycle /ˈbaɪsɪkl/：bi（二） + cycle（轮），指“自行车”，因为有两个“轮”
   - divide /dɪˈvaɪd/：di（二） + vide（分开），意思是“划分”，把东西“分成二”或多
   - dialogue /ˈdaɪəlɔːɡ/：dia（通过） + log（说），但常视为di（二） + a + log，表示“对话”，两人“二”说交流
6. 同源词根的常见音变规律: bi（拉丁常用）；di（希腊形式，b→d音变）  
7. 高频词中出现的词缀总结: cycle /ˈsɪkl/（轮）；vide /vaɪd/（看/分开）；dia /ˈdaɪə/（通过）；log /lɔːɡ/（说）

#### 3. 词根: tri + 同义/同源词根（tri/tri） + /traɪ/
2. 核心含义: 三  
3. 词源背景: 来自拉丁/希腊语“tres/tria”（三），常用于三角、三倍等词汇  
4. 记忆技巧: 想象“triangle”有“tri”三个角，“tripod”有“三”脚  
5. 3个高频词:
   - triangle /ˈtraɪæŋɡl/：tri（三） + angle（角），指“三角形”，因为有“三”个角
   - tricycle /ˈtraɪsɪkl/：tri（三） + cycle（轮），意思是“三轮车”，有“三”个轮
   - trilogy /ˈtrɪlədʒi/：tri（三） + logy（研究/系列），表示“三部曲”，“三”个相关的故事系列
6. 同源词根的常见音变规律: tri稳定，无明显变体  
7. 高频词中出现的词缀总结: angle /ˈæŋɡl/（角）；cycle /ˈsɪkl/（轮）；logy /ˈlədʒi/（系列/研究）

#### 4. 词根: quad + 同义/同源词根（quad/quadr） + /kwɑːd/
2. 核心含义: 四  
3. 词源背景: 来自拉丁语“quattuor”（四），常用于四方、四倍等词汇  
4. 记忆技巧: 想到“quadrangle”是一个“四”角形，“quadruple”是“四”倍  
5. 3个高频词:
   - quadruple /kwɑːˈdruːpl/：quad（四） + ruple（倍），指“四倍的/四重”，使数量“成四”倍
   - quadrant /ˈkwɑːdrənt/：quad（四） + rant（部分），意思是“象限/四分之一”，空间分成“四”部分
   - quadruped /ˈkwɑːdrʊped/：quad（四） + ped（脚），表示“四足动物”，有“四”只脚的生物
6. 同源词根的常见音变规律: quad → quadr（扩展形式，加r，如quadr-ant）  
7. 高频词中出现的词缀总结: ruple /ˈruːpl/（倍）；rant /rənt/（部分）；ped /ped/（脚）

#### 5. 词根: cent + 同义/同源词根（centi/cent） + /sɛnt/
2. 核心含义: 百  
3. 词源背景: 来自拉丁语“centum”（一百），常用于百年、百分等词汇  
4. 记忆技巧: 想象“century”是一个“百”年，“centimeter”是“百”分之一米  
5. 3个高频词:
   - century /ˈsɛntʃəri/：cent（百） + ury（时期），指“世纪/百年”，一百年的时期
   - percent /pərˈsɛnt/：per（每） + cent（百），意思是“百分比”，“每百”中的比例
   - centennial /sɛnˈtɛniəl/：cent（百） + ennial（年的），表示“百年纪念的”，与“百”年相关的纪念
6. 同源词根的常见音变规律: cent → centi（小单位形式，加i，如centi-meter）  
7. 高频词中出现的词缀总结: ury /əri/（时期）；per /pər/（每）；ennial /ˈɛniəl/（年的）

#### 6. 词根: mill + 同义/同源词根（milli/mill） + /mɪl/
2. 核心含义: 千  
3. 词源背景: 来自拉丁语“mille”（一千），常用于千年、千分等词汇  
4. 记忆技巧: 想到“millennium”是一个“千”年，“millimeter”是“千”分之一米  
5. 3个高频词:
   - millennium /mɪˈlɛniəm/：mill（千） + ennium（年），指“千年”，一千年的时期
   - millimeter /ˈmɪlɪmiːtər/：milli（千） + meter（米），意思是“毫米”，“千”分之一米
   - millipede /ˈmɪlɪpiːd/：milli（千） + pede（脚），表示“千足虫”，像有“千”只脚的虫子
6. 同源词根的常见音变规律: mill → milli（小单位形式，加i）  
7. 高频词中出现的词缀总结: ennium /ˈɛniəm/（年）；meter /ˈmiːtər/（米）；pede /piːd/（脚）

#### 7. 词根: dom + 同义/同源词根（domi/dom） + /dɑːm/
2. 核心含义: 家、统治  
3. 词源背景: 来自拉丁语“domus”（家）、“dominus”（主人），常用于家庭、支配等词汇  
4. 记忆技巧: 想象“domestic”像在“dom”家里的事，“dominate”是“统治”如主人  
5. 3个高频词:
   - domestic /dəˈmɛstɪk/：dom（家） + estic（相关的），指“国内的/家庭的”，与“家”相关的或内部的
   - dominate /ˈdɑːmɪneɪt/：dom（统治） + inate（使…），意思是“支配”，像主人“统治”他人
   - domain /doʊˈmeɪn/：dom（家/统治） + ain（领域），表示“领域/域名”，“统治”下的范围
6. 同源词根的常见音变规律: dom稳定；domi（名词形式，加i，如domi-cile）  
7. 高频词中出现的词缀总结: estic /ˈɛstɪk/（相关的）；inate /ɪneɪt/（使…动词）；ain /eɪn/（领域名词）

#### 8. 词根: liber + 同义/同源词根（liberi/liber） + /ˈlɪbər/
2. 核心含义: 自由  
3. 词源背景: 来自拉丁语“liber”（自由的），常用于解放、自由主义等词汇  
4. 记忆技巧: 想到“liberty”就是“自由”，“deliver”是“送”出自由  
5. 3个高频词:
   - liberty /ˈlɪbərti/：liber（自由） + ty（状态），指“自由”，处于“自由”的状态
   - liberal /ˈlɪbərəl/：liber（自由） + al（相关的），意思是“自由的/开明的”，支持“自由”观点的
   - liberate /ˈlɪbəreɪt/：liber（自由） + ate（使…），表示“解放”，使某人“成自由”
6. 同源词根的常见音变规律: liber稳定；liberi（复数/加强形式，加i）  
7. 高频词中出现的词缀总结: ty /ti/（状态名词）；al /əl/（相关的形容词）；ate /eɪt/（使…动词）

#### 9. 词根: soci + 同义/同源词根（socio/soci） + /ˈsoʊʃi/
2. 核心含义: 社会、伙伴  
3. 词源背景: 来自拉丁语“socius”（伙伴），常用于社会、关联等词汇  
4. 记忆技巧: 想象“society”是一群“soci”伙伴在一起，“associate”就是“联”成伙伴  
5. 3个高频词:
   - society /səˈsaɪəti/：soci（社会） + ety（集体），指“社会”，伙伴“集体”
   - social /ˈsoʊʃl/：soci（社会） + al（相关的），意思是“社会的/社交的”，与“伙伴”互动相关的
   - associate /əˈsoʊʃieɪt/：as（向） + soci（伙伴） + ate，表示“交往/联合”，向“伙伴”靠拢
6. 同源词根的常见音变规律: soci → socio（学术形式，加o）  
7. 高频词中出现的词缀总结: ety /əti/（集体名词）；al /l/（相关的形容词）；as /əs/（向）；ate /eɪt/（动词）

#### 10. 词根: civ + 同义/同源词根（civi/civ） + /sɪv/
2. 核心含义: 公民、城市  
3. 词源背景: 来自拉丁语“civis”（公民），常用于文明、公民等词汇  
4. 记忆技巧: 想到“civil”像“公民”间的礼貌，“civilization”是从“城市”公民发展出的文化  
5. 3个高频词:
   - civil /ˈsɪvl/：civ（公民） + il（相关的），指“公民的/文明的”，与“公民”社会相关的
   - civilization /ˌsɪvəlɪˈzeɪʃn/：civ（公民） + ilization（过程），意思是“文明”，“公民”社会的发展过程
   - civilian /sɪˈvɪliən/：civ（公民） + ilian（人后缀），表示“平民/公民的”，非军人的“公民”
6. 同源词根的常见音变规律: civ → civi（名词形式，加i）  
7. 高频词中出现的词缀总结: il /ɪl/（相关的形容词）；ilization /ɪˈzeɪʃn/（过程名词）；ilian /ˈɪliən/（人名词）

### 小作文：Unity in a Diverse World

In our society, it's vital to promote civil behavior among civilians, creating a uniform sense of community. Imagine a bicycle for two or a tricycle for kids—simple inventions that bring people together in pairs or trios. Over centuries and millennia, civilizations have evolved, from primitive tribes to modern domains where liberty reigns. We must liberate minds from old biases, associating with diverse partners to build a benevolent world. Even in a universe so vast, focusing on one primary goal can dominate chaos, like a liberal approach to unity. Though challenges prevail, embracing social bonds ensures we don't venture alone.

（注：vital /ˈvaɪtl/ (重要的); community /kəˈmjuːnəti/ (社区); evolved /ɪˈvɑːlvd/ (进化); biases /ˈbaɪəsɪz/ (偏见); chaos /ˈkeɪɑːs/ (混乱)）

中文总结：这篇文章强调在多元世界中的统一：促进公民间的文明行为、创造统一社区，通过自行车或三轮车等发明连接人们，经过世纪和千年，文明从原始部落进化到现代领域，强调自由、解放思想、与伙伴交往、构建仁慈世界，即使宇宙广阔，专注于首要目标能支配混乱，像自由主义方法，最终通过社会纽带确保不孤单。
