#### **bio**

* **核心含义**：生命
* **词源背景**：源自希腊语 *bios*，意为“生命”，古希腊哲学中用于探讨生物存在与自然规律
* **记忆技巧**：联想 biology 生物学，研究生命的学科
* **高频词**：
    * **biology** = bio(生命)+logy(学科) → 生物学 /baɪˈɒlədʒi/
    * **biography** = bio(生命)+graphy(写) → 传记 /baɪˈɒɡrəfi/
    * **antibiotic** = anti(反)+bio(生命)+tic(形容词) → 抗生素 /ˌæntibaɪˈɒtɪk/
* **词缀总结**：常见前缀 anti-(反)、micro-(微)，后缀 -logy(学科)、-graphy(写作)、-tic(形容词)

#### **geo**

* **核心含义**：地球
* **词源背景**：源自希腊语 *gē*，意为“地”，古希腊用于描述土地测绘与地理分布
* **记忆技巧**：联想 geography 地理学，研究地球的学科
* **高频词**：
    * **geography** = geo(地球)+graphy(写) → 地理学 /dʒiˈɒɡrəfi/
    * **geology** = geo(地球)+logy(学科) → 地质学 /dʒiˈɒlədʒi/
    * **geometry** = geo(地球)+metry(测量) → 几何学 /dʒiˈɒmətri/
* **词缀总结**：常见后缀 -graphy(写作)、-logy(学科)、-metry(测量)，前缀 geo-固定表示地球

#### **astro**

* **核心含义**：星
* **词源背景**：源自希腊语 *astron*，意为“星星”，古代用于占星术和天文学研究
* **记忆技巧**：联想 astronaut 宇航员，去星星的旅行者
* **高频词**：
    * **astronomy** = astro(星)+nomy(法则) → 天文学 /əˈstrɒnəmi/
    * **astronaut** = astro(星)+naut(水手) → 宇航员 /ˈæstrənɔːt/
    * **astrology** = astro(星)+logy(学科) → 占星学 /əˈstrɒlədʒi/
* **词缀总结**：常见后缀 -nomy(法则)、-logy(学科)、-naut(水手)，均与探索或研究相关

    * **macrocosm** = macro(宏)+cosm(宇宙) → 宏观世界 /ˈmækrəʊkɒzəm/

* **词缀总结**：常见后缀 -cosm(宇宙)、-scopic(看)、-economics(经济学)，前缀 macro-表示宏大

#### **chrono**

* **核心含义**：时间
* **词源背景**：源自希腊语 *chronos*，意为“时间”，古代用于描述时间顺序和计时
* **记忆技巧**：联想 chronology 年代学，研究时间顺序
* **高频词**：

    * **synchronize** = syn(共同)+chrono(时间)+ize(动词) → 同步 /ˈsɪŋkrənaɪz/
    * **chronicle** = chrono(时间)+cle(名词) → 编年史 /ˈkrɒnɪkl/
* **词缀总结**：常见前缀 syn-(共同)、后缀 -logy(学科)、-ize(动词)、-cle(名词)，均与时间相关
