### 今日词根学习（第四天：10个随机常用词根，覆盖日常、携带、书写、身体、科技等领域）

#### 1. 词根: port + 同义/同源词根（port/porto） + /pɔːrt/
2. 核心含义：拿、带、携带  
3. 词源背景：来自拉丁语“portare”（携带、运送），广泛用于交通、进出口等词汇  
4. 记忆技巧：想象一个“porter”（搬运工）在机场“port”着行李到处跑  
5. 3个高频词：
   - transport /ˈtrænspɔːrt/：trans（穿过） + port（携带），指“运输”，因为把东西“携带穿过”空间
   - portable /ˈpɔːrtəbl/：port（携带） + able（能…的），意思是“便携的”，能轻易“携带”的
   - import /ˈɪmpɔːrt/：im（进入） + port（携带），指“进口”，把货物“携带进入”国家
6. 同源词根的常见音变规律：port基本稳定，偶尔变porto（古典形式，少见）  
7. 高频词中出现的词缀总结：trans /træns/（穿过）；able /əbl/（能…的）；im /ɪm/（进入）

#### 2. 词根: cap/cept + 同义/同源词根（cap/capt/cept） + /kæp/ /sɛpt/
2. 核心含义：拿、抓、接受  
3. 词源背景：来自拉丁语“capere”（拿、抓住），cept是其过去分词形式，常见于接受、概念等词  
4. 记忆技巧：想到“accept”像“接住”别人扔来的东西，“capture”则是“抓住”  
5. 3个高频词：
   - accept /əkˈsɛpt/：ac（向） + cept（拿），指“接受”，把东西“拿向”自己
   - capture /ˈkæptʃər/：capt（拿/抓） + ure（动作名词），意思是“捕获”，直接“抓住”
   - concept /ˈkɑːnsɛpt/：con（一起） + cept（拿），指“概念”，把想法“一起拿”在脑中
6. 同源词根的常见音变规律：cap → capt（加强/过去形式）；cept（过去分词形式，c→p音变常见）  
7. 高频词中出现的词缀总结：ac /ək/（向）；ure /ər/（动作/结果）；con /kən/（一起）

#### 3. 词根: graph + 同义/同源词根（graph/gra） + /ɡræf/
2. 核心含义：写、画、记录  
3. 词源背景：来自希腊语“graphein”（写、刻画），用于记录、图形相关词汇  
4. 记忆技巧：把“graph”想象成“画图表”（graph），所有带graph的词都跟“写或画”有关  
5. 3个高频词：
   - photograph /ˈfoʊtəɡræf/：photo（光） + graph（写/记录），指“照片”，用光“记录”影像
   - autograph /ˈɔːtəɡræf/：auto（自己） + graph（写），意思是“亲笔签名”，自己“写”的
   - biography /baɪˈɑːɡrəfi/：bio（生命） + graphy（记录），指“传记”，记录一生的“生命历程”
6. 同源词根的常见音变规律：graph → graphy（名词形式，加y）；gra（少见简化）  
7. 高频词中出现的词缀总结：photo /ˈfoʊtoʊ/（光）；auto /ˈɔːtoʊ/（自己）；bio /baɪoʊ/（生命）；y /i/（名词化）

#### 4. 词根: scrib/script + 同义/同源词根（scrib/script） + /skrɪb/
2. 核心含义：写  
3. 词源背景：来自拉丁语“scribere”（写），script是其过去分词形式，常用于书面相关词  
4. 记忆技巧：想到“script”就是电影“剧本”（写出来的），scribe像古代“抄写员”  
5. 3个高频词：
   - describe /dɪˈskraɪb/：de（完全） + scribe（写），指“描述”，把细节“完全写”出来
   - manuscript /ˈmænjʊskrɪpt/：manu（手） + script（写），意思是“手稿”，用手“写”的稿子
   - prescription /prɪˈskrɪpʃn/：pre（前） + script（写） + ion，医生“提前写”好的药方
6. 同源词根的常见音变规律：scrib（动词形式）；script（名词/过去形式，b→p音变）  
7. 高频词中出现的词缀总结：de /dɪ/（完全）；manu /ˈmænju/（手）；pre /prɪ/（前）；ion /ən/（结果名词）

#### 5. 词根: man(u) + 同义/同源词根（man/manu） + /ˈmænju/
2. 核心含义：手  
3. 词源背景：来自拉丁语“manus”（手），用于手工、制造相关词汇  
4. 记忆技巧：联想到“manual”就是“用手”操作的，“manufacture”是用手“做”出来的  
5. 3个高频词：
   - manual /ˈmænjuəl/：manu（手） + al（形容词），指“手工的/手册”，与“手”相关的
   - manufacture /ˌmænjuˈfæktʃər/：manu（手） + fact（做），意思是“制造”，最初指“手工制作”
   - manuscript /ˈmænjʊskrɪpt/：manu（手） + script（写），指“手稿”，用“手”写成的
6. 同源词根的常见音变规律：man → manu（完整形式，常加u）；man（简化）  
7. 高频词中出现的词缀总结：al /əl/（形容词）；fact /fækt/（做）；script /skrɪpt/（写）

#### 6. 词根: ped/pod + 同义/同源词根（ped/pod） + /pɛd/ /pɑːd/
2. 核心含义：脚  
3. 词源背景：ped来自拉丁语“pes”（脚），pod来自希腊语“pous”（脚），用于脚相关词汇  
4. 记忆技巧：想到“pedal”是“脚踏板”，“tripod”是“三脚架”  
5. 3个高频词：
   - pedal /ˈpɛdl/：ped（脚） + al（相关），指“脚踏板”，用“脚”操作的
   - pedestrian /pəˈdɛstriən/：ped（脚） + estrian（人），意思是“行人”，用“脚”走路的人
   - tripod /ˈtraɪpɑːd/：tri（三） + pod（脚），指“三脚架”，有“三只脚”的支架
6. 同源词根的常见音变规律：ped（拉丁常用）；pod（希腊形式，e→o音变）  
7. 高频词中出现的词缀总结：al /əl/（相关）；estrian /ˈɛstriən/（人）；tri /traɪ/（三）

#### 7. 词根: therm + 同义/同源词根（thermo） + /θɜːrm/
2. 核心含义：热  
3. 词源背景：来自希腊语“therme”（热），用于温度、热能相关词汇  
4. 记忆技巧：把“therm”想象成“温度计”（thermometer），所有therm词都跟“热”有关  
5. 3个高频词：
   - thermal /ˈθɜːrml/：therm（热） + al（形容词），指“热的/热量的”，与“热”相关的
   - thermometer /θərˈmɑːmɪtər/：therm（热） + meter（测量），意思是“温度计”，测量“热”的仪器
   - thermostat /ˈθɜːrməstæt/：therm（热） + stat（站立/固定），指“恒温器”，让温度“固定”在某值
6. 同源词根的常见音变规律：therm → thermo（学术/前缀形式，加o）  
7. 高频词中出现的词缀总结：al /l/（形容词）；meter /ˈmiːtər/（测量）；stat /stæt/（固定）

#### 8. 词根: hydro + 同义/同源词根（hydr） + /ˈhaɪdroʊ/
2. 核心含义：水  
3. 词源背景：来自希腊语“hydor”（水），用于水相关科学词汇  
4. 记忆技巧：想到“hydrate”就是“加水”，“dehydrate”是“脱水”  
5. 3个高频词：
   - hydrate /ˈhaɪdreɪt/：hydr（水） + ate（使…），指“使水合/水合物”，让东西“加水”
   - hydrogen /ˈhaɪdrədʒən/：hydr（水） + gen（产生），意思是“氢”，因为氢与氧结合产生“水”
   - dehydrate /diːˈhaɪdreɪt/：de（去除） + hydrate（水合），指“脱水”，去除“水”
6. 同源词根的常见音变规律：hydro → hydr（前缀简化形式）  
7. 高频词中出现的词缀总结：ate /eɪt/（使…）；gen /dʒən/（产生）；de /diː/（去除）

#### 9. 词根: photo + 同义/同源词根（phot） + /ˈfoʊtoʊ/
2. 核心含义：光  
3. 词源背景：来自希腊语“phos/phot”（光），用于摄影、光合作用等  
4. 记忆技巧：把“photo”直接想到“照片”，因为照片是“用光”记录的  
5. 3个高频词：
   - photograph /ˈfoʊtəɡræf/：photo（光） + graph（记录），指“照片”，用“光”记录影像
   - photosynthesis /ˌfoʊtoʊˈsɪnθəsɪs/：photo（光） + syn（一起） + thesis（放置），植物用“光”合成养分
   - photon /ˈfoʊtɑːn/：phot（光） + on（粒子），指“光子”，光的粒子单位
6. 同源词根的常见音变规律：photo → phot（简化前缀）  
7. 高频词中出现的词缀总结：graph /ɡræf/（记录）；syn /sɪn/（一起）；thesis /ˈθiːsɪs/（放置）；on /ɑːn/（粒子）

#### 10. 词根: cracy/crat + 同义/同源词根（crat/cracy） + /ˈkreɪsi/
2. 核心含义：统治、权力  
3. 词源背景：来自希腊语“kratos”（力量、统治），用于各种政体名称  
4. 记忆技巧：想到“democracy”=“人民统治”（demo + cracy），crat是“统治者”  
5. 3个高频词：
   - democracy /dɪˈmɑːkrəsi/：demo（人民） + cracy（统治），指“民主”，人民“统治”
   - bureaucracy /bjʊˈrɑːkrəsi/：bureau（办公室） + cracy（统治），意思是“官僚制度”，由官员“统治”
   - aristocrat /əˈrɪstəkræt/：aristo（最好） + crat（统治者），指“贵族”，最好的“统治者”
6. 同源词根的常见音变规律：cracy（统治形式名词）；crat（统治者）  
7. 高频词中出现的词缀总结：demo /ˈdɛmoʊ/（人民）；bureau /bjʊˈroʊ/（办公室）；aristo /əˈrɪstoʊ/（最好）

### 小作文：A Busy Day of Carrying and Creating

Every morning, I carry my portable backpack full of books and a manuscript I need to finish. As a student, I must accept new concepts quickly in class, where we discuss democracy and how different forms of cracy work in the world. During break, I take a photograph of my friends standing on the grass, their feet like a natural tripod. Later, in the lab, we study thermal energy and how a thermometer measures heat, while learning about hydrogen and hydro power as clean energy sources. In the afternoon, I pedal my bicycle home, feeling the manual effort in my legs. By evening, I manufacture a simple drawing with light from my lamp, capturing the day's moments like a small photo album in my mind.

（注：backpack /ˈbækpæk/（背包）；lab /læb/（实验室）；clean /kliːn/（清洁的）；album /ˈælbəm/（相册））

**中文总结**：这篇文章描述了作者忙碌的一天：携带便携背包、接受新概念、学习民主制度、拍照、研究热能与氢能源、骑自行车回家，最后用灯光“制造”一幅小画，捕捉一天的回忆，像一本心灵相册。
