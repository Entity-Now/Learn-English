### 今日词根学习（第十一天：10个随机常用词根）

#### 1. 词根: hosp/host + 同义/同源词根（hospit/host） + /hɑːsp/ /hoʊst/
2. 核心含义: 主人、客人  
3. 词源背景: 来自拉丁语“hospes”（主人或客人），反映古代好客文化中主人与客人的双重角色  
4. 记忆技巧: 想象“hospital”是一个“host”像主人般招待病“客”的地方  
5. 3个高频词:
   - hospital /ˈhɑːspɪtl/：hosp（主人/客人） + ital（地方后缀），指“医院”，因为医院像“主人”照顾“客人”般提供庇护和治疗
   - host /hoʊst/：host（主人），意思是“主人/主持人”，因为它是接待“客人”的人
   - hospitality /ˌhɑːspɪˈtæləti/：hospit（主人/客人） + ality（品质后缀），表示“好客/招待”，品质上像“主人”对待“客人”的友好
6. 同源词根的常见音变规律: hosp → host（p省略为st，如hosp-ital vs. host-el）；hospit（扩展形式，加it）  
7. 高频词中出现的词缀总结: ital /ɪtl/（地方名词）；ality /ˈæləti/（品质名词）

#### 2. 词根: loc + 同义/同源词根（locus/loc） + /lɑːk/
2. 核心含义: 地方、位置  
3. 词源背景: 来自拉丁语“locus”（地方），常用于定位、分配等词汇  
4. 记忆技巧: 想到“location”像你“lock”住的那个“地方”  
5. 3个高频词:
   - location /loʊˈkeɪʃn/：loc（地方） + ation（结果后缀），指“位置/地点”，因为它是“地方”的具体结果或指定
   - local /ˈloʊkl/：loc（地方） + al（相关的），意思是“当地的”，属于特定“地方”的特征
   - allocate /ˈæləkeɪt/：al（向） + loc（地方） + ate（动词后缀），表示“分配”，把资源“向地方”放置
6. 同源词根的常见音变规律: loc稳定；locus（原拉丁名词形式，加us）  
7. 高频词中出现的词缀总结: ation /eɪʃn/（结果名词）；al /l/（相关的形容词）；ate /eɪt/（使…动词）

#### 3. 词根: place + 同义/同源词根（plac/plac） + /pleɪs/
2. 核心含义: 地方  
3. 词源背景: 来自拉丁语“platea”（宽阔的地方），演变为英语中位置、替换等词汇  
4. 记忆技巧: 想象“place”直接就是一个“地方”，“replace”是“再放”一个  
5. 3个高频词:
   - place /pleɪs/：直接使用词根，意思是“地方/放置”，核心是指定或放的“地方”
   - replace /rɪˈpleɪs/：re（重新） + place（地方），指“替换”，在原“地方”重新放置
   - displace /dɪsˈpleɪs/：dis（分开） + place（地方），表示“取代/移位”，从“地方”分开或移走
6. 同源词根的常见音变规律: place稳定；plac（简化形式，少见）  
7. 高频词中出现的词缀总结: re /rɪ/（重新）；dis /dɪs/（分开）

#### 4. 词根: sit + 同义/同源词根（sit/sed） + /sɪt/
2. 核心含义: 坐、位置  
3. 词源背景: 来自拉丁语“sedere”（坐），sit为其变体，常用于位置、沉积等词汇  
4. 记忆技巧: 想到“sit”就是“坐”下来，“site”是“坐”落的位置  
5. 3个高频词:
   - site /saɪt/：sit（位置） + e（连接），指“地点/网站”，固定的“位置”
   - situation /ˌsɪtʃuˈeɪʃn/：sit（位置） + uation（状态后缀），意思是“情况/位置”，当前的“位置”状态
   - reside /rɪˈzaɪd/：re（加强） + sid（=sit 坐） + e，表示“居住”，加强“坐”在某处的位置
6. 同源词根的常见音变规律: sit → sed（过去形式，t→d音变，如sed-iment）  
7. 高频词中出现的词缀总结: e /aɪt/（连接）；uation /uˈeɪʃn/（状态名词）；re /rɪ/（加强）

#### 5. 词根: pos/posit + 同义/同源词根（pose/posit） + /pɑːz/ /ˈpɑːzɪt/
2. 核心含义: 放置、位置  
3. 词源背景: 来自拉丁语“ponere”（放置），posit为其过去分词，常用于位置、组成等词汇  
4. 记忆技巧: 想象“position”是你“放置”身体的位置，“compose”是“一起放置”  
5. 3个高频词:
   - position /pəˈzɪʃn/：posit（放置） + ion（结果后缀），指“位置/职位”，被“放置”的地方或角色
   - compose /kəmˈpoʊz/：com（一起） + pose（放置），意思是“组成/作曲”，把元素“一起放置”
   - deposit /dɪˈpɑːzɪt/：de（下） + posit（放置），表示“存款/沉淀”，把东西“放置”下去
6. 同源词根的常见音变规律: pos（动词）；posit（过去/名词，加it）  
7. 高频词中出现的词缀总结: ion /ɪʃn/（结果名词）；com /kəm/（一起）；de /dɪ/（下）

#### 6. 词根: sta/stat + 同义/同源词根（stat/sta） + /steɪ/ /stæt/
2. 核心含义: 站立、状态  
3. 词源背景: 来自拉丁语“stare”（站立），常用于状态、稳定等词汇  
4. 记忆技巧: 想到“state”是一个“站立”的国家形态，“stable”就是“站”得稳  
5. 3个高频词:
   - state /steɪt/：stat（站立） + e（连接），指“状态/国家”，某种“站立”的形式
   - status /ˈsteɪtəs/：stat（站立） + us（名词后缀），意思是“地位/状态”，所“站”的位置
   - stable /ˈsteɪbl/：sta（站立） + ble（能…的），表示“稳定的”，能“站立”稳固的
6. 同源词根的常见音变规律: sta（基础）；stat（名词形式，加t）  
7. 高频词中出现的词缀总结: e /t/（连接）；us /əs/（名词）；ble /bl/（能…的形容词）

#### 7. 词根: mob/mot/mov + 同义/同源词根（mobi/mov） + /moʊb/ /moʊt/
2. 核心含义: 动、移动  
3. 词源背景: 来自拉丁语“movere”（移动），常用于运动、汽车等词汇  
4. 记忆技巧: 想象“move”就是“动”起来，“mobile”是可以“动”的  
5. 3个高频词:
   - mobile /ˈmoʊbaɪl/：mob（动） + ile（能…的），指“移动的/手机”，能“动”的设备
   - motion /ˈmoʊʃn/：mot（动） + ion（过程后缀），意思是“运动/动作”，“动”的过程
   - remove /rɪˈmuːv/：re（回） + mov（动），表示“移除”，“动”回去或移开
6. 同源词根的常见音变规律: mob（形容词）；mot（名词）；mov（动词）  
7. 高频词中出现的词缀总结: ile /aɪl/（能…的形容词）；ion /ʃn/（过程名词）；re /rɪ/（回）

#### 8. 词根: via/voy + 同义/同源词根（vi/voy） + /ˈvaɪə/ /vɔɪ/
2. 核心含义: 路、途径  
3. 词源背景: 来自拉丁语“via”（道路），voy为法语变体，常用于旅行、方式等词汇  
4. 记忆技巧: 想到“via”就是“通过路”，“voyage”是长“路”旅行  
5. 3个高频词:
   - via /ˈvaɪə/：直接使用词根，意思是“通过/经由”，表示“路”的途径
   - voyage /ˈvɔɪɪdʒ/：voy（路） + age（行动后缀），指“航行/长途旅行”，漫长的“路”途
   - conveyor /kənˈveɪər/：con（一起） + vey（=voy 路） + or（物后缀），表示“传送带”，把东西“一起带路”的装置
6. 同源词根的常见音变规律: via（基础）；voy（法语形式，i→oy音变）  
7. 高频词中出现的词缀总结: age /ɪdʒ/（行动名词）；con /kən/（一起）；or /ər/（物名词）

#### 9. 词根: curs/cur + 同义/同源词根（curs/cur） + /kɜːrs/ /kɜːr/
2. 核心含义: 跑、流动  
3. 词源背景: 来自拉丁语“currere”（跑），常用于课程、电流等词汇  
4. 记忆技巧: 想象“current”像“跑”着的电流，“cursor”是“跑”动的光标  
5. 3个高频词:
   - current /ˈkɜːrənt/：cur（跑） + rent（形容词后缀），指“当前的/电流”，像“跑”着的流动
   - occur /əˈkɜːr/：oc（向） + cur（跑），意思是“发生”，“跑”向出现
   - cursor /ˈkɜːrsər/：curs（跑） + or（物后缀），表示“光标”，“跑”动的指示器
6. 同源词根的常见音变规律: curs（名词）；cur（简化动词，s省略）  
7. 高频词中出现的词缀总结: rent /rənt/（形容词）；oc /ə/（向）；or /sər/（物名词）

#### 10. 词根: fug + 同义/同源词根（fugi/fug） + /fjuːɡ/
2. 核心含义: 逃、流  
3. 词源背景: 来自拉丁语“fugere”（逃跑），常用于逃亡、离心等词汇  
4. 记忆技巧: 想到“fugitive”是一个“逃”跑的人，“refuge”是“逃”到庇护所  
5. 3个高频词:
   - fugitive /ˈfjuːdʒətɪv/：fug（逃） + itive（形容词后缀），指“逃亡者/短暂的”，像“逃”跑的人
   - refuge /ˈrɛfjuːdʒ/：re（回） + fug（逃） + e（连接），意思是“避难所”，“逃”回的安全处
   - centrifugal /ˌsɛntrɪˈfjuːɡl/：centri（中心） + fug（逃） + al（相关的），表示“离心的”，“逃”离中心的力
6. 同源词根的常见音变规律: fug稳定；fugi（动词形式，加i）  
7. 高频词中出现的词缀总结: itive /ɪtɪv/（形容词）；re /rɛ/（回）；e /dʒ/（连接）；centri /ˌsɛntrɪ/（中心）；al /ɡl/（相关的形容词）

### 小作文：Finding the Right Place to Stay

When traveling, I always look for a hospitable host who can provide a comfortable hospital-like care in their home. The location is key—I prefer a stable spot via a quiet street, away from the mobile crowds that might displace my peace. Once I sit down to plan, I position my map and deposit my thoughts on paper, imagining the voyage ahead. Current trends show that people often occur in groups, running like a cursor across the screen to find refuge from daily chaos. But for me, the best way is to avoid fugitive plans and allocate time to truly enjoy the place.

（注：hospitable /hɑːˈspɪtəbl/ (好客的); chaos /ˈkeɪɑːs/ (混乱)）

中文总结：这篇文章描述了旅行时寻找合适住宿的过程：寻找好客的主人、关键的位置、通过安静街道、远离移动人群、坐下来规划、放置地图、沉积想法、想象旅程、当前趋势显示人们聚集、像光标奔跑寻找庇护，但作者避免逃亡式计划，分配时间真正享受地方。
