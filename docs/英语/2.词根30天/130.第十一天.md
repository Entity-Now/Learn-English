#### **jur**

*   **词根**：jur
*   **核心含义**：法律
*   **词源背景**：源自拉丁语 *jus*，意为“法律、权利”，古罗马社会中用于指代法律体系和司法权力
*   **记忆技巧**：联想“jury（陪审团）”与法律相关
*   **高频词（3个）**：
    *   **jury** = jur(法律)+y → 陪审团 /ˈdʒʊəri/
    *   **jurisdiction** = juris(法律)+diction(说) → 司法权 /ˌdʒʊərɪsˈdɪkʃən/
    *   **injury** = in(进入)+jur(法律)+y → 伤害 /ˈɪndʒəri/
*   **组合逻辑解释**：
    *   jury：法律+名词 → 审判法律案件的人 → 陪审团
    *   jurisdiction：法律+说 → 有权“说法律”的范围 → 司法管辖权
    *   injury：进入+法律 → 触犯法律或造成损害 → 伤害
*   **同源词根及音变规律**：
    *   jus（拉丁） ↔ jur（英语）
    *   音变规律：s → r
*   **词缀总结**：
    *   in-(进入)
    *   -y(名词后缀)
    *   -diction(说话)

#### **leg**

*   **词根**：leg
*   **核心含义**：法律
*   **词源背景**：源自拉丁语 *lex, legis*，意为“法律、规则”，古罗马立法体系的核心词
*   **记忆技巧**：联想“legal（合法的）”
*   **高频词（3个）**：
    *   **legal** = leg(法律)+al(形容词) → 合法的 /ˈliːɡəl/
    *   **legislate** = legis(法律)+late(带来) → 立法 /ˈlɛɡɪsleɪt/
    *   **legitimate** = leg(法律)+it+mate → 合法的 /lɪˈdʒɪtɪmət/
*   **组合逻辑解释**：
    *   legal：法律+形容词 → 与法律相关 → 合法的
    *   legislate：法律+带来 → 制定法律 → 立法
    *   legitimate：法律+使… → 符合法律 → 合法的
*   **同源词根及音变规律**：
    *   lex（拉丁） ↔ leg（英语）
    *   音变规律：x → g
*   **词缀总结**：
    *   -al(形容词)
    *   -late(带来)
    *   -mate(使…)
