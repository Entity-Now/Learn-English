*   **词根**：**manu**
    *   **核心含义**：手
    *   **词源背景**：来自拉丁语 *manus*，原指“人的手”，在古罗马法律与契约中常用“手”象征掌控与执行权。
    *   **记忆技巧**：联想“manual 是手册，用手操作的东西”。
    *   **高频词（3个）**：
        *   **manual** = manu(手)+al(形容词) → 手工的 /ˈmænjuəl/
        *   **manufacture** = manu(手)+fact(做)+ure(名词) → 制造 /ˌmænjʊˈfæktʃə/
        *   **manipulate** = manu(手)+pul(拉)+ate(动词) → 操控 /məˈnɪpjʊleɪt/
    *   **组合逻辑解释**：
        *   manual：用“手”做的 → 手工的、手动的
        *   manufacture：手+做 → 早期靠手工制作 → 制造
        *   manipulate：手+拉 → 用手灵活摆弄 → 操控
    *   **同源词根及音变规律**：
        *   man（手）↔ main（古法语“手”）
        *   音变：u → a / 手部词汇中的元音弱化
    *   **词缀总结**：
        *   -al：形容词后缀，…的
        *   -ure：名词后缀，行为/结果
        *   -ate：动词后缀，使…化

*   **词根**：**ped**
    *   **核心含义**：脚
    *   **词源背景**：来自拉丁语 *pes/pedis*，古罗马人以“脚”代表行动与移动，有时用于士兵行军的记录。
    *   **记忆技巧**：联想“pedal 踩踏板用脚”。
    *   **高频词（3个）**：
        *   **pedal** = ped(脚)+al(名词) → 踏板 /ˈpedl/
        *   **pedestrian** = ped(脚)+estri(街道)+an(人) → 行人 /pəˈdestriən/
        *   **expedition** = ex(出去)+ped(脚)+ition(名词) → 远征 /ˌekspəˈdɪʃn/
    *   **组合逻辑解释**：
        *   pedal：脚+板 → 脚踩的板 → 踏板
        *   pedestrian：脚+街道+人 → 在街上用脚走的人 → 行人
        *   expedition：出去+脚 → 用脚走出去的旅程 → 远征
    *   **同源词根及音变规律**：
        *   pod（希腊语“脚”）与 ped 为对应关系
        *   音变：p 不变，元音从 e→o
    *   **词缀总结**：
        *   ex-：向外
        *   -an：表示人
        *   -ition：行为结果

*   **词根**：**dent**
    *   **核心含义**：牙
    *   **词源背景**：来自拉丁语 *dens/dentis*，古罗马医学记录中常用此词表示牙齿结构与疾病。
    *   **记忆技巧**：联想“dentist 牙医”。
    *   **高频词（3个）**：
        *   **dental** = dent(牙)+al(形容词) → 牙科的 /ˈdentl/
        *   **dentist** = dent(牙)+ist(人) → 牙医 /ˈdentɪst/
        *   **indent** = in(向内)+dent(牙印) → 缩进 /ɪnˈdent/
    *   **组合逻辑解释**：
        *   dental：牙+的 → 与牙相关 → 牙科的
        *   dentist：牙+从事者 → 看牙的人 → 牙医
        *   indent：向内+牙印 → 咬出凹痕 → 缩进
    *   **同源词根及音变规律**：
        *   odon（希腊语“牙”）↔ dent
        *   音变：d→od（前置元音）
    *   **词缀总结**：
        *   -al：…的
        *   -ist：职业/从事某事的人
        *   in-：向内

*   **词根**：**derm**
    *   **核心含义**：皮肤
    *   **词源背景**：来自希腊语 *derma*，原意为“剥下来的皮”，古希腊医学常从动物皮观察结构。
    *   **记忆技巧**：联想“皮肤科 dermatology”。
    *   **高频词（3个）**：
        *   **dermatology** = derm(皮肤)+logy(学科) → 皮肤学 /ˌdɜːməˈtɒlədʒi/
        *   **epidermis** = epi(上)+derm(皮) → 表皮 /ˌepiˈdɜːmɪs/
        *   **hypodermic** = hypo(下)+derm(皮)+ic(形容词) → 皮下的 /ˌhaɪpəˈdɜːmɪk/
    *   **组合逻辑解释**：
        *   dermatology：皮肤+学 → 研究皮肤 → 皮肤学
        *   epidermis：上+皮 → 最外层皮 → 表皮
        *   hypodermic：下+皮 → 皮下位置 → 皮下的
    *   **同源词根及音变规律**：
        *   derm ↔ dermat（加 t 表名词延长）
        *   epi/hypo 来自希腊方位前缀
    *   **词缀总结**：
        *   logy：学科
        *   epi-：在…上
        *   hypo-：在…下

*   **词根**：**cardi**
    *   **核心含义**：心脏
    *   **词源背景**：来自希腊语 *kardia*，古希腊人认为心脏是情绪与生命的核心。
    *   **记忆技巧**：联想“cardio 健身中的有氧运动，锻炼心脏”。
    *   **高频词（3个）**：
        *   **cardiac** = cardi(心)+ac(形容词) → 心脏的 /ˈkɑːdiæk/
        *   **cardiology** = cardi(心)+logy(学科) → 心脏学 /ˌkɑːdiˈɒlədʒi/
        *   **cardiogram** = cardi(心)+gram(写/图) → 心电图 /ˈkɑːdiəʊɡræm/
    *   **组合逻辑解释**：
        *   cardiac：心+的 → 与心有关 → 心脏的
        *   cardiology：心+学 → 研究心脏 → 心脏学
        *   cardiogram：心+图 → 记录心脉波动的图 → 心电图
    *   **同源词根及音变规律**：
        *   cor（拉丁语“心”）与 cardi 同源
        *   音变：k → c，a→o
    *   **词缀总结**：
        *   -ac：形容词后缀
        *   -logy：学科
        *   -gram：图形/记录

*   **词根**：**neur**
    *   **核心含义**：神经
    *   **词源背景**：源自希腊语 *neuron*，最初指“筋、腱”，后用于指神经束状结构。
    *   **记忆技巧**：联想“neuron 神经元”。
    *   **高频词（3个）**：
        *   **neuron** = neur(神经)+on(名词) → 神经元 /ˈnjʊərɒn/
        *   **neurology** = neur(神经)+logy(学科) → 神经学 /njʊəˈrɒlədʒi/
        *   **neurosis** = neur(神经)+osis(病) → 神经官能症 /njʊəˈrəʊsɪs/
    *   **组合逻辑解释**：
        *   neuron：神经+体 → 神经的基本单位 → 神经元
        *   neurology：神经+学 → 研究神经 → 神经学
        *   neurosis：神经+病 → 神经功能异常 → 神经症
    *   **同源词根及音变规律**：
        *   nerv（拉丁语“神经”）↔ neur
        *   音变：v ↔ u
    *   **词缀总结**：
        *   logy：学科
        *   -osis：疾病
        *   -on：事物单位

*   **词根**：**oste**
    *   **核心含义**：骨
    *   **词源背景**：来自希腊语 *osteon*，古希腊解剖学中常用以描述骨骼结构。
    *   **记忆技巧**：联想“osteoporosis 骨质疏松”。
    *   **高频词（3个）**：
        *   **osteoporosis** = oste(骨)+poro(孔)+sis(病) → 骨质疏松 /ˌɒstiəʊpəˈrəʊsɪs/
        *   **osteology** = oste(骨)+logy(学科) → 骨骼学 /ˌɒstiˈɒlədʒi/
        *   **osteopath** = oste(骨)+path(治疗) → 整骨医生 /ˈɒstiəpæθ/
    *   **组合逻辑解释**：
        *   osteoporosis：骨+孔+病 → 骨头变空洞 → 骨质疏松
        *   osteology：骨+学 → 研究骨骼 → 骨骼学
        *   osteopath：骨+治疗 → 通过调整骨骼治疗 → 整骨师
    *   **同源词根及音变规律**：
        *   oss（拉丁语“骨”）↔ oste
        *   音变：st→ss（简化）
    *   **词缀总结**：
        *   -logy：学科
        *   -sis：疾病
        *   -path：治疗

*   **词根**：**hemat**
    *   **核心含义**：血
    *   **词源背景**：源自希腊语 *haima*，古希腊医学将血液视为四大体液之一。
    *   **记忆技巧**：联想“hemoglobin 血红蛋白”。
    *   **高频词（3个）**：
        *   **hematology** = hemat(血)+logy(学) → 血液学 /ˌheməˈtɒlədʒi/
        *   **hemoglobin** = hemo(血)+globin(球蛋白) → 血红蛋白 /ˌheməˈɡləʊbɪn/
        *   **hemorrhage** = hemo(血)+rrhage(流) → 出血 /ˈhemərɪdʒ/
    *   **组合逻辑解释**：
        *   hematology：血+学 → 研究血液 → 血液学
        *   hemoglobin：血+蛋白 → 携氧的血蛋白 → 血红蛋白
        *   hemorrhage：血+流 → 血大量流出 → 出血
    *   **同源词根及音变规律**：
        *   hem ↔ hemat（t为扩展形式）
        *   音变：ai→e（简化为 hemo-）
    *   **词缀总结**：
        *   logy：学
        *   -globin：蛋白
        *   -rrhage：大量流出

*   **词根**：**pod**
    *   **核心含义**：足
    *   **词源背景**：来自希腊语 *pous/podos*，古希腊生物学中用于分类动物的足部结构。
    *   **记忆技巧**：联想 tripod（三脚架）→ 三只“足”。
    *   **高频词（3个）**：
        *   **podiatrist** = pod(足)+iatr(治)+ist(人) → 足病医生 /pəˈdaɪətrɪst/
        *   **tripod** = tri(三)+pod(脚) → 三脚架 /ˈtraɪpɒd/
        *   **antipode** = anti(相对)+pode(脚) → 对跖点 /ˈæntɪˌpəʊdz/
    *   **组合逻辑解释**：
        *   podiatrist：脚+治疗+人 → 医治脚的人 → 足科医生
        *   tripod：三+脚 → 三条腿的支撑物 → 三脚架
        *   antipode：相对+脚 → 地球上脚正对脚的地方 → 对跖点
    *   **同源词根及音变规律**：
        *   ped（拉丁语“脚”）↔ pod（希腊语）
        *   音变：e → o 为语言系统差异
    *   **词缀总结**：
        *   tri-：三
        *   anti-：相对
        *   -ist：人

*   **词根**：**cephal**
    *   **核心含义**：头
    *   **词源背景**：来自希腊语 *kephalē*，在古代动物解剖中用于描述头部结构。
    *   **记忆技巧**：联想“cephalopod 头足类”。
    *   **高频词（3个）**：
        *   **encephalitis** = en(在内)+cephal(头)+itis(炎症) → 脑炎 /enˌsefəˈlaɪtɪs/
        *   **cephalopod** = cephal(头)+pod(足) → 头足类动物 /ˈsefələpɒd/
        *   **microcephaly** = micro(小)+cephal(头)+y(名词) → 小头畸形 /ˌmaɪkrəˈsefəli/
    *   **组合逻辑解释**：
        *   encephalitis：在内+头+炎 → 头部内部发炎 → 脑炎
        *   cephalopod：头+足 → 脚长在头边 → 头足类（章鱼等）
        *   microcephaly：小+头 → 头异常小 → 小头症
    *   **同源词根及音变规律**：
        *   cap（拉丁语“头”）与 cephal 同源
        *   音变：k → c/ce，p → ph（希腊语送气音）
    *   **词缀总结**：
        *   -itis：炎症
        *   micro-：小
        *   en-：内部
