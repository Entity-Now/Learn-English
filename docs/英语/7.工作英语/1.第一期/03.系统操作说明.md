# 第一期 - 03. 系统操作说明

## 1. 登录与基本操作 (Login & Basic Operations)

### 常用例句与拆解

*   **例句 1**：We need to log in to this system first.
    *   **中文翻译**：我们需要先登录这个系统。
    *   **结构解析**：We (主语) + need (谓语) + to log in (不定式短语作宾语) + [to this system] (方向状语) + [first] (副词作时间/顺序状语)。
*   **例句 2**：Please enter your username and password here.
    *   **中文翻译**：请在这里输入你的用户名和密码。
    *   **结构解析**：Please (语气词) + enter (祈使句谓语) + your username and password (并列宾语) + [here] (地点状语)。
*   **例句 3**：Click the menu button in the top left corner to switch pages.
    *   **中文翻译**：点击左上角的菜单按钮以切换页面。
    *   **结构解析**：Click (祈使句谓语) + the menu button (宾语) + [in the top left corner] (位置状语/后置定语) + [to switch pages] (不定式短语作目的状语)。
*   **例句 4**：You can use the search bar to find specific functions.
    *   **中文翻译**：你可以使用搜索栏来查找特定功能。
    *   **结构解析**：You (主语) + can use (情态动词+动词原形作谓语) + the search bar (宾语) + [to find specific functions] (不定式短语作目的状语)。
*   **例句 5**：This is the dashboard with an overview of all systems.
    *   **中文翻译**：这是包含所有系统概览的仪表盘。
    *   **结构解析**：This (主语) + is (系动词) + the dashboard (表语) + [with an overview] (with 引导的介词短语作后置定语) + [of all systems] (介词短语，修饰 overview)。

### 核心技术与办公词汇精讲

#### 1. log in
*   **音标**：/lɒɡ ɪn/ (英) /lɔːɡ ɪn/ (美)
*   **核心释义**：*v.* 登录（系统、账户等）
*   **词根解析**：log（原指航海日志，记录木块在水中漂浮的速度，后引申为记录系统事件的日志）+ in（进入）-> 登入并记录日志 -> 登录。
*   **变形拓展**：login (名词，登录/登录名)；log out / log off (注销，登出)。
*   **例句**：You must log in to your account before you pull the code.
    *   **中文翻译**：在拉取代码之前，你必须先登录你的账户。
    *   **结构解析**：You (主语) + must log in (情态动词+动词原形作谓语) + [to your account] (方向状语) + [before you pull the code] (before 引导的时间状语从句)。

#### 2. username
*   **音标**：/ˈjuːzəneɪm/ (英) /ˈjuːzərneɪm/ (美)
*   **核心释义**：*n.* 用户名，登录账号
*   **词根解析**：user（使用者）+ name（名字）。
*   **例句**：Your username is typically your company email prefix.
    *   **中文翻译**：你的用户名通常是你的公司邮箱前缀。
    *   **结构解析**：Your username (主语) + is (系动词) + typically (副词作状语) + your company email prefix (表语)。

#### 3. password
*   **音标**：/ˈpɑːswɜːd/ (英) /ˈpæswɜːrd/ (美)
*   **核心释义**：*n.* 密码，口令
*   **词根解析**：pass（通过）+ word（单词，言语）-> 能够通过的凭证词 -> 密码。
*   **例句**：Keep your password secure and do not share it with others.
    *   **中文翻译**：保证你的密码安全，不要透露给他人。
    *   **结构解析**：Keep your password secure (祈使句复合宾语结构，secure为宾补) + and (连词) + do not share it (祈使句否定式谓语+宾语) + [with others] (介词短语作状语)。

#### 4. dashboard
*   **音标**：/ˈdæʃbɔːd/ (英) /ˈdæʃbɔːrd/ (美)
*   **核心释义**：*n.* 仪表盘，控制面板
*   **词根解析**：dash（飞奔，猛撞）+ board（板）-> 原指马车前挡泥的木板，后指汽车和飞机的仪表盘，现引申为系统状态的汇总监控界面。
*   **例句**：The monitoring dashboard displays real-time backend request metrics.
    *   **中文翻译**：监控仪表盘实时显示后端请求指标。
    *   **结构解析**：The monitoring dashboard (主语) + displays (谓语，单三) + real-time backend request metrics (宾语)。

#### 5. overview
*   **音标**：/ˈəʊvəvjuː/ (英) /ˈoʊvərvjuː/ (美)
*   **核心释义**：*n.* 概览，概述，大纲
*   **词根解析**：over（在...上方）+ view（看）-> 从上往下看全局 -> 概览。
*   **例句**：Please provide a brief overview of the project architecture.
    *   **中文翻译**：请提供对该项目架构的简要概述。
    *   **结构解析**：Please (语气词) + provide (祈使句谓语动词) + a brief overview (宾语) + [of the project architecture] (of 引导的介词短语作后置定语)。

## 2. 功能与导航 (Features & Navigation)

### 常用例句与拆解

*   **例句 1**：This section contains general statistics and reports.
    *   **中文翻译**：该部分包含常规统计数据和报告。
    *   **结构解析**：This section (主语) + contains (谓语，单三) + general statistics and reports (并列宾语)。
*   **例句 2**：You can click this button to navigate to another page.
    *   **中文翻译**：你可以点击此按钮跳转到另一个页面。
    *   **结构解析**：You (主语) + can click (情态动词+动词原形作谓语) + this button (宾语) + [to navigate] (不定式短语作目的状语) + [to another page] (方向状语)。
*   **例句 3**：This dropdown menu allows you to switch between different modes.
    *   **中文翻译**：这个下拉菜单允许你在不同的模式之间进行切换。
    *   **结构解析**：This dropdown menu (主语) + allows (谓语，单三) + you (宾语) + [to switch] (不定式短语作宾语补足语) + [between different modes] (between 引导的介词短语，作状语)。
*   **例句 4**：The sidebar on the left contains all the main functions.
    *   **中文翻译**：左侧的边栏包含所有的主要功能。
    *   **结构解析**：The sidebar (主语) + [on the left] (位置状语作后置定语) + contains (谓语，单三) + all the main functions (宾语)。
*   **例句 5**：Clicking this icon will expand or collapse the menu.
    *   **中文翻译**：点击此图标将展开或折叠菜单。
    *   **结构解析**：Clicking this icon (动名词短语作主语) + will expand or collapse (并列谓语) + the menu (宾语)。

### 核心技术与办公词汇精讲

#### 1. section
*   **音标**：/ˈsekʃn/
*   **核心释义**：*n.* 部分，板块，小节，区域；*v.* 切开
*   **词根解析**：sect（切，割，来自拉丁语 secare）+ -ion（名词后缀）-> 切下来的一个部分 -> 区域，板块。
*   **常用搭配**：comment section (评论区)；profile section (个人资料板块)。
*   **例句**：The documentation section contains code snippets and configurations.
    *   **中文翻译**：文档部分包含了代码片段和配置信息。
    *   **结构解析**：The documentation section (主语) + contains (谓语，单三) + code snippets and configurations (并列宾语)。

#### 2. statistics
*   **音标**：/stəˈtɪstɪks/
*   **核心释义**：*n.* 统计数据，统计学
*   **词根解析**：state（国家，政权，来自拉丁语 status）-> 收集国家情况的学科 -> 统计学，后引申为数据。
*   **变形拓展**：statistic (名词，统计数值)；statistical (形容词，统计的)。
*   **例句**：These statistics indicate that the server load peaks at midnight.
    *   **中文翻译**：这些统计数据表明，服务器负载在午夜达到峰值。
    *   **结构解析**：These statistics (主语) + indicate (谓语) + [that the server load peaks at midnight] (that 引导的宾语从句)；
        *   *宾语从句内部*：the server load (主语) + peaks (不及物动词，达到顶峰，作谓语) + [at midnight] (时间状语)。

#### 3. dropdown
*   **音标**：/ˈdrɒppaʊn/ (英) /ˈdrɑːpdaʊn/ (美)
*   **核心释义**：*adj.* 下拉式的；*n.* 下拉菜单
*   **词根解析**：drop（落下）+ down（向下）。
*   **常用搭配**：dropdown menu (下拉菜单)；dropdown list (下拉列表)。
*   **例句**：Choose the correct branch from the dropdown menu in Git client.
    *   **中文翻译**：在 Git 客户端的下拉菜单中选择正确的分支。
    *   **结构解析**：Choose (祈使句谓语) + the correct branch (宾语) + [from the dropdown menu] (来源状语) + [in Git client] (地点状语)。

#### 4. navigate
*   **音标**：/ˈnævɪɡeɪt/
*   **核心释义**：*v.* 导航，浏览，跳转，寻找路径
*   **词根解析**：nav（船，来自拉丁语 navis）+ ig（驾驶，做，来自 agere）+ -ate（动词后缀）-> 驾船前行 -> 航行，导航。
*   **变形拓展**：navigation (导航，名词)；navigator (导航仪，浏览器内核标识)。
*   **例句**：Users can navigate through different pages using the bottom tab bar.
    *   **中文翻译**：用户可以使用底部的标签栏在不同页面间进行跳转。
    *   **结构解析**：Users (主语) + can navigate (情态动词+动词原形作谓语) + [through different pages] (方向状语) + [using the bottom tab bar] (现在分词短语作伴随/工具状语)。

#### 5. collapse
*   **音标**：/kəˈlæps/
*   **核心释义**：*v.* 折叠，收起；崩溃，瓦解；*n.* 折叠，崩溃
*   **词根解析**：col-（共同，加强）+ lapse（滑落，掉下）-> 全部掉落下来 -> 倒塌，折叠。
*   **变形拓展**：collapsible (可折叠的，可收缩的)。
*   **例句**：The sidebar will automatically collapse when the screen width is too small.
    *   **中文翻译**：当屏幕宽度过小时，侧边栏会自动折叠收起。
    *   **结构解析**：The sidebar (主语) + will automatically collapse (助动词+副词+动词原形作谓语，collapse为不及物动词) + [when the screen width is too small] (when 引导的时间状语从句)。

## 3. 数据与操作 (Data & Operations)

### 常用例句与拆解

*   **例句 1**：You can filter the data using these options.
    *   **中文翻译**：你可以使用这些选项来过滤数据。
    *   **结构解析**：You (主语) + can filter (情态动词+动词原形作谓语) + the data (宾语) + [using these options] (现在分词短语作伴随状语/工具状语)。
*   **例句 2**：This button allows you to export the data to Excel.
    *   **中文翻译**：此按钮允许你将数据导出到 Excel。

*   **例句 3**：You can refresh the data by clicking this icon.
    *   **中文翻译**：你可以通过点击此图标来刷新数据。
    *   **结构解析**：You (主语) + can refresh (情态动词+动词原形作谓语) + the data (宾语) + [by clicking this icon] (by 引导的介词短语作方式状语，clicking为动名词)。
*   **例句 4**：The system automatically saves your progress every 5 minutes.
    *   **中文翻译**：系统每隔 5 分钟自动保存你的进度。
    *   **结构解析**：The system (主语) + automatically (副词作状语) + saves (谓语，单三) + your progress (宾语) + [every 5 minutes] (时间状语)。
*   **例句 5**：You can set up notifications in the settings menu.
    *   **中文翻译**：你可以在设置菜单中设置通知。
    *   **结构解析**：You (主语) + can set up (情态动词+动词原形作谓语) + notifications (宾语) + [in the settings menu] (地点状语)。

### 核心技术与办公词汇精讲

#### 1. filter
*   **音标**：/ˈfɪltə(r)/ (英) /ˈfɪltər/ (美)
*   **核心释义**：*v.* 过滤，筛选；*n.* 过滤器，筛选条件
*   **词根解析**：源自中世纪拉丁语 filtrum（毛毡，原用于过滤液体）-> 过滤。
*   **常用搭配**：filter data (过滤数据)；search filter (搜索条件/过滤器)。
*   **例句**：We should filter the input data to prevent SQL injection attacks.
    *   **中文翻译**：我们应当过滤输入数据以防止 SQL 注入攻击。
    *   **结构解析**：We (主语) + should filter (情态动词+动词原形作谓语) + the input data (宾语) + [to prevent SQL injection attacks] (不定式短语作目的状语)。

#### 2. export
*   **音标**：/ɪkˈspɔːt/ (英) /ɪkˈspɔːrt/ (美)
*   **核心释义**：*v.* 导出，输出，出口；*n.* 导出，出口
*   **词根解析**：ex-（向外）+ port（运送，港口，来自拉丁语 portare）-> 运送出去 -> 导出，输出。
*   **变形拓展**：import (导入/进口)；exporter (导出器/出口商)。
*   **例句**：Click the button below to export the system logs to a CSV file.
    *   **中文翻译**：点击下方按钮将系统日志导出为 CSV 文件。
    *   **结构解析**：Click (祈使句谓语) + the button (宾语) + [below] (后置定语，修饰 button) + [to export the system logs] (不定式短语作目的状语) + [to a CSV file] (方向状语)。

#### 3. refresh
*   **音标**：/rɪˈfreʃ/
*   **核心释义**：*v.* 刷新（页面/数据），使恢复精神；*n.* 刷新
*   **词根解析**：re-（重新，再次）+ fresh（新鲜的）-> 重新变新鲜 -> 刷新。
*   **常用搭配**：refresh token (刷新令牌)；page refresh (页面刷新)。
*   **例句**：The application automatically uses the refresh token to request a new access token.
    *   **中文翻译**：该应用程序自动使用刷新令牌来请求一个新的访问令牌。
    *   **结构解析**：The application (主语) + automatically (副词作状语) + uses (谓语，单三) + the refresh token (宾语) + [to request a new access token] (不定式短语作目的状语)。

#### 4. automatically
*   **音标**：/ˌɔːtəˈmætɪkli/ (英) /ˌɑːtəˈmætɪkli/ (美)
*   **核心释义**：*adv.* 自动地，机械地
*   **词根解析**：auto-（自动的，自身的）+ mat（思考，行动，来自希腊语 matos 意为自发的）+ -ic（形容词后缀）+ -al（形容词后缀）+ -ly（副词后缀）-> 自动地。
*   **变形拓展**：automatic (自动的，形容词)；automation (自动化，名词)。
*   **例句**：The script will automatically backup the database files every midnight.
    *   **中文翻译**：该脚本将在每天半夜自动备份数据库文件。
    *   **结构解析**：The script (主语) + will automatically backup (助动词+副词+动词原形作谓语) + the database files (宾语) + [every midnight] (时间状语)。

#### 5. notification
*   **音标**：/ˌnəʊtɪfɪˈkeɪʃn/ (英) /ˌnoʊtɪfɪˈkeɪʃn/ (美)
*   **核心释义**：*n.* 通知，告示，警报消息
*   **词根解析**：not（知道，注意，源自 notify）+ -fication（名词后缀，表行为/状态，源于 facy 制作，使...化）-> 使大家知道的行为 -> 通知，警报。
*   **常用搭配**：push notification (推送通知)；email notification (邮件通知)。
*   **例句**：The system sends a notification to the administrator when a deployment fails.
    *   **中文翻译**：当部署失败时，系统会给管理员发送一个通知。
    *   **结构解析**：The system (主语) + sends (谓语，单三) + a notification (直接宾语) + [to the administrator] (间接宾语) + [when a deployment fails] (when 引导的时间状语从句)。
