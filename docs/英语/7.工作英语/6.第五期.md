# 常用单词记忆

1. proposal 提案；建议；

词根：pro-向前；pos放；al名词后缀；向前摆出想法=提案；

例句：I read the architecture proposal yesterday.

结构解析：I (主语) read（谓语） the architecture proposal（宾语） yesterday （时间往后扔）

2. interception 拦截；

词根：inter（在...之间）cept（抓）-ion名词后缀；在半路截抓-》拦截。

例句：The gateway handles request interception。

结构解析：The gateway(主语) handles（谓语）request interception（宾）

3. on-demand 按需的；随选的；

词根：on（按照）demand（需求）-》按需

例句：Vue3 supports on-demand import

结构解析：Vue3 (主) supports（谓语）on-demand import （宾）

4. Advantages 优势；好处（原形为advantage）

词根：advant（前面，前进）-age（名词后缀）跑在前面的地方-》优势

例句：Tailwind CSS has many advantages. Tailwind css有很多优势。

结构解析：Tailwind CSS (主) has（谓）many advantages（宾）

5. maintainable 可维护的，易保养的

词根：maintain（维护）-able（能..的，形容词后缀）-》可维护的

例句：Your TypeScript code is very maintainable.

结构解析：Your TypeScript code(主) is（系） very maintainable （表/状态）

6. encapsulate 封装，包裹

词根：en（进入）capsule（胶囊）-ate（动词后缀）-》放入胶囊（封装）

例句：This service encapsulates the API logic.

结构解析：This service (主) encapsulates（谓/单三）the API logic（宾）

7. bitwise 位元的，按位（运算）的

词根：bit(比特) -wise（方式/方向）按比特位的方式-》按位

例句：We use bitwise operators for flags 我们使用位运算来处理标志位

结构解析：We (主) use（谓）bitwise operators（宾）for flags（目地）

8. anonymous 匿名的；无名的

词根：an（无、没有）+onym（名字）+-ous（形容词后桌）没有名字的（匿名的）

例句：This is an anonymous function in Kotlin 这是kotlin中的匿名函数

结构解析：This （主） is（系）an anonymous function（表/身份）

9. typically 典型的，通常

词根：type（类型）typical（典型的）typically （副词后缀，通常）

例句： c# developer typically use Rider or VS. c#开发者通常使用Rider或VS

结构解析：c# developer (主) typically（副词修饰）use（谓）rider or Vs（宾）

10. failures 失败；故障

词根：fail（失败；挂掉）-use（名词后缀）故障、失败

例句：The database log shows dark failures. 日志显示了潜在的故障

12. Raised 提高；筹集；抛出；

词根：raise（升起/提出）-ed（形容词/过去分词后缀）

例句：An error was raised in the code.

结构解析：An error （主语）was raised（谓语）in the code（地点状语）

13. eventually 最终；最后

词根：e-（出）vent（来）ual（形容词后缀）ly（副词后缀）

例句：The bug was eventually fixed.

结构解析：The bug（主语）was fixed（谓语）eventually（副词）

14. necessary 必要的；必须的

词根：ne-（不）cede（退让）ary（形容词后缀）不可退让的

例句：This update is necessary for security

结构解析：This update （主语）is（系）necessary（表语）for security（目地状语）

15. underlying 潜在的；基础的

词根：under（在...下面）lying（躺）-》位于底层的

例句：We need to fix the underlying issue.

结构解析：we（主语）need（谓语）to fix the underlying issue（动词不定式作宾语）

16. associate 关联；关系

词根：as-（去/向）soci（伙伴/结合）-ate（动词后缀）

例句：You can associate a route with a component.

结构解析：You（你）can associate（谓语）a route（宾语）with a component（介词短语，固定搭配：associate A with B）

17. canonical 标准的；权威的

词根：canon（标准）-ical（形容词后缀）

例句：Use the canonical URL for this page.

结构解析：Use（动词原形开头）the canonical url（宾语） for this page （定语）

18. Manipulation 操作；处理

词根：mani（手）pul（引申为操作）-ation（名词后缀）-》用手操作

例句：DOM manipulation can slow down the app

结构解析：DOM manipulation（主语）can slow down（谓语）the app （宾语）

19. decouple 解耦；分离

词根：de-（除去/反转）couple（连接/结合）-》解耦

例句：We should decouple the frontend from the backend.

结构解析：We（主语）should decouple （谓语）the frontend（宾语）from the backend（介词短语）

20. contrast 对比；比照

词根：contra（相反）st（站立）-》相对而立-》对比

例句：The dark theme provides good contrast.

结构解析：The dark theme（主语）provides（谓语）good contrast（宾语）

21. offsetting 抵消；补偿

词根：off（离开）setting（放置）-》靠偏置来抵消

例句：Offsetting the margin fixes the layout

结构解析：Offsetting the margin（动名词短语作主语）fixes （谓语）the layout（宾语）

22. regardless 不管怎样；无论如何

词根：regard（关心、注意）-less（无/没有）不关系的-》不管

例句：The function runs regardless of the error

结构解析：The function（主语）runs（谓语）regardless of the errors（介词短语作状语）

23. deemed 被认为；被视作

词根：deem（认为/判断）-ed（过去分词后缀）

例句：The data was deemed unsafe.

结构解析：The data（主语）was deemed（谓语）unsafe（主语补足语）

24. therefore 因此；所以

词根：there（那里）fore（在前/为此）->由此

例句：The API is down, therefore the app fails.

结构解析：The API is down（前置句子）therefore（副词）the app fails（后置句子）

25. snapshot 快照

词根：snap（快速点击/拍摄）shot（照片）-> 快照

例句：Take a snapshot of the database

结构解析：Take（祈使句）a snapshot（宾语）of the database （定语）

26. persist 持久化；坚持

词根：per-（始终）sist（站立）-> 始终站立（持久）

例句：We need to persist the user data.

结构解析：we（主语）need（谓语）to persist the user data（动词不定式作宾语）
