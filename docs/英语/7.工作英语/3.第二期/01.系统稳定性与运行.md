# 第二期 - 01. 系统稳定性与运行

## 核心词汇精讲

### 1. robust
*   **音标/词性**：`/rəʊˈbʌst/` (英) `/roʊˈbʌst/` (美) · `adj.`
*   **核心释义**：强健的、体魄结实的；（系统、架构等）稳定可靠的、健壮的，抗错性强的
*   **词根解析**：rob（橡树，强壮，来自拉丁语 robur）+ -ust（形容词后缀）-> 像橡树一样坚固的 -> 强健的，健壮的。
*   **常用搭配**：
    *   **robust health** 强健的体魄
    *   **robust system** 稳定可靠的系统
    *   **robust architecture** 健壮的系统架构
*   **例句与拆解**：
    *   **例句 1**：He keeps **robust health** by regular exercise.
        *   **中文翻译**：他通过规律锻炼保持强健的体魄。
        *   **结构解析**：He (主语) + keeps (谓语，单三) + robust health (宾语) + [by regular exercise] (by 引导的介词短语作方式状语，exercise为名词)。
    *   **例句 2**：This software has a **robust system** to resist unexpected errors.
        *   **中文翻译**：这款软件搭载稳定可靠的系统，可抵御突发故障。
        *   **结构解析**：This software (主语) + has (谓语) + a robust system (宾语) + [to resist unexpected errors] (不定式短语作目的状语或后置定语)。

### 2. properly
*   **音标/词性**：`/ˈprɒpəli/` (英) `/ˈprɑːpərli/` (美) · `adv.`
*   **核心释义**：恰当地、正确地、妥善地，适当地
*   **词根解析**：propr（拥有，专属，合适，来自拉丁语 proprius）+ -ly（副词后缀）-> 合适地 -> 妥善地，正确地。
*   **常用搭配**：
    *   **do something properly** 妥善/正确地做某事
    *   **properly cropped** 裁剪得当的
    *   **properly configured** 正确配置的
*   **例句与拆解**：
    *   **例句 1**：Make sure the image is **properly cropped** before uploading.
        *   **中文翻译**：上传前要确保图片经过妥善裁剪。
        *   **结构解析**：Make sure (引导祈使句) + [the image is properly cropped] (宾语从句) + [before uploading] (before 引导的时间状语，uploading为动名词)；
            *   *从句内部*：the image (主语) + is (系动词) + properly (副词作状语) + cropped (形容词化过去分词，作表语)。
    *   **例句 2**：The system will not work unless it is **properly configured**.
        *   **中文翻译**：除非正确配置，否则系统将无法工作。
        *   **结构解析**：The system (主语) + will not work (谓语，否定将来时) + [unless it is properly configured] (unless 引导的条件状语从句)；
            *   *从句内部*：it (主语) + is (系动词) + properly (副词状语) + configured (过去分词表语)。

### 3. efficiently
*   **音标/词性**：`/ɪˈfɪʃntli/` · `adv.`
*   **核心释义**：高效地，有效率地
*   **词根解析**：ef-（向外，ex-在f前变体）+ fic（做，制造，来自拉丁语 facere）+ -ent（形容词后缀）+ -ly（副词后缀）-> 做出成果地 -> 高效地。
*   **常用搭配**：
    *   **work efficiently** 高效地工作
    *   **use resources efficiently** 高效利用资源
    *   **operate efficiently** 高效运转
*   **例句与拆解**：
    *   **例句**：All machines in the factory **operate efficiently** every day.
        *   **中文翻译**：工厂里的所有机器每天都在高效运转。
        *   **结构解析**：All machines (主语) + [in the factory] (地点状语作定语) + operate (不及物动词作谓语) + efficiently (副词作状语) + [every day] (时间状语)。

### 4. flawlessly
*   **音标/词性**：`/ˈflɔːləsli/` · `adv.`
*   **核心释义**：完美地、无瑕疵地，毫无缺陷地
*   **词根解析**：flaw（瑕疵，缺陷）+ -less（无，没有）-> flawless（无瑕疵的）+ -ly（副词后缀）-> 完美地。
*   **常用搭配**：
    *   **run flawlessly** 完美运行
    *   **work flawlessly** 无瑕地工作
    *   **perform flawlessly** 表现得完美无缺
*   **例句与拆解**：
    *   **例句**：The software program **runs flawlessly** on both iOS and Android.
        *   **中文翻译**：该软件程序在 iOS 和 Android 系统上都能完美运行。
        *   **结构解析**：The software program (主语) + runs (不及物动词作谓语，单三) + flawlessly (副词作状语) + [on both iOS and Android] (地点状语)。

### 5. seamlessly
*   **音标/词性**：`/ˈsiːmləsli/` · `adv.`
*   **核心释义**：无缝地，流畅地
*   **词根解析**：seam（缝隙，接缝）+ -less（无）-> seamless（无缝的）+ -ly（副词后缀）-> 没有接缝地 -> 无缝地。
*   **常用搭配**：
    *   **integrate seamlessly** 无缝集成/整合
    *   **seamless transition** 无缝过渡
    *   **work seamlessly together** 无缝协同工作
*   **例句与拆解**：
    *   **例句**：The new frontend dashboard **integrates seamlessly** with our backend APIs.
        *   **中文翻译**：新的前端仪表盘与我们的后端 API 无缝集成。
        *   **结构解析**：The new frontend dashboard (主语) + integrates (不及物动词作谓语，单三) + seamlessly (副词状语) + [with our backend APIs] (对象状语)。

### 6. pageable
*   **音标/词性**：`/ˈpeɪdʒəbl/` · `adj.`
*   **核心释义**：可分页的 (在编程中常指数据分页)
*   **词根解析**：page（页，页面）+ -able（能...的，形容词后缀）-> 能够分页处理的。
*   **常用搭配**：
    *   **pageable query** 可分页查询
    *   **pageable results** 可分页结果
    *   **pageable memory** 可分页内存
*   **例句与拆解**：
    *   **例句**：We designed a **pageable API** to load large datasets in smaller chunks.
        *   **中文翻译**：我们设计了一个可分页的 API，以便分批加载大型数据集。
        *   **结构解析**：We (主语) + designed (谓语，过去式) + a pageable API (宾语) + [to load large datasets] (不定式短语作目的状语) + [in smaller chunks] (以较小的块，方式状语)。

### 7. incoming
*   **音标/词性**：`/ˈɪnkʌmɪŋ/` · `adj.`
*   **核心释义**：传入的、来袭的、进站的，新进的
*   **词根解析**：in（向内）+ come（来）+ -ing（形容词后缀）-> 进来的 -> 传入的。
*   **常用搭配**：
    *   **incoming request** 传入的请求，入站请求
    *   **incoming mail / call** 来信/来电
    *   **incoming traffic** 入站流量
*   **例句与拆解**：
    *   **例句**：The firewall is configured to block unauthorized **incoming requests**.
        *   **中文翻译**：防火墙已配置为阻止未授权的传入请求。
        *   **结构解析**：The firewall (主语) + is configured (被动语态作谓语) + [to block unauthorized incoming requests] (不定式短语作目的/结果状语)。

### 8. overhead
*   **音标/词性**：`/ˈəʊvəhed/` (英) `/ˈoʊvərhed/` (美) · `n. & adj.`
*   **核心释义**：系统开销、运行损耗，额外开销；日常开支、管理费用；在头顶上的
*   **词根解析**：over（在...上方）+ head（头）-> 在头顶上漂浮着的开销 -> 企业的日常杂支 / 计算机系统的额外损耗。
*   **常用搭配**：
    *   **system overhead** 系统开销
    *   **performance overhead** 性能损耗/开销
    *   **reduce overhead** 降低日常开销
*   **例句与拆解**：
    *   **例句**：Using lightweight components helps minimize memory **overhead**.
        *   **中文翻译**：使用轻量级组件有助于将内存开销降至最低。
        *   **结构解析**：Using lightweight components (动名词短语作主语) + helps (谓语，单三) + [minimize memory overhead] (省 to 不定式短语，作 helps 的宾语)。

### 9. against
*   **音标/词性**：`/əˈɡenst/` · `prep.`
*   **核心释义**：反对于；防备、抵御，对抗；紧靠，倚在
*   **词源**：来自古英语 ongean（面对，相反）+ -st（副词后缀）-> 相反方向地 -> 抵御，紧靠。
*   **常用搭配**：
    *   **against the rules** 违反规则
    *   **fight against** 与...作斗争
    *   **protect against** 防御、免受
    *   **lean against** 倚靠着
*   **例句与拆解**：
    *   **例句**：The security team implemented measures to protect the network **against** cyber threats.
        *   **中文翻译**：安全团队采取了措施，以保护网络免受网络威胁。
        *   **结构解析**：The security team (主语) + implemented (谓语，过去式) + measures (宾语) + [to protect the network [against cyber threats]] (不定式短语作目的状语，protect A against B 属于固定搭配)。

### 10. establish
*   **音标/词性**：`/ɪˈstæblɪʃ/` · `v.`
*   **核心释义**：建立、确立，证实，创办
*   **词根解析**：st（站立，稳定，来自拉丁语 stare/stabilis）+ -ish（动词后缀）-> 使之站稳 -> 建立，确立。
*   **常用搭配**：
    *   **establish a connection** 建立连接
    *   **establish a business relationship** 建立业务关系
    *   **establish a standard** 确立标准
*   **例句与拆解**：
    *   **例句**：The system was unable to **establish** a connection with the remote server.
        *   **中文翻译**：系统无法与远程服务器建立连接。
        *   **结构解析**：The system (主语) + was (系动词) + unable (形容词表语) + [to establish a connection [with the remote server]] (不定式短语，作形容词unable的修饰成分)。
