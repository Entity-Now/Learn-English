# 第九期 - 02. 网络与系统技术词汇

## 核心词汇精讲

### 1. dropped
*   **音标/词性**：`/drɒpt/` (英) `/drɑːpt/` (美) · `adj. & v. (drop的过去式/过去分词)`
*   **核心释义**：断开的，掉线的，被丢弃的（在网络连接中特指连接断开或数据包丢失 packet loss）
*   **词根解析**：drop（滴下，掉落，源自古英语 dropian）+ -ed（过去分词/形容词后缀）-> 掉落下去的 -> 被丢弃的，掉线的。
*   **常用搭配**：
    *   **dropped packets** 丢包，被丢弃的数据包
    *   **connection was dropped** 连接断开/掉线
*   **例句与拆解**：
    *   **例句**：The connection was **dropped** due to network instability.
        *   **中文翻译**：连接由于网络不稳定而断开了。
        *   **结构解析**：The connection (主语) + was dropped (被动语态作谓语) + [due to network instability] (due to 引导的介词短语，作原因状语)。

### 2. ping
*   **音标/词性**：`/pɪŋ/` · `v. & n.`
*   **核心释义**：`v.` 检测网络连接是否通畅（通过发送 ICMP 包）；`n.` 探测，延迟时间
*   **词源**：拟声词（潜艇声纳发出的啪啪回波声）。在计算机网络中，表示向特定 IP 发送数据包测试连通性。
*   **常用搭配**：
    *   **ping the server** 检测（ping）服务器
    *   **low ping** 低延迟
*   **例句与拆解**：
    *   **例句**：Let me **ping** the gateway first to diagnose the latency.
        *   **中文翻译**：让我先 ping 一下网关以诊断延迟问题。
        *   **结构解析**：Let (使役动词) + me (宾语) + ping the gateway first (省 to 不定式，作宾补，first修饰ping) + [to diagnose the latency] (不定式短语作目的状语)。

### 3. reach
*   **音标/词性**：`/riːtʃ/` · `v. & n.`
*   **核心释义**：`v.` 连通，能够到达，触及（在网络中表示能路由通达并建立通信）
*   **词源**：来自古英语 ræcan（延伸，伸出）。在 IT 中指物理或逻辑网络路径是通的。
*   **常用搭配**：
    *   **reach the server** 连通服务器
    *   **unreachable** 无法连通的（形容词）
*   **例句与拆解**：
    *   **例句**：I cannot **reach** the production server from my local IP.
        *   **中文翻译**：我无法从本地 IP 连接上生产服务器。
        *   **结构解析**：I (主语) + cannot (情态动词否定式) + reach (谓语动词原形) + the production server (宾语) + [from my local IP] (地点状语)。

### 4. connect to
*   **音标/词性**：`/kəˈnekt tuː/` · `v. (动词短语)`
*   **核心释义**：连接到...，与...建立物理或逻辑连接
*   **词根解析**：con-（共同）+ nect（系，连接，来自拉丁语 nectere）-> 连到一起。to 为方向介词。
*   **常用搭配**：
    *   **connect to database** 连接到数据库
    *   **connect to server** 连接到服务器
*   **例句与拆解**：
    *   **例句**：I cannot **connect to** the server database.
        *   **中文翻译**：我无法连接到服务器数据库。
        *   **结构解析**：I (主语) + cannot (情态动词否定式) + connect to (谓语动词短语) + the server database (宾语)。

### 5. disconnect
*   **音标/词性**：`/ˌdɪskəˈnekt/` · `v.`
*   **核心释义**：断开（网络连接、电源、数据库会话等）
*   **词根解析**：dis-（否定，相反动作）+ connect（连接）-> 断开连接。
*   **常用搭配**：
    *   **disconnect from the network** 断开网络连接
    *   **get disconnected** 掉线，连接断开
*   **例句与拆解**：
    *   **例句**：I got **disconnected** during the database deployment.
        *   **中文翻译**：在数据库部署过程中我断开了连接（掉线了）。
        *   **结构解析**：I (主语) + got (系动词，表状态变化) + disconnected (形容词化的过去分词，作表语，构成 get done 结构) + [during the database deployment] (时间状语)。

### 6. time out
*   **音标/词性**：`/taɪm aʊt/` · `v. & n. (名词常拼写为 timeout)`
*   **核心释义**：`v.` 超时，因响应时间过长而中断；`n.` 超时限制，超时阈值
*   **词根解析**：time（时间）+ out（超出范围）-> 响应时间超出预设的阈值。
*   **常用搭配**：
    *   **request timeout** 请求超时
    *   **timeout settings** 超时时间配置
*   **例句与拆解**：
    *   **例句**：The client request **timed out**.
        *   **中文翻译**：客户端请求超时了。
        *   **结构解析**：The client request (主语) + timed out (不及物动词短语，作谓语，过去式)。

### 7. proxy
*   **音标/词性**：`/ˈprɒksi/` (英) /ˈprɑːksi/ (美) · `n.`
*   **核心释义**：代理，代理服务器，代理人
*   **词根解析**：procure（代理，取得，来自拉丁语 procurare）的缩写，pro-（代替）+ cura（照料）-> 替别人代理。
*   **常用搭配**：
    *   **reverse proxy** 反向代理
    *   **proxy server** 代理服务器
    *   **http proxy** HTTP 代理
*   **例句与拆解**：
    *   **例句**：Please check your browser **proxy** configuration.
        *   **中文翻译**：请检查你的浏览器代理配置。
        *   **结构解析**：Please (语气词) + check (祈使句谓语) + your browser proxy configuration (名词短语作宾语)。

### 8. kill
*   **音标/词性**：`/kɪl/` · `v.`
*   **核心释义**：强行结束，杀死（进程、服务或后台任务）
*   **词源**：源自古英语 cullen（宰杀，消灭）。在操作系统中特指通过发送强制信号（如 kill -9）强行中止进程。
*   **常用搭配**：
    *   **kill a process** 杀死进程
    *   **kill a session** 终止会话
*   **例句与拆解**：
    *   **例句**：Can you **kill** this zombie process to release memory?
        *   **中文翻译**：你能杀掉这个僵尸进程以释放内存吗？
        *   **结构解析**：Can (情态动词) + you (主语) + kill (谓语原形) + this zombie process (宾语) + [to release memory] (不定式短语作目的状语)？

### 9. terminate
*   **音标/词性**：`/ˈtɜːmɪneɪt/` (英) /ˈtɜːrmɪneɪt/ (美) · `v.`
*   **核心释义**：终止，结束，使终结（通常用于比较正式、温和地终止进程，允许其完成清理工作）
*   **词根解析**：termin（界限，终点，来自拉丁语 terminus）+ -ate（动词后缀）-> 设终点 -> 终止。
*   **常用搭配**：
    *   **terminate connection** 终止连接
    *   **terminate contract** 终止合同
    *   **terminate process** 终止进程
*   **例句与拆解**：
    *   **例句**：The system will **terminate** idle connection sessions automatically.
        *   **中文翻译**：系统将自动终止空闲的连接会话。
        *   **结构解析**：The system (主语) + will (助动词) + terminate (谓语原形) + idle connection sessions (宾语) + [automatically] (副词作状语)。

### 10. freeze
*   **音标/词性**：`/friːz/` · `v. & n.`
*   **核心释义**：`v.` 卡死，冻结，无响应；`n.` 冻结，卡顿
*   **词源**：源自古英语 freosan（结冰）。在 IT 领域形象地指系统或 UI 界面被“冻结”，无法响应用户操作。
*   **常用搭配**：
    *   **application freeze** 应用卡死/冻结
    *   **screen freezes** 屏幕卡住
*   **例句与拆解**：
    *   **例句**：The application **froze** when I clicked the button.
        *   **中文翻译**：当我点击该按钮时，应用程序卡死了。
        *   **结构解析**：The application (主语) + froze (谓语，freeze 的过去式) + [when I clicked the button] (when 引导的时间状语从句)；
            *   *从句内部*：I (主语) + clicked (谓语) + the button (宾语)。

### 11. hang
*   **音标/词性**：`/hæŋ/` · `v. & n.`
*   **核心释义**：`v.` 挂起，卡住，挂起不动（通常指进程或线程由于等待资源而处于永久阻塞状态）
*   **词源**：源自古英语 hangian（悬挂，吊着）。
*   **常用搭配**：
    *   **process hang** 进程挂起/卡住
    *   **the system hangs** 系统挂起
*   **例句与拆解**：
    *   **例句**：The compiler may **hang** if there is an infinite loop.
        *   **中文翻译**：如果存在死循环，编译器可能会卡死挂起。
        *   **结构解析**：The compiler (主语) + may (情态动词) + hang (谓语原形) + [if there is an infinite loop] (if 引导的条件状语从句)；
            *   *从句内部*：there (引导词) + is (系动词) + an infinite loop (主语)。

### 12. spin up
*   **音标/词性**：`/spɪn ʌp/` · `v. (动词短语)`
*   **核心释义**：快速拉起，启动，创建并运行（通常指快速拉起 Docker 容器、虚拟机或新进程）
*   **词根解析**：spin（旋转）+ up（向上）。最早源自磁带机或机械硬盘电机加速达到工作转速，现引申为快速拉起虚拟计算资源。
*   **常用搭配**：
    *   **spin up a VM** 拉起一台虚拟机
    *   **spin up containers** 快速启动容器
*   **例句与拆解**：
    *   **例句**：Let us **spin up** a new Docker container for local testing.
        *   **中文翻译**：让我们拉起一个新的 Docker 容器用于本地测试。
        *   **结构解析**：Let (使役动词) + us (宾语) + spin up a new Docker container (省 to 不定式，作宾补，spin up 为动词短语) + [for local testing] (目的状语)。

### 13. launch
*   **音标/词性**：`/lɔːntʃ/` · `v. & n.`
*   **核心释义**：`v.` 启动，发起，发布（程序，项目等）
*   **词源**：来自古法语 lancher（投掷，掷出）。
*   **例句与拆解**：
    *   **例句**：Please **launch** Chrome in incognito mode.
        *   **中文翻译**：请在无痕模式下启动 Chrome。
        *   **结构解析**：Please (语气词) + launch (祈使句谓语) + Chrome (宾语) + [in incognito mode] (方式/状态状语)。

### 14. restart
*   **音标/词性**：`/ˌriːˈstɑːt/` (英) `/ˌriːˈstɑːrt/` (美) · `v. & n.`
*   **核心释义**：重新启动，重启
*   **词根解析**：re- (重新) + start (开始) -> 再次开始。
*   **例句与拆解**：
    *   **例句**：You need to **restart** the daemon process.
        *   **中文翻译**：你需要重新启动守护进程。
        *   **结构解析**：You (主语) + need (谓语) + to restart the daemon process (不定式短语作宾语)。
