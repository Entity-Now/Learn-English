# 第九期 - 03. 排障与资源管控词汇

## 核心词汇精讲

### 1. bypass
*   **音标/词性**：`/ˈbaɪpɑːs/` (英) /ˈbaɪpæs/ (美) · `v. & n.`
*   **核心释义**：`v.` 绕过，跳过，不经由；`n.` 旁路，绕道
*   **词根解析**：by-（旁侧）+ pass（通过）-> 从旁边通过 -> 绕过。
*   **常用搭配**：
    *   **bypass a proxy** 绕过代理
    *   **bypass security checks** 绕过安全检查
*   **例句与拆解**：
    *   **例句**：Is there a way to **bypass** this proxy?
        *   **中文翻译**：有办法绕过这个代理吗？
        *   **结构解析**：Is (系动词) + there (引导词) + a way (真正主语) + [to bypass this proxy] (不定式短语作后置定语，修饰 way)？

### 2. workaround
*   **音标/词性**：`/ˈwɜːkəraʊnd/` (英) /ˈwɜːr kəraʊnd/ (美) · `n.`
*   **核心释义**：临时替代方案，规避方法，变通方案（通常指在彻底修复 bug 前，临时绕过该问题的方法）
*   **词根解析**：work（工作）+ around（绕着）-> 绕开问题去工作 -> 变通方案。
*   **常用搭配**：
    *   **temporary workaround** 临时替代方案
    *   **find a workaround** 找到变通方法
*   **例句与拆解**：
    *   **例句**：This is just a temporary **workaround** for the known bug.
        *   **中文翻译**：这只是针对已知 bug 的一个临时变通方案。
        *   **结构解析**：This (主语) + is (系动词) + [just] (副词作状语) + a temporary workaround (表语) + [for the known bug] (介词短语作后置定语，修饰 workaround)。

### 3. permanent solution
*   **音标/词性**：`/ˈpɜːmənənt səˈluːʃn/` (英) /ˈpɜːrmənənt/ · `n.`
*   **核心释义**：永久解决方案，根本解决方法（与 workaround 相对）
*   **词根解析**：permanent（永久的，per-始终 + manere留）+ solution（解决方案，solv-解开 + -ution）。
*   **常用搭配**：
    *   **seek a permanent solution** 寻求永久解决方案
    *   **implement a permanent solution** 实施根本解决方法
*   **例句与拆解**：
    *   **例句**：We need to find a **permanent solution** to this issue.
        *   **中文翻译**：我们需要找到这个问题的根本解决方案。
        *   **结构解析**：We (主语) + need (谓语) + to find a permanent solution (不定式短语作宾语) + [to this issue] (to 引导的介词短语，作 solution 的后置定语)。

### 4. grant
*   **音标/词性**：`/ɡrɑːnt/` (英) `/ɡrænt/` (美) · `v. & n.`
*   **核心释义**：`v.` 授予，批准（权限、访问权或资源）；`n.` 拨款，授予物
*   **词源**：源于古法语 graunter（允许，许可）。
*   **常用搭配**：
    *   **grant permission / access** 授予权限/访问权
    *   **grant admin privileges** 授予管理员特权
*   **例句与拆解**：
    *   **例句**：Can you **grant** me admin access to this repo?
        *   **中文翻译**：你能给我这个仓库的管理员权限吗？
        *   **结构解析**：Can (情态动词) + you (主语) + grant (谓语，接双宾语) + me (间接宾语) + admin access (直接宾语) + [to this repo] (定语，修饰 access)？

### 5. revoke
*   **音标/词性**：`/rɪˈvəʊk/` (英) `/rɪˈvoʊk/` (美) · `v.`
*   **核心释义**：撤销，注销，废除（权限、Token、凭证等，使其失效）
*   **词根解析**：re-（反向，收回）+ voke（呼喊，来自拉丁语 vocare）-> 喊回来 -> 收回，撤销。
*   **常用搭配**：
    *   **revoke permissions** 撤销权限
    *   **revoke a token** 使 Token 失效/注销 Token
*   **例句与拆解**：
    *   **例句**：The token has been **revoked** due to security policy.
        *   **中文翻译**：出于安全策略，该 Token 已经被注销失效。
        *   **结构解析**：The token (主语) + has been revoked (谓语，现在完成时的被动语态) + [due to security policy] (原因状语)。

### 6. capture
*   **音标/词性**：`/ˈkæptʃə(r)/` (英) /ˈkæptʃər/ (美) · `v. & n.`
*   **核心释义**：`v.` 捕获，采集，截图；`n.` 捕获，捕捉
*   **词根解析**：capt（抓，握，来自拉丁语 capere）+ -ure（名词/动词后缀）-> 抓住 -> 捕获（如捕获异常 `capture exceptions` 或捕获数据包 `capture packets`）。
*   **常用搭配**：
    *   **capture network packets** 捕获网络数据包
    *   **capture exception** 捕获异常
    *   **screen capture** 屏幕截图/录屏
*   **例句与拆解**：
    *   **例句**：We need to **capture** the network packet to analyze the request header.
        *   **中文翻译**：我们需要捕获数据包来分析请求头。
        *   **结构解析**：We (主语) + need (谓语) + to capture the network packet (不定式短语作宾语) + [to analyze the request header] (不定式短语作目的状语)。

### 7. deploy
*   **音标/词性**：`/dɪˈplɔɪ/` · `v.`
*   **核心释义**：部署，发布，配置（将编译好的代码或资源包部署到特定环境运行）
*   **词根解析**：de-（解除，展开）+ ploy（折叠，来自拉丁语 plicare）-> 展开折叠的包 -> 展开，部署。
*   **常用搭配**：
    *   **deploy to production** 部署到生产环境
    *   **deployment pipeline** 部署流水线（名词）
*   **例句与拆解**：
    *   **例句**：We will **deploy** the microservice to Kubernetes cluster tonight.
        *   **中文翻译**：我们今晚将把微服务部署到 Kubernetes 集群中。
        *   **结构解析**：We (主语) + will deploy (将来时作谓语) + the microservice (宾语) + [to Kubernetes cluster] (方向状语) + [tonight] (时间状语)。

### 8. sniff
*   **音标/词性**：`/snɪf/` · `v. & n.`
*   **核心释义**：`v.` 嗅探，监听（网络流量或数据包）；`n.` 嗅，吸气声
*   **词源**：拟声词（吸气闻味）。在安全或网络分析中特指使用工具被动监听链路上的流量。
*   **常用搭配**：
    *   **sniff traffic** 监听/嗅探流量
    *   **packet sniffer** 网络抓包工具/嗅探器
*   **例句与拆解**：
    *   **例句**：Make sure no malicious tools **sniff** local network traffic.
        *   **中文翻译**：确保没有恶意工具在监听本地网络流量。
        *   **结构解析**：Make sure (引导祈使句) + [no malicious tools sniff local network traffic] (宾语从句)；
            *   *从句内部*：no malicious tools (主语) + sniff (谓语) + local network traffic (宾语)。

### 9. bundle
*   **音标/词性**：`/ˈbʌndl/` · `v. & n.`
*   **核心释义**：`v.` 打包，构建发布包，合并；`n.` 资源包，捆
*   **词源**：由 bind（绑定）派生而来。在打包工具（如 Webpack、Vite）中特指将多个源文件合并压缩成少数几个优化后的文件。
*   **常用搭配**：
    *   **bundle size** 打包体积
    *   **resource bundle** 资源包
*   **例句与拆解**：
    *   **例句**：Webpack will **bundle** JavaScript modules into a single entry file.
        *   **中文翻译**：Webpack 会将 JavaScript 模块打包到单个入口文件中。
        *   **结构解析**：Webpack (主语) + will bundle (将来时作谓语) + JavaScript modules (宾语) + [into a single entry file] (结果状语)。

### 10. escalate
*   **音标/词性**：`/ˈeskəleɪt/` · `v.`
*   **核心释义**：（将工单、问题或Bug）升级处理，汇报给上级
*   **词根解析**：escal（阶梯，来自拉丁语 scala）+ -ate（动词后缀）-> 沿着阶梯逐级上升 -> 升级工单，提交给更高级别的负责人。
*   **常用搭配**：
    *   **escalate a ticket** 升级工单
    *   **escalate the issue to** 将问题上升汇报给...
*   **例句与拆解**：
    *   **例句**：Since the issue is severe, we must **escalate** it to the database team.
        *   **中文翻译**：既然该问题很严重，我们必须将其升级提交给数据库团队。
        *   **结构解析**：[Since the issue is severe] (since 引导的原因状语从句) + we (主语) + must (情态动词) + escalate (谓语原形) + it (宾语) + [to the database team] (对象状语)。

### 11. provision
*   **音标/词性**：`/prəˈvɪʒn/` · `v. & n.`
*   **核心释义**：`v.` 开通，配置，分配（虚拟资源、云端服务器或环境）；`n.` 准备，条款
*   **词根解析**：pro-（预先）+ vis（看，来自拉丁语 videre）+ -ion（名词/动词后缀）-> 预先看好并备齐所需资源 -> 开通，配置（如 Terraform 资源配置）。
*   **常用搭配**：
    *   **provision a server** 开通/配置一台服务器
    *   **resource provisioning** 资源分配/调配
*   **例句与拆解**：
    *   **例句**：The cloud platform takes a few minutes to **provision** a new server instance.
        *   **中文翻译**：云平台需要几分钟时间来开通一个新的服务器实例。
        *   **结构解析**：The cloud platform (主语) + takes (谓语，单三) + a few minutes (宾语，表示时间开销) + [to provision a new server instance] (动词不定式短语，作真正动作主语/时间开销动作)。

### 12. deprecate
*   **音标/词性**：`/ˈdeprəkeɪt/` · `v.`
*   **核心释义**：弃用，不赞成使用，废弃（在 API 设计或依赖库升级中表示已过时，建议开发者更换）
*   **词根解析**：de-（反向，除去）+ prec（祈祷，来自拉丁语 precari）+ -ate（动词后缀）-> 祈求让其远离 -> 弃用。
*   **常用搭配**：
    *   **deprecated API** 已弃用的 API
    *   **deprecation warning** 弃用警告
*   **例句与拆解**：
    *   **例句**：We will **deprecate** the old API in the next major release.
        *   **中文翻译**：我们将在下一个重大版本发布中弃用旧版 API。
        *   **结构解析**：We (主语) + will deprecate (将来时作谓语) + the old API (宾语) + [in the next major release] (时间状语)。

### 13. roll back
*   **音标/词性**：`/rəʊl bæk/` (英) `/roʊl bæk/` (美) · `v. (动词短语，名词为 rollback)`
*   **核心释义**：回滚，退回（至先前的状态或版本）
*   **词根解析**：roll（滚动）+ back（向后）-> 倒着滚回去 -> 回滚（如数据库事务回滚、代码版本回滚）。
*   **常用搭配**：
    *   **roll back a transaction** 回滚事务
    *   **roll back deployment** 回滚部署
*   **例句与拆解**：
    *   **例句**：If the deployment fails, the pipeline will **roll back** to the previous stable release.
        *   **中文翻译**：如果部署失败，流水线将自动回滚至上一个稳定版本。
        *   **结构解析**：[If the deployment fails] (if 引导的条件状语从句) + the pipeline (主语) + will roll back (谓语) + [to the previous stable release] (方向状语)。

### 14. authenticate
*   **音标/词性**：`/ɔːˈθentɪkeɪt/` · `v.`
*   **核心释义**：认证，鉴权，验证身份（验证用户的账号密码或密钥是否真实有效）
*   **词根解析**：authentic（真实的，权威的）+ -ate（动词后缀）-> 确认为真 -> 验证身份，鉴权。
*   **常用搭配**：
    *   **authenticate users** 验证用户身份
    *   **multi-factor authentication (MFA)** 多因素身份验证（名词）
*   **例句与拆解**：
    *   **例句**：Users must **authenticate** using multi-factor credentials to access this network.
        *   **中文翻译**：用户必须使用多因素凭证进行身份验证以访问该网络。
        *   **结构解析**：Users (主语) + must authenticate (情态动词修饰谓语动词原形) + [using multi-factor credentials] (现在分词短语作方式状语) + [to access this network] (不定式短语作目的状语)。

### 15. override
*   **音标/词性**：`/ˌəʊvəˈraɪd/` (英) `/ˌəʊvərˈraɪd/` (美) · `v. & n.`
*   **核心释义**：覆盖，重写（配置、样式或父类方法），越权
*   **词根解析**：over（在...上方）+ ride（骑，跨越）-> 骑在最上面 -> 压制，覆写，使之具有更高优先级（如 OOP 中的 `@Override`，或 CSS 中的 `!important` 覆盖）。
*   **常用搭配**：
    *   **override a method** 重写/覆写方法
    *   **override configurations** 覆盖配置
*   **例句与拆解**：
    *   **例句**：You can **override** the default configurations by editing the local setting file.
        *   **中文翻译**：你可以通过编辑本地配置文件来覆盖默认配置。
        *   **结构解析**：You (主语) + can override (情态动词修饰谓语) + the default configurations (宾语) + [by editing the local setting file] (by 引导的介词短语作方式状语，其中 editing 为动名词)。
